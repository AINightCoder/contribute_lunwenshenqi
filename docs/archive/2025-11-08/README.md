## 🔍 核心概念速查

### 三步骤生成流程

1. **步骤1: 写作设定** - 配置文章类型、风格、篇幅、标题、模型
2. **步骤2: 段落撰写** - AI生成提纲，逐章节编写内容，可选配图
3. **步骤3: 预览编辑** - Markdown预览，复制或导出到在线编辑器

详见: [流程图文档 - 主流程](./flow-diagrams.md#11-主流程三步骤文章生成)

### 技术栈

- **前端**: Vue 3 + Element Plus + Markdown Editor
- **AI服务**: OpenAI兼容API（支持GPT/Gemini/Claude）
- **部署**: Docker + Nginx / Vercel Serverless

详见: [架构文档 - 技术栈](./architecture.md#12-技术栈)

### 核心API

- `makeSingleChat()` - SSE流式对话接口
- `getArticleKey()` - 获取文章编辑Key
- `updateArticle()` - 更新文章内容

详见: [架构文档 - API接口层](./architecture.md#32-api接口层)

### 数据存储

- **内存**: Vue响应式状态（formData, outline, articleSections）
- **本地**: localStorage（历史记录、用户配置）

详见: [流程图文档 - 数据存储流程](./flow-diagrams.md#23-数据存储流程)
