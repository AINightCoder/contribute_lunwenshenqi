
# 2融资租赁资产证券化概述

# 2.1融资租赁资产证券化的相关概念

# 2.1.1融资租赁

融资租赁是一种独特的融资方式，通常涉及一方（租赁公司）购买特定资产，然后将其租赁给另一方（承租人）以获取资金。这一模式允许承租人在不直接购买资产的情况下，获得并使用所需的设备、机械、车辆或其他实物资产。融资租赁的本质是一种合同安排，其中租赁公司作为资产的拥有者，在约定的租期内将其使用权授予承租人。这种形式的融资不仅为企业提供了获取必要资产的途径，同时也为租赁公司创造了稳定的租金收入。

融资租赁在商业和经济领域发挥着多重关键作用。通过提供无需大额一次性投资的方式，帮助企业获取和使用必要的资产，如设备和车辆，特别适用于中小型企业。同时，融资租赁使企业能够更灵活地运用资金，分期支付租金减轻了财务压力，为企业提供了降低财务负担的途径。此外，通过融资租赁，企业能够及时更新设备，跟进最新技术，提高生产效率，保持竞争力。这种灵活的融资方式还能带来潜在的税收优势，有助于企业在竞争激烈的市场中更好地适应变化、实现可持续发展。目前，融资租赁行业不断拓展业务范围，涵盖汽车、船舶、飞机等多个领域，并涌现出新兴产品如融资租赁 ABS。技术创新推动了行业数字化和高效化。

# 2.1.2资产证券化

资产证券化（Asset-backed Securities，ABS）是一种金融工具，其核心思想在于将企业的特定资产转变为法律上可交易的证券，以实现融资和风险管理的目的。这一过程涉及企业将特定资产的未来现金流作为支持，通过巧妙设计的封装和重组，将这些资产转化为市场上可流通的证券。资产证券化的关键目标是隔离和分散风险，提高资产的可流动性和可交易性，从而吸引更广泛的投资者群体。

在资产证券化的实践中，特殊目的载体（Special Purpose Vehicle，SPV）是一个重要的概念。SPV的作用是将企业选定的资产进行集中存储和包装，形成资产池，并以此作为基础发行资产支持证券。通过 SPV 的设计，旨在实现资产与原始权益人（发行人）之间的“破产隔离”，确保在任何情况下资产证券化不会影响到发行人的其他财务责任。这种机制为投资者提供了更可控、透明的投资选择，从而促进了资本市场的发展和企业融资的多样化。

# 2.1.3融资租赁资产证券化

融资租赁资产证券化是一项金融工具，旨在通过将租赁资产转化为可流通的证券，以实现融资和投资的目的。在这个过程中，租赁公司将其租赁业务中所持有的资产进行分类整合，并通过结构化的方式将其打包成可以在资本市场上进行交易的证券产品，这种方法使得原本受限于传统融资方式的租赁公司能够更广泛地获取资金。

融资租赁资产证券化的核心理念在于将租赁资产的现金流进行再造和优化，使其成为可交易的金融工具。通过将租金流作为资产支持证券的偿付来源，这一过程不仅提高了租赁公司的融资效率，还使得投资者能够参与并分享租赁资产的收益。这种金融工具的设计和发行通常涉及特殊目的载体（SPV）的运用，通过 SPV将租赁资产进行隔离，从而形成独立的资产池，提高证券的信用质量。

总体而言，融资租赁资产证券化是一项创新性的金融手段，不仅满足了租赁公司多样化融资需求，也为资本市场提供了更多元的投资选择。其成功实施不仅取决于资产证券化市场的发展成熟，还与国家政策、法律环境以及市场参与者的理解和信任密切相关。

# 2.2 融资租赁资产证券化的运作要点

融资租赁资产证券化运作包含资产池构建、设立特殊目的载体(SPV）、建立交易结构、信用评级、产品交易、到期付款管理六个环节，各环节运作要点如下：：

（1）资产池构建：在构建资产池时，融资租赁公司需充分考虑租赁资产的现金流稳定性和风险分散效应。选择高质量、多元化的租赁资产有助于提高整个资产池的信用质量，从而更容易获得资本市场的认可。

(2）设立特殊目的载体（SPV）：SPV的设立不仅仅是为了满足法律和会计的要求，更是为了实现破产隔离，保护投资者免受其他业务风险的影响。选择适当的 SPV结构和合理的法律安排是确保融资租赁资产证券化成功的重要环节。

(3）建立交易结构：融资租赁公司在建立交易结构时需密切关注监管要求和市场需求，确保符合法规标准的同时，提高产品的流动性和市场可交易性。有效的交易结构设计有助于降低投资者的交易风险，增强产品的市场竞争力。

(4）信用评级：选择信誉卓越的评级机构进行信用评级对于吸引更多投资者至关重要。租赁公司在资产证券化前应通过提高基础资产池的质量，采用风险分散策略，并可能提供增信措施，以确保产品获得较高的信用评级，降低投资者的信用风险。

(5）产品交易：在产品交易阶段，销售机构需要通过市场推广和有效的营销策略确保证券成功售出。了解投资者需求，提供有吸引力的收益和灵活的交易结构是成功销售的关键。此外，积极回应市场反馈，并根据需求进行产品创新，有助于提高产品的市场接受度。

(6）到期付款管理：在产品到期付款管理方面，融资租赁公司需建立健全的资产管理系统，以确保实时监控租赁资产池的现金流状况。通过利用技术手段和数据分析，公司可以更准确地预测和管理到期付款，提高运营效率和资产池的整体表现。

# 2.3融资租赁资产证券化的风险管理

融资租赁资产支持证券以基础资产的租金收入作为支付证券本息的来源，并以分层以及差额支付承诺等为增信手段，因此需要重点关注基础资产构建阶段风险，交易结构阶段以及存续阶段风险的识别、评估以及控制。

在基础资产构建阶段，主要存在基础资产高集中度风险、原始权益人信用风险、现金流预测风险以及第三方参与机构尽职风险。基础资产信用风险主要是指基础资产的集中度及承租人情况对基础资产现金流的不利影响。原始权益人信用风险是由于各交易主体的信息不对称性，原始权益人可能会利用信息优势，受利益驱动，导致道德风险发生。现金流预测风险是指底层租金的现金流变动对融资租赁资产证券化项目的本金和利息产生直接的影响。

在交易结构设计阶段，主要存在破产隔离风险、信用增级风险、资金混同风险以及信用评级风险。破产隔离风险破产隔离不充分风险是指在主体企业发生破产时，由于基础资产未实现真正的销售而造成的基础资产被回收和清偿。信用增级风险是不充分会给资产证券化产品的偿付带来负面的影响信贷增级措施的缺失。资金混同风险是当资产服务企业的资信状况不断恶化、偿债能力下降、甚至倒闭时，这些资产的回款就会与资产服务企业的其它资金混杂，从而使证券化产品的利息不能得到应有的偿还。信用评级风险是评级机构撤销或降低资产支持证券的现有评级可能对资产支持证券的价值带来负面影响。

在存续阶段，主要存在着差额支付承诺人履约风险、承租人提前退租风险以及操作风险。差额支付承诺人履约风险受差额支付承诺人经营状况和信用风险的影响，可能导致其未能履行差额支付义务，进而影响专项计划向资产支持证券持有人支付本金及预期收益。承租人提前退租风险源于承租人因利益变动或利率调整而提前退租，使得基础资产的资金流与预期不符，进而影响资产质量。环境风险包括了经济、法律和政治环境的不确定性，而操作风险则主要涉及人员、流程、系统和外部事件因素引发的风险。

# 2.3.2风险评估

# 2.3.2.1基础资产构建阶段风险评估

基础资产集中度风险的评估要考察基础资产的数量和分散程度。结合《企业应收账款资产支持证券挂牌条件确认指南》的规定，当基础资产中债务人的数量超过10名，债务人之间不存在关联关系，且单一债务人的入池资产占比不能超过资产池的 $50 \%$ 时，就认为该基础资产的集中度风险较低。

承租人违约风险的评估要考察对基础资产预计回收率和预计回收时间的分析。基础资产预计回收率主要受债务人资信情况、担保情况、债务受偿顺序、所在地区经济和司法环境等因素的影响，其中对于具有抵质押担保的基础资产或本身具有变现价值的基础资产，回收情沉取决于抵质押权的优先顺序，抵质押物的种类及性质，以及抵质押物的变现价值等。

原始权益人信用风险的评估要考察原始权益人的经营能力良好的情况下，要考察基础资产进行的信息披露是否完整，能如实反应对承租人的信用特征，如年龄、收入情况、行业、逾期历史情况等详细信息。

现金流预测风险的评估主要考察管理人披露的各期现金流预测结果和相应覆盖倍数的呈现，分析是否全面考虑各种因素对未来现金流变化可能造成的影响。根据现金流预测和覆盖倍数的测算结果，评价分析其未来现金流是否能够为资产支持证券的偿付提供安全范围内的保障。

# 2.3.2.2 交易结构阶段

对于破产隔离风险的评估，需要深入分析发起机构破产可能对交易产生的影响，并关注基础资产现金流是否会因此遭遇延迟或短缺。因此，在分析时要判断其是否实

现租赁债权与原始权益人的隔离，以及租赁资产与计划管理人的隔离。

信用增级风险的评估可以根据担保类型、抵质押资产类型、企业资质差异等角度进行分析，判断其是否为优先级产品提供足够的信用支持，并是否设计有效的信用触发机制用于缓解潜在的风险事件，并为发生时提供适时的应对措施。

资金混同风险的评估要着重关注是否清晰说明了基础资产现金流自产生至分配给投资人的整个过程，具体涵盖各个账户环节、资金的流入与流出时间、可能涉及的风险以及相应的监管措施，以保证现金流的合理分配。

信用评级风险的评估要考察评级机构是否有可能撤销或降低资产支持证券的现有评级，从而对资产支持证券的价值带来负面影响。

# 2.3.2.3存续阶段

差额支付承诺人履约风险的评估要需要承担差额支付承诺义务的原始权益人或者第三方机构的信用和整体经营管理能力进行评估。

承租人提前退租风险的评估主要考察承租人的回收期限，评估产品的风险暴露时间是否较长，以及行业或者政策等对承租人提前退租的影响。

# 2.3.3风险控制措施

基础资产构建阶段，要建立严格的准入机制。为避免基础资产的信用风险，在检验基础资产的合规性时，要注意对基础资产的质量进行检验，如承租人集中度、承租人情况等等。为防范原始权益人信用风险，需要对原始权益人进行充分的尽职调查，全面评估其持续经营能力。防范现金流预测风险中，在现金流预测中在以历史数据为基础，充分考虑影响未来现金流变化的各种因素进行合理预测，并对现金流进行覆盖测试等，从而为基础资产制定特定的产品计划。

交易结构设计阶段，在评估交易的真实出售和破产隔离的有效性时，需要深入分析发起机构破产可能对交易产生的影响，并关注基础资产现金流是否会因此遭遇延迟或短缺。在信用增级层面，要确保触发机制能起到事先预警的功能，同时管理人应采用差额支付承诺、分层、信用触发机制等内外部增信措施。资金混同风险要考虑现金流归集和偿付安排，涉及多个方面，包括资金的归集与划付以及资产收益的分配等。

融资租赁资产证券化存续阶段，差额支付承诺人履约风险要对差额支付承诺人的经营能力等进行评估，保证可以履行差额支付的义务。承租人提前退租风险应该要提前约定好未按照规定日期退租要进行一定的补偿。环境风险和操作风险要从基本的现金流入手，确保资金充足，并不断追踪和分析未来的政策风险。

# 2.4风险管理的理论基础

# 2.4.1资产重组原理

由于资产证券化的基础资产是构成混合基础资产池，所以可以包含来自同一企业内部或者不同地域及行业的多个企业的多项资产。融资租赁资产证券化的资产重组原理是通过重新组合租赁资产的未来现金流，并进行结构化设计，构建成可形成证券化的资产池，来达成融资租赁公司或金融租赁公司对资产的有效管理和融资的目标。这种资产证券化的形式可以更高效地管理和重新组合多元化资产，为投资者提供更为吸引人的证券产品。

# 2.4.2 风险隔离原理

在融资租赁资产证券化中，风险隔离原理通过设立特殊目的载体（SPV）实现资产的独立法律实体化，将租赁资产与融资租赁公司的其他业务隔离开来。SPV的独立性、资产剥离、独立管理以及破产违约迅速处置等关键机制确保了资产证券化产品在融资租赁公司面临财务困境时仍能保持相对稳定。这一原理的实施有效降低了投资者的信用风险，提高了产品的信用评级，进而促进了融资租赁市场的发展。通过风险隔离，投资者可以更加信任其投资的安全性，即使融资租赁公司整体面临风险，资产证券化产品仍能独立运作，维护投资者权益。

# 2.4.3信用增级原理

在融资租赁资产证券化中，信用增级原理通过对租赁资产支持证券进行信用评级和采取不同级别的信用支持措施，旨在提高产品的整体信用质量，降低投资者面临的信用风险。首先，评级机构根据一定标准对产品进行评级，划分为不同的信用等级，为投资者提供了更清晰的信用状况了解。其次，发行机构采取过渡性储备金、担保和信用保险等增级措施，为不同信用级别的证券提供不同程度的信用支持，创造了分层设计，满足了不同投资者对收益和风险的不同需求。信用增级能够在融资租赁资产证券化过程中有效提高证券的信用等级，降低资产支持证券的风险，确保融资租赁产生的现金流能够保障本息兑付。这一信用增级原理促进了融资租赁行业在资产证券化融资方面的发展，为市场参与者提供了更多选择。

# 2.4.4 信息不对称理论

信息不对称现象普遍存在于市场经济活动中，具体表现为交易双方信息获取的不平等。在传统交易中，卖方往往相较于买方拥有更详尽的商品信息，从而占据优势。在资产证券化的复杂流程中，发起人通常身兼数职，这一特点加剧了信息不对称的效应。在资产证券化的起始阶段，作为原始权益人的发起人出售基础资产时，往往面临“逆向选择”的问题，即原始权益人由于不愿以优质资产池换取较低的价格，可能存在出售质量较低资产的倾向。另一方面，在资产证券化产品的存续阶段，资产支持证券的持有者可能难以全面了解证券的管理状况，这导致资产服务机构、计划管理人等参与机构可能出现不尽职尽责的现象，即所谓的“道德风险”。逆向选择和道德风险都是信息不对称在资产证券化过程中的具体体现，它们共同增加了资产支持证券持有者面临的风险。因此，为了保障市场的公平性和效率，需要采取措施减少信息不对称的影响。
