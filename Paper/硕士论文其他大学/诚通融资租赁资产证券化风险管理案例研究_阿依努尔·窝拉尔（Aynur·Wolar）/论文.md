
# 1绪论

# 1.1 研究背景及意义

# 1.1.1研究背景

自20 世纪 50 年代起，融资租赁行业在全球迅速发展，然而我国融资租赁行业近年来在发展中遭遇一系列挑战，包括监管升级、竞争激烈以及创新不足。资金筹措成为制约发展的重要障碍，尤其是中小型融资租赁公司面临融资渠道狭窄和高成本的问题。为解决这一困境，2020 年1月，银保监会发布了《融资租赁公司监督管理暂行办法（征求意见稿）》，强调了服务装备制造业、技术升级和进出口的使命。在此背景下，租赁公司转型的必然选择是大力发展直接租赁和经营租赁。截至2021 年底，《“十四五”促进中小企业发展规划》中提到“发挥信托、租赁、保理等在中小企业融资中的作用”，进一步支持融资租赁行业的发展。同时，租赁行业也面临着市场多元化和承租人支持的压力，因为分散的承租人有助于风险分散，提升公司的抗风险能力。

自 20 世纪 70 年代以来，美国率先引入资产证券化，而租赁资产证券化因具备物权明晰、现金流稳定和风险分散等特点，逐渐成为市场中备受关注的金融工具。融资租赁行业因此迎来了新的发展机遇。对于企业而言，尤其是通过将融资租赁资产转化为证券，实现融资租赁资产证券化有助于企业优化资产负债表结构。通过这种优化，这种形式可以提高财务灵活性、减轻负债压力的同时，还能够提升企业的盈利能力，因为它为企业提供了稳定的资金支持，从而支持企业的扩张和盈利活动。对于投资者而言，融资租赁资产证券化提供了多样化的投资选择，降低了投资风险，还促进了资本市场的多层次发展。从市场方面看，资产证券化还能够提高市场的流动性和透明度，降低融资成本，增强融资租赁行业的竞争力。在国家层面上，融资租赁资产证券化推动了金融创新和国家经济发展，满足了实体经济的融资需求，为国家金融体系的健康成长做出了积极的贡献。

综合而言，通过深入研究融资租赁资产证券化风险管理，我们能够深刻理解其对融资租赁行业发展的潜在影响。解决融资难题、提升企业盈利能力、推动金融市场更健康发展成为必然目标。监管政策的支持、市场趋势的引领、资产证券化的创新等将共同助力融资租赁行业实现更加稳健和可持续的发展，为中国经济的繁荣和金融体系的健康作出重要贡献。

# 1.1.2 研究意义

(1）理论意义

当前，融资租赁资产证券化作为一个复杂而又涉及多个学科领域的课题，已经引起了广泛的研究关注。在学术界，研究者们通过对市场数据的分析、金融模型的构建以及相关政策法规的解读，逐渐揭示了融资租赁资产证券化的风险管理要点。这些研究不仅深化了对融资租赁资产证券化风险管理的理论认识，还在一定程度上指导了实际操作和监管政策的制定。因此，怎样做到有效管理和控制经济周期中的风险成为资产证券化的重要课题。本文通过对案例中的风险管理过程展开研究，并对融资租赁资产证券化项目的风险管理建议，旨在丰富融资租赁资产证券化的风险识别和防范研究层面上作出贡献。

# (2）实践意义

在实践视角出发，由于融资租赁行业来说头部融资租赁公司数量相对较少，非头部融资租赁公司在行业中所占据了行业的主体，这些非头部融资租赁公司的整体运营状况更能真实反映行业的整体态势，所以关注非头部融资租赁公司发行的资产证券化产品的潜在风险，并做好风险管理措施是十分重要的。本文将以诚通租赁的融资租赁资产支持专项计划为案例进行深入的风险管理分析，并提出针对性的防控建议。本文不仅对防控融资租赁企业资产证券化风险具有实际意义，更对维护整个资产证券化市场的稳定运行具有重要的参考价值。

# 1.2国内外文献综述

# 1.2.1融资租赁资产证券化的意义

融资租赁资产证券化作为一种融资手段，在国际学者和国内学者的研究中得到了广泛关注，因此对融资租赁资产证券化的优势研究也开始较早，企业融资租赁资产证券化的意义主要体现在以下几个方面。

第一，拓宽融资渠道。Kothari（2003）认为向投资者发行证券产品，提供了一种新颖的融资形式，为企业的融资渠道提供了新的选择[52]；章连标(2015）指出将融资租赁资产证券化可以达到相对固定收益，并且也能满足投资者的资金需求[42]；朱志恒(2017)认为融资租赁资产证券化有助于拓宽企业和个人投融资渠道[45]；刁新玉 (2018)通过金融租赁资产证券化可以适应创新需求，推动行业纵深发展[6]；聂方义（2022）融资租赁公司运作资产证券化有助于拓展融资渠道并扩大业务规模[24]。

第二，提高资金流动性。Thomas Hugh(1999）认为，由于企业在租赁市场中充当出租人和承租人的双重角色，难以实现税收优势，因此融资租赁资产证券化的主要动机在于其便利性和灵活度[57；朱志恒（2017）认为融资租赁资产证券化有助于增强金融市场流动性[45]；刘静漪（2019）企业资产证券化能够提升企业资产的流动性[19]；杨一桐和王寒笑（2020）指出租赁资产证券化能够盘活资金存量[411；蔡梦琳（2021）融资租赁企业发展融资租赁资产证券化业务的优势在于提高资金周转率[2]。

第三，优化资产负债结构。李为一，刘毅（2019）指出资产证券化融资方式对降低国有企业资产负债比率有巨大作用，能够优化负债结构[16]；杨一桐和王寒笑 (2020)指出租赁资产证券化能够优化资产负债结构[41；孙瑞霖（2020）金融租赁资产证券化能够对相关企业资产负债结构起到积极的帮助作用[31]。

第四，有效转移风险。Christopher W.Frost（1997）指出，融资租赁资产证券化在企业面临破产风险时具有显著的隔离效果，有效降低了融资失败的风险，提高了企业经营的安全性[49]；章连标(2015）指出将融资租赁资产证券化能够对资产进行风险转移[42]；张媛媛（2019）认为，在设计资产证券化产品时，SPV作为特殊法律结构专门设立，有效隔离企业破产对证券化交易带来的风险[44]；杨一桐和王寒笑(2020）指出租赁资产证券化能够降低融资风险。聂方义(2022）融资租赁公司运作资产证券化可以实现风险转移[41]。

第五，降低融资成本。Fabozzi（2007）指出，随着证券化产品的评级越来越高，融资成本就会越来越低，降低了融资成本[48]；Kopff（1990）认为，融资租赁公司通过资产证券化可以降低成本、收取费用，增加股东财富。杨一桐和王寒笑（2020）指出租赁资产证券化能够降低融资成本[511；蔡梦琳（2021）融资租赁企业发展融资租赁资产证券化业务的优势在于降低国有企业市场融资成本[2]。

# 1.2.2 融资租赁资产证券化的运作模式

国内外学者主要聚焦于融资租赁资产证券化的核心要素，包括基础资产、特定目的结构（SPV）和增信措施等进行研究。Shawn D.Halladay（2005）提出了较为新颖的融资租赁资产证券化的运作模式，指导以后设计更为丰富的资产证券化的产品[34]。Stevin.L.Shwartz(2008）阐明 SPV模式的设计是融资租赁资产证券化的核心步骤，通过合理的 SPV 设计有助于维持稳定现金流，有效隔离风险[35]。

陈楠(2010）分析了我国的融资租赁资产证券化的融资竞争力和 SPV模式两方面，着重说明了合理的资产池和合适的 SPV模式对发展融资租赁资产证券化的重要程度[3]。陈岳虹（2018）文献指出融资租赁行业存在融资渠道窄、资本不充足的问题。为了解决这一问题，租赁资产证券化被认为是一种有效的新型融资途径[4]。孙浩钧等 (2021)该文献着重研究了租赁资产证券化的脉络，如资产的优化结构、破产隔离和定价方式，进一步提出了改善市场环境、服务本地企业、优化发行定价等建议[18]。

# 1.2.3 融资租赁资产证券化的风险管理

对于风险方面的探讨，国外学者 Schlmit 和 Mathias（2004）研究了如何在资产证券化过程中的识别和管理信用风险[53]。对于风险识别、分析以及管理方面的研究，我国许多学者也颇有建树，并提出了自已的观点，其风险识别、评估和控制主要从基础资产构建、交易结构设计阶段以及存续阶段等角度出发。

在风险识别方面，在基础资产构建阶段，在国内张文玲（2016）发现融资租赁资产证券化业务中有基础资产池过度集中和股东背景薄弱等风险[43]；胥晚舟（2018）认为当前租赁行业对资产证券化的认知和活跃度相对较低，这一发展还受到行业集中度高、基础资产利率低、优质租赁资产减少等制约[40]；嵇可成和赵南（2019）我国融资租赁公司的资产证券化基础资产过于集中、违约率逐年增加的风险[10]；郑祖昀和黄瑞玲（2020）则关注了资产层面的基础资产集中度过高的风险[46]；李文娜（2022）指出融资租赁企业整体的资产证券化年化违约率逐年上升。在交易结构设计阶段，马喆（2019）提出我国证券化产品的外部增信措施存在合规性瑕疵[23]；吴滢（2021）指出我国信用评级机构整体上水平不高，信用评级机构角色缺位，提出应该穿透监管、规范信用评级机构[3⁸]。在存续阶段，刘晓（2018）对我国当前住房租赁市场资产证券化的发展进行分析，说明了还存在信用风险、操作风险、市场风险等众多风险[20]。

在风险评估方面，Binghui Wu（2018）对资产证券化投资的风险给出了风险概率和损失率两个评估角度，并运用这两个纬度构造了资产证券化收益的计算公式[5⁸]，谢赤（2018）通过KMV模型估算资产证券化的信用风险，认为部分产品的信用风险被低估，其中中小型银行风险隐患更大，合理根据风险传染机制能帮助预先防范系统风险[39]。

在风险控制方面，在基础资产构建阶段，胥晚舟（2018）认为公司应加大对业务筛选、承租人尽职调查和风险管理的把控力度，并且严格遵守贷后管理要求[43]；卢文华(2019)建议商业银行在开展业务时明确客户筛选标准、审慎把控基础资产质量等[22]；嵇可成和赵南（2019）提出要加强中介机构的监管和自律，应当强化资产证券化现金流等信息披露要求，并提高租赁公司专业化经营和风险管理能力[10]；郑祖昀和黄瑞玲(2020）提出了构建多元化基础资产池、关注宏观经济形势的建议[50]；李文娜（2022)提出融资租赁企业应要不断提升自身的风险管理能力[17]。在交易结构设计阶段，李宁等（2018）提出要完善信用增级措施，可以将租赁资产证券化分为“出表”的租赁资产和“不出表”的租赁资产[14]；马喆（2019）建议信用增级的合规性问题要充分关注监管政策的变化，与监管部门进行充分沟通[24]；洪祥骏（2019）指出应该在专项计划设计中保证对底层资产的信息披露才能彻底消除投资者对潜在风险的顾虑[8]；郑磊 (2014)提出信贷资产证券化市场监管机构提出的双评级制度可以作为参考，用来降低信用评级机构的道德风险[49]。在存续阶段，聂方义(2022）说明融资租赁资产证券化市场基础设施建设有待完善，应当加强市场基础设施建设，完善政策法规[24]。这些研究为风险识别、评估和管理提供了丰富的理论和实践支持。

# 1.2.4文献评述

融资租赁资产证券化作为解决融资租赁行业流动性不足问题的必然趋势，在国内外学者的深入研究中得到了广泛的关注。对于这一趋势，研究者们认为是由于融资租赁资产证券化包含拓宽融资渠道、提高资金流动性、优化资产负债结构、有效转移风险、降低融资成本等意义，具体表现为对投资者需求、金融市场流动性和企业经营安全性的关注。在运作模式方面，学者们提供了多方面的探讨。其中，针对融资租赁公司资产证券化的发展概况和现有模式，互联网金融与融资租赁资产证券化相融合的创新思路也备受关注。关于风险方面，学者们聚焦于基础资产构建、交易结构设计阶段以及存续阶段等角度出发，为了防范这些风险，提出了多方面建议。综合而言，融资租赁资产证券化在解决行业流动性问题的同时，也带来了一系列挑战和风险。学者们的深入分析为业界提供了理论指导和实践建议。因此，本文将采用案例分析的方式，重点探讨融资租赁资产证券化的风险管理，努力总结规律性的经验，为融资租赁资产证券化产品的风险管理提供指导，推动我国融资租赁行业更加多元化的实践发展，以促进整个行业的进步。

# 1.3.2研究方法

本文采用的研究方法有文献研究法和案例分析法。

（1）运用文献研究法，文献研究法是一种通过搜集、综述和分析已发表的文献、学术文章等信息来获取研究对象相关领域的理论和研究成果的方法。该方法通过对过去研究的系统整理，为当前研究提供理论支持和背景资料。本文在进行融资租赁资产证券化风险管理的研究中，文献研究法可以用于回顾和总结过去关于融资租赁、资产证券化以及风险管理等方面的研究概况。

# 2融资租赁资产证券化概述

# 2.1融资租赁资产证券化的相关概念

# 2.1.1融资租赁

融资租赁是一种独特的融资方式，通常涉及一方（租赁公司）购买特定资产，然后将其租赁给另一方（承租人）以获取资金。这一模式允许承租人在不直接购买资产的情况下，获得并使用所需的设备、机械、车辆或其他实物资产。融资租赁的本质是一种合同安排，其中租赁公司作为资产的拥有者，在约定的租期内将其使用权授予承租人。这种形式的融资不仅为企业提供了获取必要资产的途径，同时也为租赁公司创造了稳定的租金收入。

融资租赁在商业和经济领域发挥着多重关键作用。通过提供无需大额一次性投资的方式，帮助企业获取和使用必要的资产，如设备和车辆，特别适用于中小型企业。同时，融资租赁使企业能够更灵活地运用资金，分期支付租金减轻了财务压力，为企业提供了降低财务负担的途径。此外，通过融资租赁，企业能够及时更新设备，跟进最新技术，提高生产效率，保持竞争力。这种灵活的融资方式还能带来潜在的税收优势，有助于企业在竞争激烈的市场中更好地适应变化、实现可持续发展。目前，融资租赁行业不断拓展业务范围，涵盖汽车、船舶、飞机等多个领域，并涌现出新兴产品如融资租赁 ABS。技术创新推动了行业数字化和高效化。

# 2.1.2资产证券化

资产证券化（Asset-backed Securities，ABS）是一种金融工具，其核心思想在于将企业的特定资产转变为法律上可交易的证券，以实现融资和风险管理的目的。这一过程涉及企业将特定资产的未来现金流作为支持，通过巧妙设计的封装和重组，将这些资产转化为市场上可流通的证券。资产证券化的关键目标是隔离和分散风险，提高资产的可流动性和可交易性，从而吸引更广泛的投资者群体。

在资产证券化的实践中，特殊目的载体（Special Purpose Vehicle，SPV）是一个重要的概念。SPV的作用是将企业选定的资产进行集中存储和包装，形成资产池，并以此作为基础发行资产支持证券。通过 SPV 的设计，旨在实现资产与原始权益人（发行人）之间的“破产隔离”，确保在任何情况下资产证券化不会影响到发行人的其他财务责任。这种机制为投资者提供了更可控、透明的投资选择，从而促进了资本市场的发展和企业融资的多样化。

# 2.1.3融资租赁资产证券化

融资租赁资产证券化是一项金融工具，旨在通过将租赁资产转化为可流通的证券，以实现融资和投资的目的。在这个过程中，租赁公司将其租赁业务中所持有的资产进行分类整合，并通过结构化的方式将其打包成可以在资本市场上进行交易的证券产品，这种方法使得原本受限于传统融资方式的租赁公司能够更广泛地获取资金。

融资租赁资产证券化的核心理念在于将租赁资产的现金流进行再造和优化，使其成为可交易的金融工具。通过将租金流作为资产支持证券的偿付来源，这一过程不仅提高了租赁公司的融资效率，还使得投资者能够参与并分享租赁资产的收益。这种金融工具的设计和发行通常涉及特殊目的载体（SPV）的运用，通过 SPV将租赁资产进行隔离，从而形成独立的资产池，提高证券的信用质量。

总体而言，融资租赁资产证券化是一项创新性的金融手段，不仅满足了租赁公司多样化融资需求，也为资本市场提供了更多元的投资选择。其成功实施不仅取决于资产证券化市场的发展成熟，还与国家政策、法律环境以及市场参与者的理解和信任密切相关。

# 2.2 融资租赁资产证券化的运作要点

融资租赁资产证券化运作包含资产池构建、设立特殊目的载体(SPV）、建立交易结构、信用评级、产品交易、到期付款管理六个环节，各环节运作要点如下：：

（1）资产池构建：在构建资产池时，融资租赁公司需充分考虑租赁资产的现金流稳定性和风险分散效应。选择高质量、多元化的租赁资产有助于提高整个资产池的信用质量，从而更容易获得资本市场的认可。

(2）设立特殊目的载体（SPV）：SPV的设立不仅仅是为了满足法律和会计的要求，更是为了实现破产隔离，保护投资者免受其他业务风险的影响。选择适当的 SPV结构和合理的法律安排是确保融资租赁资产证券化成功的重要环节。

(3）建立交易结构：融资租赁公司在建立交易结构时需密切关注监管要求和市场需求，确保符合法规标准的同时，提高产品的流动性和市场可交易性。有效的交易结构设计有助于降低投资者的交易风险，增强产品的市场竞争力。

(4）信用评级：选择信誉卓越的评级机构进行信用评级对于吸引更多投资者至关重要。租赁公司在资产证券化前应通过提高基础资产池的质量，采用风险分散策略，并可能提供增信措施，以确保产品获得较高的信用评级，降低投资者的信用风险。

(5）产品交易：在产品交易阶段，销售机构需要通过市场推广和有效的营销策略确保证券成功售出。了解投资者需求，提供有吸引力的收益和灵活的交易结构是成功销售的关键。此外，积极回应市场反馈，并根据需求进行产品创新，有助于提高产品的市场接受度。

(6）到期付款管理：在产品到期付款管理方面，融资租赁公司需建立健全的资产管理系统，以确保实时监控租赁资产池的现金流状况。通过利用技术手段和数据分析，公司可以更准确地预测和管理到期付款，提高运营效率和资产池的整体表现。

# 2.3融资租赁资产证券化的风险管理

融资租赁资产支持证券以基础资产的租金收入作为支付证券本息的来源，并以分层以及差额支付承诺等为增信手段，因此需要重点关注基础资产构建阶段风险，交易结构阶段以及存续阶段风险的识别、评估以及控制。

在基础资产构建阶段，主要存在基础资产高集中度风险、原始权益人信用风险、现金流预测风险以及第三方参与机构尽职风险。基础资产信用风险主要是指基础资产的集中度及承租人情况对基础资产现金流的不利影响。原始权益人信用风险是由于各交易主体的信息不对称性，原始权益人可能会利用信息优势，受利益驱动，导致道德风险发生。现金流预测风险是指底层租金的现金流变动对融资租赁资产证券化项目的本金和利息产生直接的影响。

在交易结构设计阶段，主要存在破产隔离风险、信用增级风险、资金混同风险以及信用评级风险。破产隔离风险破产隔离不充分风险是指在主体企业发生破产时，由于基础资产未实现真正的销售而造成的基础资产被回收和清偿。信用增级风险是不充分会给资产证券化产品的偿付带来负面的影响信贷增级措施的缺失。资金混同风险是当资产服务企业的资信状况不断恶化、偿债能力下降、甚至倒闭时，这些资产的回款就会与资产服务企业的其它资金混杂，从而使证券化产品的利息不能得到应有的偿还。信用评级风险是评级机构撤销或降低资产支持证券的现有评级可能对资产支持证券的价值带来负面影响。

在存续阶段，主要存在着差额支付承诺人履约风险、承租人提前退租风险以及操作风险。差额支付承诺人履约风险受差额支付承诺人经营状况和信用风险的影响，可能导致其未能履行差额支付义务，进而影响专项计划向资产支持证券持有人支付本金及预期收益。承租人提前退租风险源于承租人因利益变动或利率调整而提前退租，使得基础资产的资金流与预期不符，进而影响资产质量。环境风险包括了经济、法律和政治环境的不确定性，而操作风险则主要涉及人员、流程、系统和外部事件因素引发的风险。

# 2.3.2风险评估

# 2.3.2.1基础资产构建阶段风险评估

基础资产集中度风险的评估要考察基础资产的数量和分散程度。结合《企业应收账款资产支持证券挂牌条件确认指南》的规定，当基础资产中债务人的数量超过10名，债务人之间不存在关联关系，且单一债务人的入池资产占比不能超过资产池的 $50 \%$ 时，就认为该基础资产的集中度风险较低。

承租人违约风险的评估要考察对基础资产预计回收率和预计回收时间的分析。基础资产预计回收率主要受债务人资信情况、担保情况、债务受偿顺序、所在地区经济和司法环境等因素的影响，其中对于具有抵质押担保的基础资产或本身具有变现价值的基础资产，回收情沉取决于抵质押权的优先顺序，抵质押物的种类及性质，以及抵质押物的变现价值等。

原始权益人信用风险的评估要考察原始权益人的经营能力良好的情况下，要考察基础资产进行的信息披露是否完整，能如实反应对承租人的信用特征，如年龄、收入情况、行业、逾期历史情况等详细信息。

现金流预测风险的评估主要考察管理人披露的各期现金流预测结果和相应覆盖倍数的呈现，分析是否全面考虑各种因素对未来现金流变化可能造成的影响。根据现金流预测和覆盖倍数的测算结果，评价分析其未来现金流是否能够为资产支持证券的偿付提供安全范围内的保障。

# 2.3.2.2 交易结构阶段

对于破产隔离风险的评估，需要深入分析发起机构破产可能对交易产生的影响，并关注基础资产现金流是否会因此遭遇延迟或短缺。因此，在分析时要判断其是否实

现租赁债权与原始权益人的隔离，以及租赁资产与计划管理人的隔离。

信用增级风险的评估可以根据担保类型、抵质押资产类型、企业资质差异等角度进行分析，判断其是否为优先级产品提供足够的信用支持，并是否设计有效的信用触发机制用于缓解潜在的风险事件，并为发生时提供适时的应对措施。

资金混同风险的评估要着重关注是否清晰说明了基础资产现金流自产生至分配给投资人的整个过程，具体涵盖各个账户环节、资金的流入与流出时间、可能涉及的风险以及相应的监管措施，以保证现金流的合理分配。

信用评级风险的评估要考察评级机构是否有可能撤销或降低资产支持证券的现有评级，从而对资产支持证券的价值带来负面影响。

# 2.3.2.3存续阶段

差额支付承诺人履约风险的评估要需要承担差额支付承诺义务的原始权益人或者第三方机构的信用和整体经营管理能力进行评估。

承租人提前退租风险的评估主要考察承租人的回收期限，评估产品的风险暴露时间是否较长，以及行业或者政策等对承租人提前退租的影响。

# 2.3.3风险控制措施

基础资产构建阶段，要建立严格的准入机制。为避免基础资产的信用风险，在检验基础资产的合规性时，要注意对基础资产的质量进行检验，如承租人集中度、承租人情况等等。为防范原始权益人信用风险，需要对原始权益人进行充分的尽职调查，全面评估其持续经营能力。防范现金流预测风险中，在现金流预测中在以历史数据为基础，充分考虑影响未来现金流变化的各种因素进行合理预测，并对现金流进行覆盖测试等，从而为基础资产制定特定的产品计划。

交易结构设计阶段，在评估交易的真实出售和破产隔离的有效性时，需要深入分析发起机构破产可能对交易产生的影响，并关注基础资产现金流是否会因此遭遇延迟或短缺。在信用增级层面，要确保触发机制能起到事先预警的功能，同时管理人应采用差额支付承诺、分层、信用触发机制等内外部增信措施。资金混同风险要考虑现金流归集和偿付安排，涉及多个方面，包括资金的归集与划付以及资产收益的分配等。

融资租赁资产证券化存续阶段，差额支付承诺人履约风险要对差额支付承诺人的经营能力等进行评估，保证可以履行差额支付的义务。承租人提前退租风险应该要提前约定好未按照规定日期退租要进行一定的补偿。环境风险和操作风险要从基本的现金流入手，确保资金充足，并不断追踪和分析未来的政策风险。

# 2.4风险管理的理论基础

# 2.4.1资产重组原理

由于资产证券化的基础资产是构成混合基础资产池，所以可以包含来自同一企业内部或者不同地域及行业的多个企业的多项资产。融资租赁资产证券化的资产重组原理是通过重新组合租赁资产的未来现金流，并进行结构化设计，构建成可形成证券化的资产池，来达成融资租赁公司或金融租赁公司对资产的有效管理和融资的目标。这种资产证券化的形式可以更高效地管理和重新组合多元化资产，为投资者提供更为吸引人的证券产品。

# 2.4.2 风险隔离原理

在融资租赁资产证券化中，风险隔离原理通过设立特殊目的载体（SPV）实现资产的独立法律实体化，将租赁资产与融资租赁公司的其他业务隔离开来。SPV的独立性、资产剥离、独立管理以及破产违约迅速处置等关键机制确保了资产证券化产品在融资租赁公司面临财务困境时仍能保持相对稳定。这一原理的实施有效降低了投资者的信用风险，提高了产品的信用评级，进而促进了融资租赁市场的发展。通过风险隔离，投资者可以更加信任其投资的安全性，即使融资租赁公司整体面临风险，资产证券化产品仍能独立运作，维护投资者权益。

# 2.4.3信用增级原理

在融资租赁资产证券化中，信用增级原理通过对租赁资产支持证券进行信用评级和采取不同级别的信用支持措施，旨在提高产品的整体信用质量，降低投资者面临的信用风险。首先，评级机构根据一定标准对产品进行评级，划分为不同的信用等级，为投资者提供了更清晰的信用状况了解。其次，发行机构采取过渡性储备金、担保和信用保险等增级措施，为不同信用级别的证券提供不同程度的信用支持，创造了分层设计，满足了不同投资者对收益和风险的不同需求。信用增级能够在融资租赁资产证券化过程中有效提高证券的信用等级，降低资产支持证券的风险，确保融资租赁产生的现金流能够保障本息兑付。这一信用增级原理促进了融资租赁行业在资产证券化融资方面的发展，为市场参与者提供了更多选择。

# 2.4.4 信息不对称理论

信息不对称现象普遍存在于市场经济活动中，具体表现为交易双方信息获取的不平等。在传统交易中，卖方往往相较于买方拥有更详尽的商品信息，从而占据优势。在资产证券化的复杂流程中，发起人通常身兼数职，这一特点加剧了信息不对称的效应。在资产证券化的起始阶段，作为原始权益人的发起人出售基础资产时，往往面临“逆向选择”的问题，即原始权益人由于不愿以优质资产池换取较低的价格，可能存在出售质量较低资产的倾向。另一方面，在资产证券化产品的存续阶段，资产支持证券的持有者可能难以全面了解证券的管理状况，这导致资产服务机构、计划管理人等参与机构可能出现不尽职尽责的现象，即所谓的“道德风险”。逆向选择和道德风险都是信息不对称在资产证券化过程中的具体体现，它们共同增加了资产支持证券持有者面临的风险。因此，为了保障市场的公平性和效率，需要采取措施减少信息不对称的影响。

# (4）有效转移风险

诚通租赁在租赁资产业务中存在承租人违约风险，并且还有多笔租赁合同为同一承租人持有，更增加了信贷风险的集中度。为了应对承租人可能违约带来的较大经济冲击，诚通租赁运用资产证券化策略，将其持有的多个融资租赁合同打包成证券，并分散到多个投资者手中。这一策略的核心在于将部分信用风险转移至投资者，以此来降低和分散原本可能集中在诚通的风险。通过这种方式，诚通租赁不仅成功地保障了自身的利益，确保了业务的持续稳定，同时也为投资者提供了一种具有吸引力的、分散化的投资机会，使得整个融资租赁市场的生态更为健康和稳定。

# (4）降低融资成本

传统的银行贷款，尽管为众多企业提供了融资的途径，但其成本也不可忽视，高昂的贷款利率、行政手续费和与之相关的负担组成的实际费用都会对公司的经营造成压力，成为制约企业灵活度的重要因素。在资本市场的动态背景下，诚通租赁对此进行了深刻的洞察，转而选择了资产证券化这一创新融资模式。在全球低利率的大环境下，资产证券化不仅提供了更低的边际融资成本，还显著减少了与融资相关的发行与管理成本。这种策略性的决策不仅反映了诚通对于宏观经济与资本市场的敏感度，更彰显了其对于优化资本结构、提高资金使用效率的追求。利用资产证券化，诚通成功地为自身融资活动注入了更大的灵活性，确保了企业在竞争激烈的市场环境中保持其核心竞争力。

结合以上四个方面的动因，从战略层面看，诚通租赁希望能通过资产证券化打开一个新的融资渠道，不仅仅是为了满足其即时的资金需求，还基于多方面的综合考虑，包括增强资金流动性、优化资本结构、有效地转移风险以及降低融资成本等角度，旨在为公司带来长期的利益，满足其持续发展的需求。更重要的是，这也是其在国内金融市场上建立与投资者关系、提高知名度和信誉的一个有效途径。

# 4 诚通融资租赁资产证券化项目风险管理分析

诚通融资租赁资产证券化项目的风险可以分为基础资产构建阶段、交易结构设计阶段与存续阶段三个阶段。

# 4.1基础资产构建阶段风险管理

基础资产构建阶段包括的风险有基础资产高集中度风险、承租人违约风险、原始权益人信用风险和现金流预测风险。

# 4.1.1基础资产高集中度风险管理

# 4.1.1.1基础资产高集中度风险识别和评估

（1）从承租人所在地区角度

高度集中的地区分布会使资产证券化项目产生风险，因为若该地区出现经济、政治或其他不利因素，将会对资产池的整体表现产生较大影响。评估地区集中风险的标准可以基于前五大承租人所在地区的未偿本金余额占比合计。若前五大承租人所在地区的未偿本金余额占比合计超过 $50 \%$ ，则认为地区集中风险较高；若未偿本金余额占比合计低于 $50 \%$ ，则认为地区集中风险相对较低。本项目基础资产对应的承租人分布于11个省市，覆盖了北方如黑龙江、内蒙古，中部如河南、陕西，以及南方如江西和云南，承租人地区分布较为分散。但是，结合承租人为偿付本金余额看，前五大承租人所在地区未偿本金余额占比合计为 $6 8 . 4 6 \%$ ，其中浙江省的租金余额和本金余额占比最高，分别达到 $2 4 . 9 6 \%$ 和 $2 4 . 7 4 \%$ ，且合同笔数占比也达到 $3 3 . 3 3 \%$

由于前五大承租人所在地区未偿本金余额占比合计为 $6 8 . 4 6 \%$ ，超过了 $50 \%$ ，表明未偿付本金余额的承租人地区集中度风险高，因此从承租人所在地区集中度看基础资产集中度风险，该项目存在较高的地区集中度风险。

# 4.1.3原始权益人信用风险管理

# 4.1.3.1原始权益人信用风险识别和评估

(1）从原始权益人的信息披露角度

在资产证券化的当前阶段，专项计划亦需警惕原始权益人可能带来的信用风险。这一风险首先源自交易各方间的信息不对称，可能导致原始权益人出于利益驱动，利用信息优势以次充好，从而给后续现金流带来显著风险。此外，原始股东还可能出于自身利益考虑，故意隐瞒或提供不实信息。为了在一定程度上缓解这种信用风险，融资租赁资产证券化产品的抽样尽职调查方法被视作一种有效手段。根据《融资租赁债权资产证券化业务尽职调查工作细则》（后简称《细则》），入池资产可以采用抽样尽职调查方法。当抽样方法严格遵循《细则》所规定的标准，且样本资产在地域分布和行业特征等方面与整体资产池保持高度一致性，能够充分展现资产池的主要价值构成，同时具备全面性、重要性、代表性和随机性时，这些样本资产便能够较为准确地反映真实基础资产的情况。尽管有这样的尽职调查方法，但由于未公布样本资产的详细统计数据，原始权益人的信用风险仍然存在。所以在资产证券化过程中，对原始权益人的信用评估和监督仍需保持高度警惕。

综上诚通租赁发布的说明书仅对基础资产进行了简单的披露，对承租人的信用特征，如年龄、收入情况、行业、逾期历史情况等未进行披露，披露存在着不完善的情况，容易产生原始权益人信用风险。

# 4.1.4现金流预测风险管理

# 4.1.4.1现金流预测风险识别与评估

对于管理人披露的各期现金流预测结果和相应覆盖倍数的情况，如果没有全面考虑各种因素对未来现金流变化可能造成的影响，那么说明未来有可能其他因素导致现金流无法及时偿付。同时，根据现金流预测和覆盖倍数的测算结果，其未来现金流如果不能够为资产支持证券的偿付提供安全范围内的保障，也会面临现金流无法偿付的风险。

# 4.1.4.2 现金流预测风险控制

对于影响基础资产未来现金流的测试时以原始权益人的租赁合同损失率及提前还款率的历史数据为基础，进行了充分的压力测试，在压力测试情景下专项计划现金流

均能覆盖优先级本息。

对于部分入池资产的利率类型为浮动利率，而本期证券利率类型为固定利率，入池资产的利率波动可能造成利差缩减的风险，本项目综合考虑了利率波动对优先档证券本息偿付的影响，在加压条件下，基础资产本息回收仍能对优先级本息形成 1.05 倍覆盖；且基础资产池加权平均信用等级为 AAs/AA-s，资产池信用质量较好，可以为专项计划优先级资产支持证券本息的足额兑付提供了有力的保障。

对于基础资产本息流入对专项计划优先级本息覆盖倍数较低的风险，由于本次专项计划基础资产中的融资租赁债权有11 笔设置了保证、质押等担保措施，评级机构在考虑了基础资产对应的担保措施及承租人主体信用，给出基础资产池加权平均信用等级为AAs/AA-s，资产池信用质量较好。

并且，本专项计划设置的信用增级措施包括优先级/次级结构化分层，超额现金流覆盖，差额补足承诺及不合格基础资产赎回。信用增级措施的设置将有助于在出现现金流预测偏差时在一定程度上缓释优先级资产支持证券持有人的利益。

# 4.2 交易结构设计阶段风险管理

交易结构设计阶段的风险管理包括破产风险隔离、信用增级、现金流收付安排和信用评级设计四方面。

# 4.2.1 破产隔离无效风险管理

4.2.1.1 破产隔离无效的风险识别和评估(1）从实现租赁债权与原始权益人的隔离角度

该专项计划的租赁债权和原始权益人隔离机制主要仰赖《资产买卖协议》的签署。通过这一协议，原始权益人将牵涉基础资产的租赁债权的所有权益和法律权利完全转交给了计划管理人中信证券。这一步骤的实施实现了租赁债权与原始权益人之间的法律上的分离，为专项计划的稳健运作提供了法律保障。尽管协议解决了法律权益的转移问题，但由于《物权法》对租赁物所有权的限制，租赁物的实际所有权并未因租赁债权的转让而发生变化。这表明在专项计划的存续期内，原始权益人的破产可能导致租赁物被纳入破产资产，从而对租金款项的正常回收带来潜在影响，因此本项目仍然面临原始权益人破产风险。

(2）从实现租赁资产与计划管理人的隔离角度

在专项计划的体系中，中信证券作为计划管理人的角色至关重要。专项计划设立时的明确规定确保了租赁资产的独立性，就是为了应对可能发生中信证券破产或清算的情况下依然可以维护租赁资产的安全。专项计划期间当中信证券发生破产，专项计划的新管理人将尽快接手，来确保租赁资产得以继续稳健运作。并且为了确保管理的有效性，中信证券对基础资产进行了独立管理，这也使得基础资产受到了更为精细专业的管理。

# 4.2.1.2 破产隔离无效风险控制措施

自专项计划设立日起，原始权益人不再拥有基础资产的任何权利，这些权利已作为专项计划的财产，与原始权益人的个人财产分开处理。这一措施不仅有助于规避法律上的不确定性，而且即使在原始权益人破产的情况下，这些权益也不会被视为其破产财产，实现了有效的破产隔离。

另外，专项计划文件对权利完善事件做出了明确规定。一旦发生任何权利完善事件，原始权益人有责任将基础资产租赁物件的所有权转让给计划管理人，并向承租人、担保人和保险人发出权利完善通知，确保相关方了解租赁物件和基础资产的转让情况。这一规定有助于维护专项计划的权益和稳定运作。

此外，针对差额补足承诺人的长期信用等级下调，专项计划文件通过调整回收款转付期间的长度，灵活应对降级情况，缩短回收款在专项计划收款账户的保留时间，以降低资金混同风险。这些周密的安排为专项计划在面对潜在风险时提供了有效的保护。

# 4.2.2 信用增级不足风险管理

# 4.2.2.1 信用增级不足的风险识别和评估

本项目采用多层次的信用增级机制，旨在确保投资者权益和维护资产池的质量。在内部增信方面，通过优先级/次级结构化分层，这是一种常见的增级方式，以确保本息偿付的有序性，从而在资产池违约时实现损失有序分担。诚通租赁资产支持证券本次分为优先级A1、A2、A3、A4级和次级证券。优先级证券本息索偿次序优先，若产品发生违约导致现金流分配不足，优先级证券将得到优先偿付，次级证券则只提供了有限的信用支持。

在差额支付承诺机制上，综合了内部和外部增信机制的结合，诚通租赁作为原始权益人，担任第一差额补足义务人，提供基础资产现金流与各项费用和优先级本息之间的差额部分的全额差补。这是一种内部增信机制，因为原始权益人直接为资产支持证券提供增信。中国诚通控股集团有限公司作为第二差额补足义务人，对基础资产现金流与各项费用以及优先 A3 级、优先 A4 级的预期收益和未偿本金余额、证券回售价款或回购价款之间的差额部分提供差补。这是作为外部增信机制，为证券提供了额外的增信。

出于对资产质量的考虑，专项计划还设定了不合格基础资产和特殊基础资产赎回机制，确保在专项计划存续期间，及时发现并赎回不合格或特殊的基础资产。信用触发机制在不同等级设置违约事件和权利完善事件，触发后将采取相应措施，包括混同回收款用于偿付优先级资产支持证券的预期收益和未偿本金，以及在权利完善事件中涉及各方的权利转移和必要手续，确保专项计划的顺利运作。

综上所述，在专项计划信用增级方式中，以基础资产产生的现金流作为第一还款来源。通过优先级/次级结构提供综合增信保障，但对优先级资产支持证券本金及预期收益的保障是有限的。如果基础资产现金流入规模大幅下降，可能会给优先级资产支持证券持有人带来损失。此外，差额补足承诺人一的补足事项中包括为其自身的回收兑付义务提供差补，实际并不会起到额外的增信效果，此外，差额补足承诺人二仅对优先A3 级及优先 A4 级资产支持证券本金及收益提供差补义务。因此，本项目将面临差额补足增信效果较弱的风险。

# 4.2.2.2 信用增级不足风险控制

对于差额补足增信效果较弱风险，专项计划调查显示差额补足承诺人一经营稳健，财务状况良好，能为专项计划提供一定的增信效果，此外，差额补足承诺人二也为其提供了较强的增信效果。入池基础资产影子评级等级较高，且同类基础资产历史回款

情况较好，未出现违约情况，触发差补概率较小。

# 4.2.3资金混同风险管理

# 4.2.3.1资金混同风险识别和评估

在本项目所涉的回款流转阶段，资产池所回收的现金与资产服务机构自有资金间可能存在潜在的混淆风险。为了有效管理这一风险，资产证券化产品通常采用一种基于风险层次的分层结构设计，其中，风险水平的核心考量因素涵盖现金流的归集与偿付策略。在进行现金流的归集工作时，必须严谨考量资金混同与挪用问题，以确保资金流动的清晰性和安全性。

在现金流的归集方面，存在诚通租赁开展其他融资租赁业务使用专项计划基础资产的账户进行现金流归集的情况，在基础资产转让之后，承租人仍按照融资租赁合同约定向诚通租赁收款账户划款，存在基础资产现金流与诚通租赁正常经营业务现金流混同的风险。

# 4.2.3.2 资金混同风险控制

资产服务机构将依照季度周期发布详尽的《资产服务报告》，母的是为了周期性披露基础资产回收情况的最新进展。针对本专项计划依据差额补足承诺人的主体长期信用评级，灵活设定了差异化的回收款转付时间节点。一旦差额补足承诺人的主体长期信用等级跌至预设的特定阈值之下，本专项计划将立即提升回收款的回笼频率，以确保资金流的稳定性。即使差额补足承诺人的信用等级随后恢复，回收款转付日的频次亦将维持当前的高频状态，但也有效降低资金混同的潜在风险。为了进一步加固风险防控机制，本专项计划还特别引入了权利完善措施。在差额补足承诺人的主体评级出现大幅滑落的情况下，基础资产相关联的承租人、担保人及保险人将直接负责将应付租金划转至专项计划账户，以确保资金流的直接性和透明性，从而有效缓解资金混同风险。这一机制的实施，将为本专项计划提供更为坚实的风险保障。

# 4.2.4信用评级风险管理

# 4.2.4.1信用评级风险识别和评估

评级机构对资产支持证券的评级是对其预期收益或本金偿付可能性进行一个判断，评级可能会根据未来的具体情况可能产生撤销或降低该证券的评级的情况。一旦出现这种评级调整可能对资产支持证券的价值产生负面影响，并且由于资产证券化产品的多样性和复杂性，不同的评估方法可能会产生不同的评级结果。我国目前长期存在评级虚高的问题，近年来高评级债券违约频发，使得评级公司的外部评级公信力下降。

# 4.2.4.2 信用评级风险控制

在专项计划中要求计划管理人作为一方当事人，确保相关的所有协议、承诺和文件的签署、交付或履行，并已经获得中国现行法律规定的政府审批、许可或完成备案。在资产支持证券信用等级调整（降低）事件发生时，计划管理人需要将及时进行信息披露，并与评级机构及其他相关机构充分沟通，尽量减少因信用级别调整对投资者造成的不利影响或损失。

虽然诚通租赁入池基础资产存在一定的集中度风险，但由于诚通租赁入池基础资产信用质量较为良好，为其提供了证券化的保障。首先，在基础资产池的数据中可以看出，诚通租赁正常类的租金余额与总租金余额一致，说明诚通的基础资产池中的租赁合同，目前没有出现违约或者逾期的情况，这是一个非常积极的信号，证明诚通的信贷管理效果良好。其次，根据诚通租赁内部对于资产水平进行的评级分类，入池资产符合正常类的资产才能进入基础资产池，在一定程度上表明其对于入池资产的客户还款能力具有信心，按期足额支付租金，租赁物运营正常，经营业绩良好，足够保障现金流的偿付。并且，基础资产的担保方式以保证为主，附有保证的基础资产均为企业保证，对基础资产的信用水平有一定提升作用。资产池中共有11 笔基础资产设置了保证金机制，显示出公司对风险控制和资产安全的高度关注，这些保证金比例虽然相对较低，但足以为投资者提供一定程度的资金安全垫，可以在一定程度上降低潜在的信用风险，确保基础资产的稳定回款，同时也增强了投资者对资产池的信心。从资产池承租人的信用等级中看，有大部分高信用等级的承租人，提高资产池的稳定性和抗风险能，基础资产池中，入池资产的加权平均影子级别为AAs/AA-s，中高信用评级的资产占据了绝大部分，这表明资产池的整体质量较好，信用水平也较高，可以做到有效分散相关风险并产生持续稳定的现金流。由于优质的基础资产具有稳定的现金流和较低的违约风险，有助于提升资产证券化产品的信用等级，所以租赁公司可以通过选择优质的租赁物和承租人、建立健全的租赁合同、加强风险管理等方式，提高基础资产的信用水平，从而实现信用增级。

总之在融资租赁资产证券化产品结构设计中，关注和构建资产池中基础资产的信用状况非常重要，该信用状况直接影响产品的评级水平，只有高信誉的基础资产才有助于获得更优的评级，提高产品在市场中的吸引力。而且基础资产的信用质量也直接决定了融资成本，较好信用质量的基础资产可以做到降低融资成本，增强产品的盈利性和吸引力，而且投资者也更倾向于选择信用状况良好的资产，因此，构建具有良好信用质量的基础资产不仅提高了产品在市场上的接受度，还有助于吸引更广泛的投资者。所以通过积极管理基础资产的信用状况，可以降低承租人违约风险，减少投资者的担忧，促使融资租赁资产证券化产品成功发行并取得市场成功。

# 5.1.2引入丰富的信用增级方式

诚通租赁设计并实施了足够且富有多样性的增信机制，涵盖了优先级/次级结构化分层、差额补足承诺、不合格基础资产、特殊基础资产赎回机制以及信用触发机制。诚通租赁提出的不合格基础资产、特殊基础资产赎回机制确保了资产池的质量，一旦计划管理人或资产服务机构发现这些资产，原始权益人有责任并有权利赎回它们，从而减少了不确定性和潜在损失的风险，这种全面的计算方法确保了专项计划不会因赎回这些资产而遭受财务损失。其中，差额补足承诺进一步加强了对投资者的保护，在差额补足承诺机制中，差额支付承诺人一是诚通租赁，也就是原始权益人，中国诚通控股集团有限公司作为第三方公司的外部增信主体，担任第二差额补足义务人是对优先级 A3、A4 承担差额补足承诺。由此可以看出，当内部增信措施都是基于原始权益人以及差额支付承诺人信用状况和财务状况良好的情况下，如果真的因经营不善或者其他原因导致诚通租赁出现经济问题，外部增信机制的第三方差额支付承诺人就可以发挥作用。这里的第三方差额支付承诺人诚通集团实际控制人为国务院国资委，极大地提高了证券化产品的吸引力。同时，诚通租赁采取的权利完善事件的触发为专项计划提供了额外的保障。还有根据差额补足承诺人信用等级的降低，增加回收款的频率，这种方式可以进一步降低现金流回收不足导致的风险。

在资产证券化过程中，多种信用增级措施的综合运用受到信用增级方式的可行性、投资者的接受度以及定价等多方面因素的共同制约。通过采取内外部增信机制相结合的方式，不仅可以提升市场对融资租赁资产证券化产品的信心，降低投资风险，还能增强证券的市场流动性。此外，这种结合机制通过强化信用验证，降低了融资成本，吸引更多投资者参与，促进了融资租赁行业的健康发展。

因此，引入丰富的信用增级方式是资产证券化领域中提高交易品质、降低风险、吸引更多投资者的关键举措之一。信用增级是指通过各种手段提升资产证券化产品的信用等级，从而增加其市场吸引力和投资价值。在引入丰富的信用增级方式方面，可以采取多种策略和措施，包括提高资产质量、引入信用保险、设立子级证券、制定差异化还款计划。设立子级证券是一种常用的信用增级方式。在资产证券化交易中，可以将资产划分为不同级别的证券，如优先级证券和次级证券等。优先级证券享有优先偿付权利，而次级证券则承担更高的违约风险，但相应地可以获得更高的收益。通过设立子级证券，可以满足不同投资者的风险偏好，提高整个交易的吸引力和灵活性。其次，引入第三方担保也是一种有效的信用增级方式。通过引入第三方担保机构，可以为资产证券化产品提供额外的担保和信用支持，提高产品的信用等级。第三方担保机构可以提供信用评级、追索权和支付保证等服务，为投资者提供额外的保障和信心。

# 5.1.3选择可靠的外部增信主体

通常第三方担保方式主要是由发起人的关联公司来进行担保。诚通租赁在外部增信机制的选择上采用了第三方作为差额支付承诺人的方式，通过引入第三方担任差额支付承诺人，诚通租赁旨在提升融资租赁资产支持证券的信用评级，增强投资者信心，以及促进市场对证券的广泛认可。这一机制的主要是通过外部增信主体的介入，对特定优先级 A3 和 A4 进行差额支付承诺，从而弥补潜在的支付差额风险，确保投资者在证券化产品中享有更高的信用保障。

本案例第三方选择了诚通租赁的关联方诚通集团，而诚通集团的实际控股人为国务院国资委，具备较高的信用水平，能够为诚通租赁提供更为可靠的资金支持。但是，尽管诚通集团是比较有实力的大公司，破产的可能性非常小，但是当租赁公司发行证券化产品期数增多时，其第三方担保实力往往可能会不足。这种担保模式不是由其他无关独立第三方提供，因此，破产隔离的作用并不会非常显著。并且诚通集团近年负债增长显著，若集团债务攀升引发流动性危机，将可能引发差额补足义务执行不力的风险，届时本专项计划资产支持证券持有人的利益将会受到不利影响。但是目前根据背景调查显示其盈利水平、多样化的业务领域、与金融机构的良好合作关系、上市公司的融资能力以及健康的营业收入、为其提供了一定的流动性和偿债能力，减小了因流动性危机导致的差额补足义务履约风险。

选择可靠的外部增信主体具有重要的意义和多方面作用。这些外部增信主体通常拥有较高的信用评级和信誉，其介入对提升整体信用质量、增强投资者信心、促进证券市场接受度等方面起到关键作用。首先，外部增信主体的参与可以显著提升融资租赁资产支持证券的信用评级，从而吸引更广泛的投资者，尤其是那些对信用风险敏感的机构和个人投资者。其次，这种外部信用评级的加持有助于降低投资者的信用风险，为投资者提供更为可靠的投资选择，防范潜在的投资风险。此外，外部增信主体的介入拓展了融资渠道，使融资租赁公司能够更为便利地通过资产支持证券获得融资。由于证券得到更高的信用评级，公司能够吸引更多投资者，从而实现融资规模的扩大。同时，外部增信的存在也提高了证券在市场上的流动性，因为外部增信机构的独立、专业评级使得证券更容易在二级市场上进行交易，为投资者提供更多流动性选择。此外，外部增信主体的介入有助于降低公司的融资成本，因为高信用评级的证券更受投资者青睐，投资者更愿意接受较低的利率，从而降低了公司的融资成本。总体而言，选择可靠的外部增信主体不仅对于融资租赁资产支持证券的成功发行至关重要，而且对于形成更加健康、有序的资本市场环境具有深远的影响。这种战略决策有助于提高融资租赁市场的透明度和规范性，促使更多投资者参与其中，最终实现资本市场的稳健发展。

# 5.1.4 持续关注参与主体的履约能力

持续关注相关参与主体的履约能力也是资产证券化交易中非常重要的环节，参与主体包括发行人、担保人、租赁公司、承租人等，他们的履约能力直接关系到资产证券化产品的质量和稳定性。所以对这些参与主体的履约能力进行持续监测和评估，可以有助于及时发现潜在风险，来保障交易的顺利进行和投资者的利益。

本专项计划对于原始权益人的信用风险中，要求计划管理人在后续管理过程中对该风险进行持续关注，确保每年度对原始权益人财务报表等报告文件进行详细披露。同时，还通过持续关注资产服务机构定期披露基础资产的回收情况，关注差额补足承诺人主体长期信用等级变化，如果下降将提高回收款的回收频率，包括采取权利完善机制来应对差额补足承诺人的主体评级下降的情况，都起到了缓释资金混同风险的作用。

监测相关参与主体的履约能力主要是有助于及时发现潜在的违约风险和问题，从而保障投资者的利益。因为投资者通常也会关注发行人、担保人和租赁公司等主体的信用状况，这些主体的违约可能会导致资产证券化产品的价值受损或者无法按时兑付，影响投资者的收益。作为发行方应该做到监测相关参与主体的履约能力，可以有助于保障交易的顺利进行。如果参与主体出现了履约能力不足或者违约的情况可能会影响到交易的进行，导致交易无法完成或者延迟，可以采取相应的措施。通过持续披露相关参与主体的履约能力，主要是投资者通常会关注的参与主体的信用状况和违约风险等内容，可以提高交易的透明度和可信度以及投资者的信心和认可度。

# 5.2 诚通融资租赁资产证券化项目风险管理存在的不足

# 5.2.1 外部措施较弱，较难降低基础资产集中度高的风险

基础资产池的质量是实施融资租赁资产证券化的关键，它的好坏会直接影响到整个过程的各个环节。2018 年沪深交易所发布的指南对融资租赁ABS 作出了详细规定。例如，单个债务人的集中度不得超过 $50 \%$ ，前五大债务人的集中度不得超过 $70 \%$ 。总体而言，诚通融资租赁有限公司的基础资产池由大量小额租赁合同组成，成功规避了单一大额债务人违约的风险。即使某些债务人违约，对整体资产池的影响也较小。然而，该资产池在地区和行业的分散度方面仍存在不足。诚通租赁的前五大承租人所在地区未偿本金余额占比合计为 $6 8 . 4 6 \%$ ，超过了 $50 \%$ ；三大行业未偿本金余额合计占比$8 3 . 7 5 \%$ 。这种集中度风险可能导致某些行业和地区在经济或政策变化时无法保障现金流的按时偿付。

当基础资产集中于某些行业和地区的问题依然存在，导致地区和行业集中度较高，就会造成难以准确判断基础资产池的信用风险，从而并采取适当的风险控制措施，从而影响对 ABS 项目真实风险的识别。此外，外部增信措施相对薄弱。尽管有第二差额补足义务人对优先级 A3、A4 承担差额补足承诺，但这难以有效降低基础资产集中度高的风险。加强信息披露和提升外部增信能力，对于更好地识别和控制基础资产池的风险至关重要。

# 5.2.2原始权益人信息披露不完全，导致信用风险上升

诚通租赁发布的说明书在基础资产披露方面显得较为简略，没有详尽揭示承租人的信用特征，包括年龄、收入状况、行业背景及逾期记录等关键信息，这种信息披露的不完善性易导致原始权益人信用风险的滋生。信息不对称无疑是风险滋生的核心因素。及时并且全面准确的信息披露对于提升债券发行成功率至关重要，它有助于投资者监督管理人将募集的资金投向规定用途，防止资金被挪用至其他目的，进而降低信用风险。然而，在诚通租赁的资产支持证券计划中，计划管理人未能按要求及时、完整地披露相关报告文件，这无疑增加了原始权益人信用风险的可能性。因此，在融资租赁资产支持证券的运作过程中，管理人应当严格遵守信息披露规定，全面并准确地披露所发行项目的相关信息，并对发行项目信息制定严格的披露要求。这将有助于降低原始权益人的信用风险，从而保障资产支持证券市场的稳健运行。

# 5.2.3评级机构公信力不足，无法降低信用评级风险

融资租赁资产证券化的发行流程较为繁琐，涉及多个参与方，在此过程中评级机构所发布的评级报告具有举足轻重的影响，然而本项目存在评级机构可能撤销或调降资产支持证券评级的风险。同时，评级机构间的激烈竞争有时会引发为迎合发行人而降低评级标准、人为提升证券信用评级的现象，这无疑削弱了信用评级机构的公信力，进而带来信用评级风险。尽管我国当前的评级市场已相对成熟，但其运作模式较为单一，主要是发起者付费模式，这在一定程度上限制了评级的客观性和公正性。为了提升评级体系的全面性和准确性，未来可以考虑引入投资人付费的评级模式。

# 5.3结合诚通融资租赁资产证券化项目提出的风险管理建议

# 5.3.1 注重基础资产分散度，丰富底层资产

在选择基础资产时，首要考虑承租人的分散性、地域分布和行业分散性，非集中于特定地区。在基础资产池的选择过程中，必须确保其具备良好的分散程度。若某笔租赁合同金额异常高，应将其与其他金额较小的租赁合同放入同一资产池中。这考虑了基础资产的合理选择，以最大程度降低潜在风险，为证券市场的健康运作提供了稳固的基础。

针对基础资产集中度风险，要增加底层资产的多样性，并进行合理的入池资产配置。若底层资产类型过于单一，当政策环境发生变化或特定行业出现突发事件，很可能导致大量承租人同时违约，进而产生无法按期支付优先级证券本息的风险。为了降低风险，管理人应该实现资产的多元化配置，减少特定因素对整个资产池的影响。丰富底层资产种类意味着在资产池中包含多种类型的资产，这些资产可以是不同类型的租赁合同、债务债权、抵押品等。丰富的资产种类有助于降低投资组合的特定风险，提高整体的风险分散度。为了实现这一目标，资产证券化交易者需要广泛地收集各种类型的资产，并进行全面的市场调研和评估。这可能涉及到不同行业、不同地区、不同类型的资产。

# 5.3.2 完善底层资产信息披露，强化尽职调查

在资产证券化领域，完善底层资产信息披露并强化尽职调查是确保交易的透明度、稳健性和成功性的关键步骤之一。底层资产的质量和特征直接影响到投资者对资产证券化产品的信心和投资决策，因此，充分披露底层资产的相关信息和进行全面的尽职调查对于降低风险、保障投资者权益至关重要。

首先，完善底层资产信息披露是保障交易透明度和投资者利益的关键措施之一。信息披露应该涵盖底层资产的各个方面，包括承租人或借款人的基本信息，以及信用状况、行业背景、经营状况等；租赁或借贷合同的主要条款和条件，包括租金或利息支付安排、还款计划等；资产的物理属性和质量，如房产的位置、面积、年龄等；相关的法律和合规问题，如土地使用权、权属纠纷等。完善底层资产信息披露可以通过以下方式实现：第一，制定明确的信息披露政策和标准，规定需要披露的内容和披露的方式。第二，设立专门的信息披露平台或网站，向投资者公开相关信息。第三，与独立第三方合作，对底层资产进行独立审计和评估，向投资者提供权威的信息报告。

对于第三方参与机构来说，建议在发行融资租赁资产支持专项计划时，参与机构一定要加强对产品的尽职调查。尽职调查应该全面、深入地了解底层资产的各个方面，发现潜在的风险和问题，为投资者提供充分的保障。严格遵守尽职调查的程序和标准，确保所有的调查活动都按照既定的规定进行。并且要加强团队的专业能力和技术水平，提高对底层资产的识别和评估能力。还可以与独立第三方合作，进行独立的尽职调查和审计，确保调查结果的客观和准确性。

总之完善底层资产信息披露并强化尽职调查是确保资产证券化交易成功和投资者利益得到保障的关键。只有通过充分的信息披露和全面的尽职调查，投资者才能充分了解底层资产的情况，做出明智的投资决策，从而确保交易的稳健性和成功性。

# 5.3.3规范信用评级机构，发展再评级市场

为了实现整治金融行业乱象、落实强监管、去杠杆的目标，并防范化解系统性金融风险，近年来监管政策日益收紧。作为金融业的重要组成部分，融资租赁行业也需遵循穿透式监管的原则。传统的融资租赁行业容易产生权益纠纷，尤其是在资产证券化模式下，监管部门对该模式的良性运作起到了关键的引导和督促作用。信用评级机构是融资租赁企业发展资产证券化产品的核心环节，其专业性和客观性亟需增强。由于融资租赁资产证券化产品对于投资者而言是一种新颖的金融衍生品，因此，需要一套完善的信用评级体系来进行科学评估。评级机构对投资者的投资决策有着重要影响，如果在企业信用等级及资质的评定上出现失误，无论是低估还是高估信用等级，都将增加投资者的风险。从资产证券化产品的质量和保护投资者利益的角度来看，信用评级机构的作用尤为重要。除了行业自律规范外，机构对其实施穿透监管也有助于规范其行为。

此外，还应发展再评级市场，鼓励投资者自费聘请独立第三方评级机构对产品进行评级。将传统的“发行人付费”模式转向“投资人付费”模式可以改善评级现象。在“发行人付费"模式下，评级机构可能会为了维护与发行人的关系而放松标准，导致评级虚高，无法反映产品的真实风险。而“投资人付费”模式下，评级机构无法从发行人处获得利益补偿，这将有助于减少国内评级虚高现象，降低融资租赁公司的融资成本。因此，“投资人付费”模式相比“发行人付费”模式，因为没有受到节约成本的限制更能保障投资者利益。这一模式的推广将有效提升评级的客观性和公正性，为市场参与者提供更为可靠的决策依据。

# 5.3.4 提升自身的风险管理能力，健全风险管理体系

诚通租赁在资产证券化过程中，为了获得投资者的信任与支持，采取了一些风险的管理措施。首先，关注到原始权益人面临的破产风险，其采取的破产隔离风险应对措施显旨在保护投资者、专项计划和原始权益人的利益，确保在特定的风险场景下，资产证券化的结构和支付流程不会受到不利影响。专项计划与原始权益人资产的完全隔离确保了即使原始权益人进入破产程序，专项计划的资产也不会受到侵扰。这为投资者提供了一个强有力的保障，减少了对原始权益人破产风险的担忧。权利完善事件的触发机制也进一步強化了破产隔离的效果，避免了原始权益人破产时的司法不确定性给专项计划带来的不利影响。同时，根据差额补足承诺人的长期信用等级调整回收款转付的频率的方式降低了资金混同的风险，而且确保了回收款能够及时用于满足投资者的需求，降低了资金流动性风险。但与此同时，在进行融资租赁资产证券化的过程中依旧存在基础资产池集中度高、差额补足增信效果较弱等不足。

有效的风险控制措施不仅可以确保资金流动性，而且根据不同风险特点提供了针对性解决方案，能够更加迅速和有效地解决风险资产，确保资产负债表的稳定，从而充分利用资产证券化的优势。针对资产证券化过程中可能出现的风险，制定了具体的应对策略，这不仅增强了其资产证券化产品的可靠性，还有助于吸引投资者。正因为涉及多个参与者并存在多重风险，强化融资租赁公司的风险管理能力是至关重要的。短期内，这有助于增加产品的信誉度和防止因现金流不足而引发的发行失败。而从长远角度看，它有助于完善公司的资产创造能力，持续提供高质量的基础资产，并在证券化市场上获得更广泛的认可。

融资租赁企业缺乏有效的风险管理能力是其在资产证券化融资过程中面临的一个主要问题。提高企业的专业化经营和风险管理能力是推动资产证券化健康发展的关键。租赁公司作为资产证券化交易的重要参与者，其专业化水平和风险管理能力直接影响到资产证券化产品的质量、稳定性和投资者的信心。租赁公司需要不断提升自身的专业能力和风险管理水平，以应对日益复杂的市场环境和风险挑战。租赁公司应建立健全的组织架构和管理体系。部分融资租赁企业没有根据行业特点构建适合自身的风险防控体系，而是简单地照搬银行的风险防控体系，忽视了许多特有的风险点。为提升风险管理能力，租赁公司应在借鉴银行风险防控系统的基础上，建立一套符合自身实际发展的风险防控体系，明确各部门的职责和权限，确保专项计划的高效、协调运行。设立专门的风险管理部门，负责监测和评估公司的风险暴露，并提出相应的应对策略。融资租赁企业应制定明确的发展战略目标，提升直接租赁业务的占比，走上专业化经营的道路。积极引进风险管理和专业化人才，建立一支由经验丰富的租赁业务人员、资产评估专家和风险管理专家组成的专业团队。通过这些措施，租赁公司能够有效提升其专业化经营和风险管理能力，从而提高资产证券化产品的质量和市场吸引力，增强投资者的信心，促进企业的长期健康发展。

综上所述，提高租赁公司的专业化经营和风险管理能力是确保资产证券化交易成功和投资者利益得到保障的重要举措。租赁公司应当建立健全的组织架构和管理体系，拥有专业化的团队和风险管理人才，此外还应该加强租赁业务的规范化和标准化管理，加强信息技术支持和数据分析能力，加强合规管理和内部控制，以及加强与监管机构和行业协会的合作，共同推动融资租赁资产证券化业务的发展和进步。

# 6结论与展望

# 6.1研究结论

本文研究的是基于融资租赁资产的证券化产品，这是一种将融资期限与租赁收入期限相匹配的融资方法。近年来，这种资产证券化在我国资本市场迅速发展。本文以理论为基础，结合实际情况，首先概述了融资租赁资产证券化的相关内容，然后通过一个诚通租赁的资产证券化项目详细分析了在基础资产构建阶段、交易结构设计阶段和存续阶段的风险识别、评估以及控制措施。总结得出，融资租赁资产证券化过程中要构建高信誉的基础资产，引入丰富的内外部信用增级方式，展开背景调查选择可靠的外部增信主体以及持续关注参与主体的履约能力，这为促进融资租赁资产证券化风险管理提供了有益的经验启示。同时，根据案例的风险管理过程发现一些不足之处：第一，外部措施较难降低基础资产集中度高的风险；第二，原始权益人信息披露不完全导致信用风险上升；第三，评评级机构公信力不足无法降低信用评级风险。最后，根据诚通租赁资产证券化案例的不足之处，提出以下风险管理建议：第一，注重基础资产池分散程度，丰富底层资产；第二，完善底层资产信息披露，强化尽职调查；第三，规范信用评级机构，发展再评级市场；第四，提升自身的风险管理能力，健全风险管理体系。

诚通租赁这项专项计划的发行为融资租赁企业提供了更多的融资方式，有助于推动融资租赁资产证券化市场的发展。作为央企背景下的优质资产证券化产品，它对整个融资租赁行业的信誉也有积极推动作用，提升了投资者对融资租赁资产证券化产品的信心，促进市场的健康发展。随着市场竞争的激烈，融资租赁资产证券化产品需要不断创新和优化以适应投资者需求和市场变化。诚通融资租赁资产支持证券的发行为市场注入了新的思路和创新，加速了市场的更新与优化。
