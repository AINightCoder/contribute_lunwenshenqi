
# 2小贷资产证券化概述

# 2.1资产证券化的概念

资产证券化（Asset-backed Securities，ABS）指的是原始权益人将该公司某种类型的资产打包后，以该资产的未来可获得的现金流为支撑，通过发行机构设计成证券类投资产品并发行的过程。资产证券化是一种较为新颖的融资途径，对于原始权益人来说，资产证券化可以加快资金周转，在短时间内通过将未来收入抵押的方式获得大量资金；对投资者来说，资产证券化产品也可以带来较为稳定的投资收入。

一般来说，资产证券化的分类是由其基础资产池的资产类型所决定。例如在欧美，最早出现的资产证券化就是住房抵押贷款证券化（Mortgage-backed Securitization,MBS），该资产证券化产品的基础资产大部分都是由住房抵押贷款组成。ABS发展至今，其基础资产的类型也层出不穷，比较常见的有消费信贷和基建收费权等。ABS 产生时间相对较晚，具有增加资金流动性、分散和转移风险等优势，逐渐被各个国家效仿使用。

我国划分的方式与国际划分方法略有不同，国内主要将资产证券化产品分成四种类型，分别是信贷资产证券化（信贷ABS）、企业资产证券化（企业ABS）、资产支持票据（ABN）及保险资产支持计划（保险ABS）。这四类产品各自的特点如表2-1所示。

# 2.2风险管理相关理论

# 2.2.1风险管理的概念

风险管理是指企业在创造、维持和实现价值的过程中，执行企业制定的战略，遵循管理风险的文化、能力和实践，通过对风险的识别、衡量和控制，以最小的成本使风险所致损失达到最低程度的管理方法。

# 2.2.2 风险管理的基本程序

风险管理是分析识别企业或项目在运营中的潜在风险，依据特定的方法对风险进行预测评估，采取针对性的防范措施，并执行风险监测控制手段，使风险最大程度降低其危害的循环管理过程。风险识别、风险评估、风险控制是风险管理的基本程序。

# 2.2.3.1 风险识别

风险识别是风险管理中至关重要的起点。它是在风险发生之前，运用各种系统的手段和方法对潜在的风险进行判断、分类和鉴定风险性质的过程。风险识别可以通过感性认识和历史经验判断，更重要的是依据客观的资料，数据进行分析，归纳，总结，发现各种风险可能发生的损害程度。

# 2.2.3.2 风险评估

风险估测是依据科学的方法或是模型，量化风险的等级和发生概率。风险估测基本流程是基于风险识别的结果，收集大量的相关数据资料，运用定性和定量的方法，估计和预测各种风险发生的概率和损失幅度，为风险管理的后续环节提供可靠数据。并以此决定各类风险是否需要采取控制措施以及何种措施。

# 2.2.3.3风险控制

风险控制是指根据风险评价的结果，利用最优，最有效的措施最大限度地降低各种风险的动态过程。是对风险识别，估量，评价，防范全过程的监视和控制，根据发生的情况，及时修正风险应对措施，保证企业和项目的运营达到预期的效果。

# 2.3小贷资产证券化概念及其特点

# 2.3.1小贷资产证券化的概念

小贷资产证券化，究其实质，应被归在信贷资产证券化的范畴内。通常来说，小贷资产证券化指的是在小贷公司的主导下，通过出售收益权的手段来对公司内部部分小额贷款设计相应的资产证券化结构，并以此推出资产证券化的金融产品以及金融服务。在这一过程中，投资者能够获取的是该小贷企业打包的小额贷款资产中后续带来的稳定的现金流。借助小贷资产证券化的过程，可以对以小额贷款为代表的无法流通的资产进行一定程度地转化，使其以证券等方便交易的市场形式而存在，在这一过程中对小贷资产进行盘活处理，达到弱化融资杠杆的效果。

# 2.3.2小贷资产证券化的特点

虽然小额贷款品种较多，不同产品在目标客群、额度、期限、利率（费率）、还款方式等方面存在一定差异，但总体来看，小贷资产证券化具有资产分散度较高、期限较短、合同利率/手续费率较高等特征。

# 2.3.2.1单笔贷款金额较小，入池资产分散度较高

不同类型的小额贷款在贷款金额方面具有一定差异，但整体上存在单笔金额较小的特征，通常单笔金额在几百元与 50 万元之间，大部分金额都在10 万元以下。入池资产分散度方面，小额贷款 ABS 产品的基础资产笔数通常为几万笔到几十万笔，有的甚至达到上百万笔，单笔金额较低，平均未偿本金余额往往在5 万元以下，资产池分散度极高。

# 2.3.2.2 期限较短

小额贷款的合同期限一般较传统的车贷、房贷等零售类贷款期限更短。截至2021年末，已发行小额贷款 ABS 产品加权平均合同期限均值为 20.22 个月，已发行产品的加权平均剩余期限均值为14.46 个月，整体来看产品期限相对较短。

# 2.3.2.3客群资质分化，合同利率/手续费率较高

小额贷款贷款的收益率一般高于车贷、房贷，这主要是由于这类贷款通常为无担保的信用贷款，具有更高的风险溢价。如果将各产品的名义收益率换算成年化内部收益率（IRR），各类小额贷款的收益率基本都在 $1 5 \%$ 以上。另外，从客户资质来看，小额贷款公司的目标客群多为非银行信用卡持卡用户，如大学生群体、初入职场的白领群体、蓝领群体等。该部分客户群体还款能力有限，而且属于征信“小白户”。因此，小贷资产证券化的收益率要略高于其他同类信贷产品。

# 2.4小贷资产证券化的风险种类

# 2.4.1基础资产相关风险

# 2.4.1.1基础资产质量风险

基础资产质量风险一般有两种类型，一种是信用违约风险，另一种则是提前偿付风险。

其中，信用违约风险指的是借款人由于拒绝履约或无法履约而导致违约，由此使得债权人因此承受相应损失的概率。互联网条件下的小额贷款可借助互联网的手段来招引客户的参与，这些客户各自的质量不一，且贷款的金融、期限以及利息等方面也存在极大的不同，和审核条件更为严苛复杂的银行贷款比较来说，其违约风险相比来说明显更高。一旦互联网小额贷款面临大规模的逾期，尤其是催收之后也很难再保住本金的情况下，资产证券化产品所对应的未来现金流将会在短时间里走向恶化，进而出现难以偿付的局面。

对基础资产所具有的违约风险进行合理防范，是对资产证券化进行风险管理的过程中尤为关键的一环，必须从源头上做好风险管理工作，方能够确保资产证券化产品的顺利偿付。所以，与类似的网络平台相比而言，蚂蚁金服以及京东金融这二者表现出了更为强大的风险管理能力。无论是阿里巴巴还是京东都具备独属于自身的电商平台以及独立的制度体系，因而消费者或商户的各类经济交易活动在平台上都会生成相应的历史记录，在进行小额贷款的放款工作之前可借助此类数据信息来就借款人展开综合评估，判断其是否满足放款条件。从贷中管理的角度来看，蚂蚁金服以及京东在发展的过程中也具备了独立的风险监测体系，其中，阿里小贷便以 A-GDS 预警监测系统为工具，来对客户的资产及信用状况展开密切监控，对各类潜在风险进行专门处理。就贷后环节而言，在处理逾期贷款的过程中，蚂蚁金服和京东金融在一般情况下都会灵活地展开贷款催收活动，以此尽可能实现损失的最小化；除此之外，平台还可采用其他手段来提高违约行为的成本，包括暂停网点营业以及对个人账户进行冻结处理等。

提前偿付风险具体指的是借款人违背事先的还款约定提前交付应还的款项，受此举的影响，资产证券化产品所对应的未来现金流会出现相应的波动和偏差，因而投资方可能会面临一定程度的损失。如果市场利率下调到比贷款合同利率更低的水平，借款人能够以更低水平的贷款成本来展开筹资工作，其便会按照现行利率标准再度展开贷款，并借此填补先前高利率条件下的贷款数额。具体而言，提前贷款的活动可能导致资产证券化出现以下三种常见风险：第一，借款方做出提前偿付行为之后，本金将面临被提前偿清的结果，进而产生利息损失，使得资产池内的资产总额因此缩水；第二，借款方做出提前偿付行为之后，前期现金流会因此加重，后续的现金流将会因此受到波及，进而使得对现金流所作的预测出现一定程度的偏差；第三，对资产进行提前偿付后，由此便进入循环购买之后的其他环节，面对如此情况，如果市场利率不断下跌，同时又没有找到契合标准的新入池资产，那么可能会出现再投资之后收益水平不比之前的投资风险。

在推行资产证券化产品的过程中，对于提前偿付所导致的收益不足的风险，原始权益人需要对借款人的身份信息进行记录并管理，以数据库的形式对其还款情况展开专门评估，据此判断其体现还款的概率。要想尽可能降低损失水平，一些小额贷款企业会推出相应的条约，具体而言，某公司推出的“拿去花”系列贷款便作出了专门的合同说明，即用户在做出提前偿付行为的情况下，依旧要承担所有的服务费。不同于原始权益人，计划管理人则应当结合已有的历史数据搭建相应的偿付模型，据此对后续的现金流展开针对性预测。现阶段，发达国家早已具备独属于自身的提前偿付模型，尤以美国高盛的提前偿付模型等为代表。

# 2.4.1.2 现金流预测风险

为保证资产证券化产品方案的可行性，需要在产品还未发布时对资产证券化产品的基础资产池进行量化的评估，从而判断该基础资产未来的现金流是否能够满足资产证券化的条件，是否存在风险。而在对小贷资产证券化产品的基础资产池进行评估时比较重要的几个指标包括违约率、早偿率以及回收率等，这些相关指标都可能对未来现金流的波动带来不同程度的影响。加之互联网小额贷款方面的业务发展历程比较有限，样本数据规模较小，因此在展开定量分析的过程中，实现的假设以及参数数据在部分情况下可能会和真实情况表现出一定的偏差；还应指出的是，互联网小额贷款领域现阶段的发展还欠缺稳定性，但凡出现突发性质的事件都可能会影响行业发展的稳定性，这就使得对未来现金流的波动态势展开预测时需要面临一系列的不确定性因素。这就需要计划管理人专门对现金流展开精确的压力测试，在资产违约率持续攀升之际，对现金流的波动状态进行密切关注。尤其循环放贷作为一种全新交易结构的出现，导致新入池的资产在运作过程中难免存在各类不确定因素。如果实际情况和事先的预测之间有着较大的落差，实际偿付水平也会随之呈现出落差。

# 2.4.2原始权益人相关风险

# 2.4.2.1原始权益人破产风险

小贷公司从其性质上来看属于原始权益人，此类企业只有保证自身运行和发展的稳定性，方能够为资产证券化产品的偿付提供重要的基础。倘若互联网小贷企业由于自身发展的不合理导致一定程度的经济损失乃至走向破产，那么小贷资产证券化的体系便会因此走向崩溃，在此情形下，证券的买方必然会因此承受一定程度的损失。通过资产证券化这一名义加进的资金一般来说不会被置于表内，这就使得互联网小贷企业做出超杠杆放贷的活动，因而面临着极高水平的风险。2017 年，根据国内“现金贷”的实际管理需求，我国专门推出《关于规范整顿“现金贷”业务的通知》，强调对于网贷企业来说，其以资产证券化这一金融活动的名义融得的所有资金要和表内的融资数量进行合并处理。因为相当一部分企业前期推出了一系列的资产证券化产品，一些企业难以顺应监管层面的具体需求，倘若企业后续难以实现注册资本规模的扩大，那么可能会因为过高水平的杠杆率而接受来自监管部门的整顿。所以，精准定位互联网小贷企业运作过程中的财务风险可谓十分关键。

在评估企业财务破产风险时，要根据报表信息，分别从盈利水平、偿债水平、营运水平以及现金流等多重维度出发来判断企业的财务状况。通常而言，较高的盈利水平、合理的资产负债结构以及丰富的现金流能够有效避免公司因此陷入财务危机。对此，不仅要重视企业的财务情况，还应当从未来的行业前景、企业战略以及产品竞争力等维度出发，对公司的运营态势进行全方位地评估，据此判断其后续出现财务危机的概率。

在资产证券化的过程中，规避原始权益人破产风险的最通用方式是推出一系列合理的规章制度从而能够将资产证券化产品的基础资产池与原始权益人进行破产隔离，并在发行过程中实时监控原始权益人的信用水平。若原始权益人本身在履约上的能力有所下降，在此情形下后一机制的存在则会以动态调整偿还顺序的手段，将优先级证券置于前列，先对其展开本息的兑付工作。就目前国内已推出的法律政策来说，专门面向资产证券化产品而推出的破产隔离制度目前还未推出相应的强制性约束。

# 2.4.2.2原始权益人其他相关风险

原始权益人的道德风险，指的是原始权益人可能会在就特定贷款进行转让处理的时侯，会出现权属模糊或信息失真的现象，从而之后的各个环节都会因此受到影响。而在互联网领域所展开的小贷资产证券化工作，不管在现金回收环节还是循环放款环节都会在不同程度上涉及各类主体，包括原始权益人以及第三方支付机构等。在这一过程中，考虑到自身的实际利益，互联网小额贷款企业在部分情况下可能会和相关利益主体展开共谋，做出违规操作的行为，进而带来一定程度的损失。所以，对于管理人来说，要积极打造相对健全的基础资产监控体系，对贷款质量进行专门监督，结合合同细则展开针对性地管理。

原始权益人业务混同风险。在互联网运作的小额贷款公司一般来说有着极为复杂多元的业务内容，不仅产品本身在类型上极为多元，并且借款人的覆盖面也极广。在特定的存续期范围内，原始权益人主要负责对资产证券化过程中的小额贷款的动态监控以及资产管理工作，倘若原始权益人所展开的内部操作行为欠缺应有的规范，那么可能会因此和其他小额贷款产品之间相互混淆。所以，在对各项资产支持计划所对应的小额贷款进行管理时，应当将其和其他类型的贷款资产进行分类管理；但从技术角度来看，要在企业 IT 系统内展开精细的数据区域划分处理，对各类资产支持计划之下所对应的小额贷款数据进行各自的存储。在这一过程中，各机构则需要密切关注小额贷款的各项指标波动，对其展开实时的监督以及管理。

# 2.4.3资产证券化产品相关风险

# 2.4.3.1利率风险

利率风险是指由于未来市场利率的不可预测性，存在一定的可能使得资产证券化产品的利率与未来市场利率之差变化较大从而影响产品的收益水平。市场利率水平在持续攀升之际，债券所对应的市场价格会不断走向下跌。但对于资产证券化产品来说，其在交易市场上所呈现的预期收益水平基本上处于较为稳定的态势，同时该类产品持有期相对金融市场的大多数产品来说属于期限较长一类，所以会存在未来市场利率攀升而导致投资者的相对收益率水平下跌的可能。因为投资者在这一过程中的利率风险水平，相对于同期限条件下的信用债券来说，资产证券化产品通常会划定更高水平的利率。

# 2.4.3.2信用评级风险

考虑到信息不对称现象的存在，且互联网小额贷款资产证券化产品本身在资产结构上的复杂性，投资人员要想实施精准的风险评估可谓难度颇大，因而信用评级机构所出具的评级结果通常被视为重要的投资参考。但是，基础资产运作过程中的质量水平持续下降，或资产的原始权益人存在明显危机的状况之下，相关信用评级机构可能会下调该资产证券化产品的评级，从而影响该产品在资产证券化二级市场的交易价格。但凡出现评级下降的现象，计划管理人便有必要与评级机构展开有效沟通，同时展开实时的信息披露工作，最大程度地削弱其消极影响，进而保证投资人员判断的稳定性。

信用评级机构由于本身的收入来源的直接影响，所以可能会与负责资产证券化的发行机构之间存在利益纠葛关系，为抢占业务，前者在部分情况下更倾向于降低评级标准，其极有可能对市场投资人员带来不同程度的误导。举例而言，在美国爆发的次级贷款中，其中 $7 5 \%$ 左右的贷款都顺利实现了AAA 的评级，而此次评级结果中在 BBB及以下程度的资产规模只有 $7 \%$ 。信用评级机构给出的评级水平过高，可能会导致刺激贷款在市场上的需求规模持续扩张，进而导致一系列的风险。次贷危机爆发后，欧美监管机构提高了对信用评级机构的官方审查标准，且完善了已有的监管体系。和欧美各国比较起来，中国的信用评级工作开始得相对来说比较晚，历经二十多年的成长历程，现阶段大公国际、联合评级以及中诚信等各类评级机构渐趋成为这一新兴行业的代表。但是，相比于以标准普尔以及穆迪为代表的全球知名机构来说，这些国内机构与之尚且还存在较远的距离。其一，国内机构现行的评级手段以及监控制度还有待进一步完善，评级体系不够健全，并且发展经验也明显不足；其二，国内评级机构在运作的过程中和发行人之间有着更为紧密的利益互动，就国内债券市场上的具体评级结果来说，AA-级以及超过这一级别的评级结果占据 $9 8 \%$ 以上的比例，哪怕是存在违约情况的债券，在发行的过程中评级结果也都超过了 $\mathrm { A A ^ { - } }$ 。除此之外，现阶段国内市场上的违约债券在规模上比较小，违约率尚且不到 $1 \%$ ，这一数值和整个的市场规模来说微乎其微，因而评级机构出具的评级结果很难接受来自自由市场的真实检验。

# 2.4.3.3产品流动性风险

现阶段，国内证券市场上推出的资产证券化产品大部分都是通过一级市场进行发行，虽然确实存在资产证券化产品的二级交易市场，但目前该二级市场的交易活跃度极低。

就国内资产证券化产品二级市场的发展动态来说，其流动性不足的原因具体在于三点：其一，现阶段国内推出的ABS 标准化水平不高，单只产品的推出规模相对比较有限，各类产品无论是基础资产规模还是现金的流动性程度上都有着极大的差异，这就使得二级市场的资金流动面临着更高的难度；其二，就资产证券化的二级市场来说，信息披露程度不足、评级体系有待进一步完善，这使得投资者受困于信息的不对称而无法进行更加理性的投资，进而降低了市场的活跃度；其三，资产证券化产品的一级市场和二级市场都对一般投资者来说门槛过高。

从资产证券化产品的角度而言，其在二级市场上的流动性水平不高，并欠缺应有的交易对手，这就使得投资人员会倾向于做出更高水平的利率补偿要求，导致一级市场发行利率水平偏高，企业融资过程中的成本也随之提升。另外，对于资产证券化产品的交易和流动过程来说，在面临收益损失等情况时，投资人员也无法再短时间里对持有产品进行变卖处理。最近几年，国内相关机构以及发行公司都在努力作为，不断提高资产证券化产品在二级市场上的流动水平，具体策略有刺激做市商的参与动力、回购资产证券化产品、优化报价系统运作效率等。

# 2.4.4其他外部风险

# 2.4.4.1 行业政策风险

目前，中国面向互联网金融领域所出台的监管政策大部分都聚焦于 P2P 网络借贷领域，除此之外的其他领域现有的政策法规则相对比较匮乏，特别对于部分创新型的金融板块来说更是如此。法律层面的空白导致了两大问题：其一，目前国内互联网领域所展开的小贷资产证券化工作各方面的门槛水平都比较高，并且业务审批的难度也不容忽视。就发行环节来说，大型互联网金融集团负责大部分发行工作，由此挤占了中小型企业的发展空间，此类企业很难在互联网小额贷款资产证券化产品方面获得官方审批。另外，审批周期过长的同时，审批流程也极为繁琐，举例而言，京东白条在展开资产证券化工作之际，单就审批工作便花费了八个月；其二，就已实现上市的产品而言，现阶段的信息披露机制明显有待进一步健全，且信用评级水平大部分偏高，其有效性程度也有待展开更进一步地验证，一般投资人员很难对产品的风险进行综合判断。除此之外，国内资产证券化二级市场总体的成长态势尚且称不上成熟，不管是公开报价系统亦或是撮合机制都尚且处于空白状态。且专门面向互联网金融领域的监管工作尚且处于起步状态，各类机构正积极推出各类政策发挥来对该行业的运作动态进行规范，因此政策风险是推行互联网小贷资产证券化过程尤为关键的影响因素。一旦监管机构决定以更为严苛的立场来对互联网金融领域和创新业务展开监管，那么资产证券化产品在此情形下可能会变得不符合规定，不仅收益水平会明显下降，且严重情况下可能会因此走向违约。所以，原始权益人以及计划管理人需要做好实时监管工作，根据现行的法律法规来展开各项审批工作，不断健全已有的业务流程，从而确保业务运作的稳定性和持续性。

# 2.5资产证券化的理论基础

从资产证券化这一活动的基本原理来看，可将其理论基础归纳为三种，分别是资产重组、信用增级以及风险隔离。其中，资产重组原理指的是基础资产分别从风险和收益两个层面出发进行重组，从这两重维度出发来就现金流展开研究；信用增级原理则指的是对基础资产进行合理管理，以更高的信用级别水平来提高发行证券所对应的信用级别，以此提高证券所具有的市场价值，同时帮助发行人实现更高的信用水平，以此使证券所具有的信用级别能够与投资者的实际需求之间进行匹配，主要从信用等级的维度出发来就现金流展开研究。资产证券化要想实现顺利地发行，那么尤为关键的三项基本条件为：其一，对基础资产展开优化重组，其二，隔离资产值以及其他资产可能潜在的风险因子，其三，通过信用增级的手段来对资产池加以管理，以此拉近实际资产与现实需求这二者间的距离。

# 2.5.1资产重组理论

所谓的资产重组，也就是对基础资产展开一定程度地拆解以及重组，通过这一过程来对资产结构加以优化。但需要指出的是，并非一切基础资产都能够被纳入资产证券化的过程，通常需要对资产首先展开质量检验之后，排除质量明显不符合条件的资产之后，将剩余的资产纳入到证券化的过程中去。

在展开资产重组工作的过程中，尤为关键的便是对资产所作的筛选。一般来说，资产重组工作要面临三项条件：其一，拆解资产，以此尽可能控制资产集中度过高而导致各资产相互影响的潜在风险。其二，扩大基础资产的规模，降低融资成本。其三，关注可以创造未来现金流的资产。一般来说，主要是通过特定的金融工具来对其中的特定资产进行提取处理，从而完成资产重组的工作，此类资产往往可以带来一定水平的未来现金流。

深究资产重组背后的原理可知，其主要是对特征比较接近的资产进行重组，在这一过程中延续单笔资产所具有的特征，以此丰富资产池内资产类型的多样性，同时控制单笔资产可能产生的整体风险，通过控制风险的方式来确保资产集合之后带来的整体收益。

对于资产的现金流来说，其在现实中往往存在三种偿还形式，分别为提前偿还、延迟支付以及无法偿还。根据大数定律的一般原理，就历史数据进行深入分析后，结合资产所呈现出的现金流特点，可以计算出平均现金流的规模和现金流产生时间。在资产重组的过程中，贷款所呈现的流动性程度也会随之发生改变，以往相对稳定的债务关系在此情况下也可能发生转变。

# 2.5.2 风险隔离理论

破产隔离具体指的是在原始权益人的作用下，在资产证券化的过程中，以“真实出售”的手段对其中的基础资产进行让渡处理，将其转移到特定的机构（SPV）中，对这些资产所具有的风险以及收益进行一同转让。破产隔离可借助法律以及合同的形式对 SPV 所具有的权利义务展开约定，以此确保其相对于资产发行人以及管理人来说的独立地位。

结合风险隔离理论可知，可借助破产隔离的手段来防止由于发行人自身走向破产而导致的各类产品风险。通常而言，产品风险大部分都来自于基础资产，其本身有着风险隔离的特性。因此，对资产证券化的过程展开系统的风险评估，在某种程度上来说也就是就基础资产所展开的风险评估活动，这一过程中对产品发行人不需要再展开既定的信用评估工作。

资产证券化无需受到发行人本身信用评级结果的影响。归根结底，资产证券化的操作指的是对基础资产所对应的现金流展开专门的拆解处理，将其转化为更便于出售的一种市场化的证券形式。其中的关键主要有两点：第一，能够预期的未来现金流；第二，风险隔离。

# 2.5.3 信用增级理论

信用增级理论即通过一系列手段来提高该资产证券化产品的信用水平，提供相应的担保使得投资者的投资风险得以降低，进而能够更加吸引投资者的关注。信用增级有两种基本形式，一种是内部增级的形式，另一种则是外部增级。其中，前者指的是对基础资产所对应的现金流展开抵押、打折以及结构的优化等操作，以此实现更高的信用级别。后者则指的是借助第三方的力量来就产品展开担保，在这一过程中实现信用增级的目的。

在信用增级的过程中，部分信用等级原本有限的基础资产可以在这一过程中实现资产信用的提升以及增补，进而对资产证券化产品在信用上进行有力地补足，以此实现融资成本的有效降低。所以，在推出资产证券化产品之前，一般都会就基础资产展开相应的信用增级操作，以此实现更高的信用质量，更好地争取来自投资者的关注，进而实现市场价值的有效提升。
