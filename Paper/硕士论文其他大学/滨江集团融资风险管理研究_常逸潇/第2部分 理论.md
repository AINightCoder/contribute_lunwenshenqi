
# 第二章 概念界定及理论基础

# 2.1概念界定

# 2.1.1融资

融资可以为企业筹集必要的资金支持，这些企业通过发行股票等方式向投资方筹集资金或向金融机构和个人通过借款的方式获得资金，以满足企业日常经营管理活动的资金需要。融资行为贯彻企业运行始末，企业所筹集的资金不但可用以满足自身经营活动的资金需求，同时筹集的资金还可用于投资或者清偿企业债务等。

企业的融资渠道主要分为内源融资和外源融资两种。内源融资包括企业内部留存收益可供企业自由使用的资金或者向企业内部员工进行筹集的资金。总体来说，企业内源融资风险较低，让员工与企业收益共享、风险共担，能够调动员工的积极性，但是筹集的资金有限；外源融资主要是向外部金融机构或投资者进行融资，风险较高但能解决企业大部分的资金需求，目前企业筹集生产经营所需的资金主要来源于外源融资渠道。

因此，资金是企业发展的基础，筹集大量的资金对企业来说并不是一件容易的事情，一方面企业需要考虑如何获得资金，另一方面企业还需要考虑资金的使用效率。企业融资既需要遵循国家法律法规合法筹集资金，也应该合理地根据自身发展的要求、资本规模、企业的经营目标和流动资金需求量合理地选择融资的渠道和方式，确保资金的有效利用，保障企业的平稳运营。

# 2.1.2 融资风险

风险指的是一种可能导致企业的经营状况脱离预期，从而对企业产生不利影响的不确定性。其来源既有可能是内部因素，也有可能是外部因素。融资风险指的是企业为维持正常运营或扩大规模进行融资活动时，由决策不当导致无法达到预期目标，出现不利于企业的经营发展的隐患，甚至破产。

影响企业融资风险的因素可能有：融资结构、利率水平、经济环境以及融资成本等。通常而言，经济环境、利率水平等宏观融资风险是无法通过企业的决策等有效措施进行控制的，属于不可控风险，企业只能通过相应的措施分散风险。而由融资成本过高、融资结构不合理等内部因素导致的融资风险可以通过有效措施进行控制，属于可控风险。因此企业应重点关注此类因素。

融资风险具体表现为三种情况：第一，企业融资失败会导致资金链断裂，影响信用和融资能力，同时影响员工工资和供应商款项支付，严重时可能面临破产危机。第二，举债经营时，无论经营状况如何都需要按期支付利息和偿还本金，经营状况不佳时可能产生支付风险。第三，向股东融资需支付股利，市场价值下降或利润损失可能导致不能支付满意股利，影响下一轮融资。

# 2.2理论基础

# 2.2.1 风险管理理论

风险管理是指运用计划、组织和控制等方法有意识地降低风险的损失的活动。1931年，学者霍普钠首次提出了风险管理的概念，在早期风险管理理论具有一定的局限性，单纯只关注企业的安全问题，对于具体的运营过程所产生的风险没有涉及。随后，风险管理理论在早期理论的基础之上，进一步关注企业的内部控制问题，对于公司的财务管理和日常运行有了更多的关注。近年来，风险管理理论开始融入到国家战略之中，人们认识到必须要引入国家层级或高级第三方监管机构。很多国家都在这一时期发布了政策文件，将对于不同企业的风险管理进行规定。

房地产行业是我国经济构成的重要组成部分，有其特殊性，其风险管理应该包含分析与决策的过程。结合房地产融资风险类经营活动，其风险管理是指房地产企业在面临融资每个环节所带的风险时所采取的措施。房地产企业的融资风险管理大致分为三个部分，风险识别、评估与控制。风险管理的第一步是风险识别，国家政策作为非市场影响因素对企业的风险管理决策存在重要作用与反作用。风险评估是风险管理的第二步。通过分析数据，采用熵值法、功效系数法，相对准确地评估融资风险，最后通过分析结果，可以提出响应国家政策、符合当前市场情况的绿色风险控制对策。

# 2.2.2 优序融资理论

优序融资理论也被称为“啄食顺序理论”，其研究对象为公司的资本结构。这一理论认为公司在计划新项目融资时，会优先使用公司内部的融资作为来源，如果金额不足会利用债券进行融资，而股权融资是最后才会被公司考虑的融资方式。在考虑所得税因素时，负债越多企业的财务杠杆利益越大，企业的价值也就越高。

企业价值确实可以依靠提高负债来完成，但随着负债的增加其效果也会随之变差，而企业所要付出的成本则会增加，所以企业需要使其债务保持在一个合理的范围之内，保持一个最优资本结构。而企业的最优资本结构情况是何种状态，存在动态权衡理论与静态权衡理论两种不同的理论体系。信息不对称理论也是优序融资理论的理论基础之一，该理论认为在市场中，各方所握的信息是存在差异的，掌握信息质量较高的一方会获得竞争优势，从而比竞争对手处于更为有利的位置。该理论证明了市场机制存在缺陷，需要政府进行一定程度的干预。对于企业来说，保持在一个相对合理范围内的最优资本结构是优序融资理论的核心，在企业融资过程中，优序融资理论保障了企业融资结构的合理，股权融资与债券融资相结合的方式，通过合理的结构从而实现提高价值。

因此，特别是对于如房地产领域的大型企业、成熟企业，优序融资理论的表现更为明显，遵循内源融资优先级大于外源融资，债权融资大于股权融资的原则，有助于企业达到资产结构的最优解。目前，优序融资理论属于金融行业普遍认可的理论，企业融资实践中反复印证了该理论的实用价值。

# 2.2.3 资本成本理论

1890 年学者Bohm Bawerkd 在其文章中首次提出资本成本理论，其从企业利息的角度分析，认为在企业运营中为了获得融资，企业需要付出一定的成本，这些成本就是资本成本。资本成本往往会超过资本的价值，可以视为一种利息，即获得这些资金所需付出的利息。该理论 20 世纪 80 年代被引入我国，钱法仁教授最早引用了这一概念，认为企业通过借债或募股等方式获得资金来源时所付出的代价就是资本成本。这种资本成本在交易过程中以股票分红和债权人股息等形式出现，在财务管理中资本成本是常用的衡量企业融资能力的指标。资本成本是评估企业价值创造状况和经营业绩的指标，目前其已广泛受到全世界业内人士关注，同时资本成本也是评估企业融资水平的重要指标，更少的资本成本代表着更高的收益。对于房地产企业来说，在企业的融资风险管理的评估过程中，加权平均资本成本可作为企业融资水平的指标来体现，过高的加权平均资本成本会提高企业的融资风险。
