# 风险管理在建筑工程安全生产中的应用研究

# 张学仕

摘要：建筑工程安全生产是建筑行业发展的重要基础，也是保障人民生命财产安全的重要任务。但由于建筑工程的特殊性和复杂性，施工现场往往存在着诸多安全风险，给施工者和环境带来巨大安全隐患。因此，如何有效管理和控制风险，提高建筑工程的安全生产水平，成为当前亟待解决的问题。基于此，该文通过梳理建筑工程安全生产中管理风险的类型，分析风险管理在建筑工程安全生产中的应用困境，提出建筑工程安全生产中应用风险管理的具体路径。

关键词：风险管理；建筑工程；安全生产doi：10.3969/j.issn.1672-2167.2024.11.026

目前,风险管理在建筑工程中的应用愈发深入,其可帮助建筑企业识别、评估、控制和监测风险,从而减少事故的发生,保障建筑工程的安全和质量。然而,目前建筑工程安全生产中的风险管理还存在一些问题,如风险评估方法不够科学、风险控制措施不够有效等。因此,建筑企业应进一步加强对风险管理的研究和应用,提高风险管理的水平和效果。

# 1 建筑工程安全生产中管理风险分类

# 1.1 安全管理风险

建筑工程安全生产中经常面临着安全管理风险,具体表现在以下几方面：第1,建筑工程项目的安全管理需要有一个完善的组织机构和管理体系,如果项目组织管理不到位,缺乏有效的安全管理措施和制度,就会增加安全事故发生的概率。第2,人员管理是安全管理的重要环节,如果项目中存在人员素质低、技术水平不高、工作经验不足等问题,会增加安全事故发生的概率。第3,建筑工程项目中使用的各种机械设备,如果管理不善（如设备维护不及时、设备老化、设备出现故障等）或存在缺陷,则会增加安全事故发生的风险。

# 1.2 技术管理风险

技术管理风险包含设计管理风险和材料管理风险两类[1]。设计管理风险指的是建筑工程设计不合理、存在缺陷,或者设计人员缺乏相关经验和技术,从而增加安全事故发生的概率。材料管理风险是指建筑工程项目中使用的各种材料,如果管理不善或存在质量问题,会增加安全事故发生的潜在风险。常见的材料管理问题包括材料质量不合格、材料存放不当、材料使用不当等。

# 1.3 外部环境风险

外部环境风险是指建筑工程项目所处的自然环境、社会环境和经济环境等因素对安全生产的影响和威胁[2]。从自然环境风险来看,恶劣的气候条件（如台风、暴雨等）会导致施工现场存在水灾隐患,而不稳定的地质条件（如地震、地质滑坡等）可能引发土方工程塌方、坍塌等安全事故。社会风险指的是政策法规和社会文化等因素,诱发施工工艺和管理方式的调整,增加安全事故发生的风险。经济环境的影响一般表现在资金链条存在缺口,可能导致施工工艺和材料的选择受限,增加安全事故发生的风险。

# 1.4 人为主观风险

人为主观风险常常是由工作人员素质低下和不当行为造成的[3]。建筑工程安全生产要求施工人员具备专业的技术水平、安全意识和职业道德等,如果技术水平不高、安全意识不强、职业道德缺失,将会导致施工操作不规范,增加安全事故风险。此外,不良的行为方式也可能导致施工人员凭主观经验办事,忽略安全生产的重要性,进而诱发恶果。

# 2 风险管理在建筑工程安全生产中的应用困境

# 2.1 与风险管理相适应的安全管理体系不健全

在建筑工程安全生产中,风险管理工作是建立在健全的安全管理体系基础之上。但目前风险管理在实际应用中往往面临着安全管理体系不健全的困境。一些建筑企业虽然制订了安全管理制度和规章制度,在实际操作中却存在制度不完善、执行不到位的问题。从制度本身来看,还不够全面,未能涵盖所有可能的安全风险和应对措施。同时,这些制度往往过于复杂或简单,难以实际操作和执行。此外,部分人员在制订制度时缺乏对实际情况的充分了解和考虑,导致制度与实际操作脱节。

# 2.2 现场管理执行力度不足

在风险管理应用中,现场管理执行力度不足是常见的问题之一。一些建筑企业虽然建立健全了施工现场组织管理制度,但在实际施工现场,这些制度因管理人员忽视或者工人不遵守而形同虚设。一方面管理人员对安全管理重视程度不够,加之其对安全管理制度认识不足,导致在实际施工现场无法有效地实施安全管理让制度成为了纸老虎、稻草人；另一方面,部分工人未按照安全管理制度开展现场施工,导致安全管理职责落实不到位。此外,安全设施的设置和维护也会因成本原因而大打折扣,存在安全隐患,无形中影响风险管理的实际运用效果。

# 2.3 技术支持不足

在建筑工程安全生产中,风险管理的应用同样离不开技术的支持。但由于一些建筑企业在技术支持方面的投入不足,对风险管理的应用造成掣肘,导致实践中无法及时发现和处理施工现场的安全风险：

1）从技术支持人员方面来看,技术人员负责提供技术指导,确保施工过程中的安全操作,但部分企业对技术支持不足,导致技术人员缺乏专业培训,无法有效地应对施工现场的各种安全风险。

2）从技术设备层面来看,安全监测设备对发现和处理施工现场的安全风险至关重要。这些设备可以实时监测施工现场的温度、湿度、有害气体浓度等关键参数,及时发现潜在的安全隐患。但目前,部分建筑企业对安全监测设备投入不足,导致设备老化、性能下降,无法有效监测施工现场的安全风险。

3）有些建筑企业也会因为缺乏对新技术的应用和推广,导致无法利用现代科技手段提高施工现场的安全管理水平。

# 2.4 安全风险识别与预警机制不完善

风险管理在建筑工程安全生产的实际应用中,需要有配套的安全风险识别与预警机制。但目前,多数建筑企业对此并未给予足够重视,主要表现为：第1,风险识别是安全风险管理的基础,需要全面考虑所有可能的安全风险因素。但在实际操作中,风险识别往往存在盲点,导致无法全面识别所有潜在的安全风险。第2,风险评估和分类是确定风险优先级和制订应对措施的关键步骤。但在实践中,多数建筑企业设计的风险评估和分类往往缺乏科学性,从而导致评估结果不准确，无法准确反映风险的严重程度和影响范围。第3,风险预警机制是安全风险管理的重要一环,但在实际应用中存在因预警不及时,导致工作人员无法第一时间发现和应对安全风险的问题 [4]。

# 2.5 法律监督不力

法律监督不力是影响建筑工程安全生产风险管理的重要因素之一,这一困境对许多建筑企业而言是长期存在的。法律监督不力主要体现在：第1,我国立法部门并未针对建筑企业的安全生产建立完善的法律制度,或者即使制定了相关制度,也可能因为监督机制不完善而无法有效执行；第2,部分建筑企业与相关部门和机构的合作不够紧密,导致法律监督工作难以落实,从而使安全生产风险管理工作因法律监督缺位而导致许多管理事项无法得到贯彻执行。

# 3 建筑工程安全生产中应用风险管理的具体路径

# 3.1 健全安全管理体系

建筑企业应制订安全管理制度。这些制度应涵盖建筑工程安全生产的各个方面,如施工现场管理、设备使用与维护、作业操作规范等。在制订时,应参考相关法律法规和标准,确保其科学合理、可操作性强[5]。

建筑企业应建立安全管理组织机构。该机构应包括安全管理部门和安全管理人员。安全管理部门负责制订和实施安全管理制度,组织安全培训和教育,开展安全检查和隐患排查,处理事故和应急事件等。安全管理人员负责具体的安全管理工作,包括监督施工现场的安全状况、指导工人进行安全施工操作、处理安全隐患等。

建筑企业应加强对施工现场的安全管理。施工现场应设置明显的安全警示标志,划定安全区域和禁止区域,确保工人的安全。同时,建筑企业应配备必要的安全设施和安全防护用品,如安全帽、安全绳、防护网等,保障工人施工安全。

# 3.2 强化现场管理

建筑企业应建立健全施工现场组织管理制度,包括施工现场的组织结构、责任分工、工作流程等内容,提高施工现场规范化管理程度[6]。在管理制度中,建筑企业应明确施工现场的管理人员和工人的职责、权限,确保施工现场的管理有序进行。

建筑企业应加强施工现场的安全设施设置。安全设施包括安全警示标志、安全通道、安全防护网等[7]。建筑企业应根据施工现场的具体情况合理设置安全设施,确保工人在施工过程中的安全。此外,建筑企业还应定期检查和维护安全设施,确保其能够正常运行。

建筑企业还应加强对施工现场的安全检查和隐患排查。安全检查是指对施工现场进行定期或不定期的检查,发现和排除安全隐患,确保施工现场的安全。值得注意的是安全检查不是一种形式主义,建筑企业应制订安全检查制度,明确检查的内容、频次和责任人。此外,建筑企业还应建立安全隐患排查制度,对施工现场的各个环节进行全面排查和处理。

# 3.3 强化技术支持

建筑企业应做好技术支持的准备工作。包括建立健全的技术支持体系、培养专业的技术支持人员和采购先进的技术设备。建筑企业应建立技术支持部门或岗位,负责施工现场的技术支持工作。培养专业的技术支持人员,提高其技术水平和专业素养。采购先进的技术设备,如安全监测设备、风险评估软件等,支持施工现场的安全管理工作。

建筑企业应加强风险评估与预测。风险评估与预测是指对施工现场的安全风险进行全面、科学的评估和预测,以确定可能存在的安全风险及其潜在影响。建筑企业应采用专业的风险评估方法和工具,对施工现场的各个环节进行风险评估。评估结果包括风险等级、潜在影响和相应的控制措施。建筑企业还应根据评估结果,制定相应的预防和控制措施,以降低施工现场的安全风险。

建筑企业应加强技术支持培训和交流。技术人员应定期接受相关培训,提高自身技术水平和素养。建筑企业应加强与其他企业和专业机构的交流与合作,共享技术支持的经验和成果。借助培训和交流,建筑企业能不断提升技术支持的能力和水平,为施工现场的安全管理提供更好的技术支持。

# 3.4 强化安全风险识别与预警

强化安全风险识别与预警是建筑企业做好风险管理的关键一环。安全风险识别与预警的步骤包括风险识别、风险评估、风险分类和优先级设定以及风险预警机制构建[8],即：

1）风险识别涉及确认可能导致安全事故的风险来源。建筑施工人员应采用专业的方法和技术（如现场检查、数据分析、专家咨询、历史案例回顾等）,对所有潜在风险进行识别。

2）评估风险的可能性和影响。可能性是指风险在工程期间发生的概率,影响是指风险发生后对人员安全、工程进度和成本的潜在影响。风险评估工具和技术包括定性分析（如故障树分析、危险和可操作性研究）和定量分析（如蒙特卡洛模拟、统计分析）。

3）根据风险的可能性和影响,对风险进行分类并设定优先级,以确定哪些风险需要立即关注和处理,哪些风险可以暂时搁置或在未来某个时点处理。根据风险的可能性和影响,可以将风险分为不同的类别,如设计风险、施工风险、材料风险、环境风险等。再根据风险的可能性和影响,设定优先级。例如,通过风险矩阵将风险的可能性和影响作为2个维度评估,将风险分为高、中、低共3个等级。高风险需要立即关注和处理,中风险需要在项目过程中持续关注和处理,低风险可以暂时搁置或在未来某个时点处理。

4）建立风险预警机制,以便在风险迹象显现时及时发出警报。通过配合安全监管、设立监控系统,并建立风险信息共享平台等,对潜在的风险进行识别,然后根据风险等级及时发出预警。

# 3.5 做好法律监督

建立健全法律制度。法律制度是指建筑企业根据国家法律法规和相关标准,制订安全生产管理制度[9]。制订包括安全生产责任制、安全生产管理制度、安全生产操作规程等在内的一系列法律制度。这些法律制度应明确安全生产的要求和标准,规范施工现场的安全行为,确保施工现场的安

全管理合法合规。

建立监督机制。建筑企业应建立专门的安全监督部门或岗位,负责对施工现场的安全生产进行监督和管理。监督人员应具备相应的资质和能力,能独立、客观地进行监督和管理。监督人员应定期对施工现场进行检查和评估,及时发现和处理安全隐患,确保施工现场的安全规范管理。

加强与相关部门和机构的合作。建筑企业应与相关部门和机构建立良好的合作关系,共同推进建筑工程安全生产的法律监督工作。例如,建筑企业应积极参与相关部门和机构组织的安全培训和教育活动,了解最新的法律法规和标准,提高自身的法律监督能力。还应与相关部门和机构共享安全信息和相关经验,加强对施工现场的监督和管理,确保施工现场的安全规范管理。

# 4 结语

风险管理在降低建筑工程事故发生概率和提高安全生产水平方面的重要作用愈发凸显。但一些建筑企业在生产过程中仍存在安全意识有待提升、现场管理不规范、材料安全存隐患、内部监管不到位、风险防范能力有待提高等问题。本文主要从健全安全管理体系、强化现场管理、强化技术支持、做好法律监督几方面提出风险管理提升的具体策略。 $\circledast$
