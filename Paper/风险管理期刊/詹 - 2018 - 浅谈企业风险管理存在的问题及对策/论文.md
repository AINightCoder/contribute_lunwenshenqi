# 浅谈企业风险管理存在的问题及对策

清远华清再生资源投资开发有限公司 詹史斌

摘 要：本文从风险管理在企业成长中的重要性出发，围绕当前企业普遍存在风险意识、制度结构、执行力、监督这四大方面的风险管理问题进行了详细的论述，简要阐述了问题形成原因，对企业的危害。就如何解决这四大问题方面，本文进行了一一对应解决方案简要的阐述，并辅以点明企业在遇到风险的时候如何应对和决策，希望能够引起企业对风险管理系统建设的重视。

# 1 企业风险管理的重要性

中国改革开放40年了，从2001年11月10日开始“入世”，到后来2013年9～10月习近平提出的“一带一路”，看奔驰的中欧班列，带着经济腾飞的希望来回穿梭，再到2007－2009年环球金融危机，再到习近平同志主政的防腐攻坚，老虎苍蝇一起打的廉政建设。现在正在进行得如火如荼的中美贸易大战，“大战”不止对“中兴”动手，同时影响中国的所有企业。1999年马云的阿里巴巴电商平台，腾讯帝国，百度，被人们简称“BAT”不断发展壮大，滴滴打车、共享单车等共享经济的兴起，然后就是我国房地产行业高速发展的十几年，并且房地产带动了建筑、装饰行业的百家争鸣，几乎有点疯狂的味道，我国各种有牌的、没牌的，有证没证的作坊式企业或者中小型企业如雨后春笋般平地而起。只要是跟上潮流这班车的，或沾上一点边的、基本上没有任何规划就上马，伴随这房地产的泡沫式的经济腾飞，人们开始跌入了不惜一切代价挣钱，不惜一切贷款融资投入到房地产和炒房的怪圈当中，并无形当中推动了房地产泡沫的更加膨胀，在这种没有多大科技含量的经济腾飞当中，中国真正有基础的科技发展其实是被拖后了。当市场变动带来的“黑天鹅”，使得很大一部分中小企业都没有风险防范，没能熬过漫长的冬天，都死在了没有风险防范的运作当中，即使如今的国家减税扶持政策下，依旧有很多的企业挣扎在濒死边缘，挣扎在资金链断裂，销售滞后，库存堆积如山之中。让我们跳出圈外看圈内，其实可以发现一个很现实的问题，就是很多中小企业是不堪一击的，一旦“黑天鹅”来了，根本抵挡不住。甚至是大型企业仍然如此。比如国产手机的没落，曾经盛极一时的“波导”“夏新”转而进入造车业，“金立”“科健”等手机行业巨头，全部不约而同的倒在了极度膨胀却不注重科技和市场的行业发展规划当中，究其原因就是没有一个长远的风险控制。“韶钢”“山东钢铁济南分公司”江苏南通的“明德重工”等一大批大型的，有着几十年基础的企业，依旧倒在去产能的大潮中。去年7月份，金融会议隆重召开的首日，《人民日报》在头版印发评论员的文章《有效防范金融风险》，本文里面提到了防“黑天鹅”，也要防“灰犀牛”，对各类风险小火苗绝对不能掉以轻心，虽然该文阐述的是金融业的风险，但是需要从中悟道，看到的是所有行业都要增强风险意识。

# 2 企业风险管理存在的问题

# 2.1 企业全员缺乏风险管理意识

我国大部分企业除了管理层外的员工，在风险管理的概念上其实一样是存在偏差的，普遍认为企业内部的各项规章、纲领、制度就属于风险管理，认为风险管理就是在日常工作中对职工的管理，导致忽视了真正大层面上的风险管理。另外，许多企业的风险管理的力度和韧性不足，企业整体风险文化氛围差，使很多的企业出现重大“黑天鹅”事件时仍然用“灰犀牛”的心态去应对。第二，我国多数企业里，并没有灌输全员参与到企业风险管理中去的观念，他们会天真地以为企业的风险管理就仅仅是管理层或者企业正式员工所需要完成的工作而已。或者误认为只是走过场罢了，而与基层职工或者临时工没多大关系。在这种带偏差的错误观念影响下，公司或企业里面普通管理人员、基层员工和临时职工失去了投入到企业风险管理中的机会，同时失去风险管理的兴趣，要知道风险管理针对的问题其实大部分是从基层工作中表现出来的，因此，很多关键和直接的风险小火苗不能够被及时发现和及时扑灭，没有做到日常防患，许多风险小火苗就会汇集成燎原大火，不能形成上下一致的风险管理意识合力。

# 2.2 企业管理结构不利于风险管理的实施

良好的公司管理结构是企业实施风险管理机制的环境基础，也是确保企业风险管理高效运行的前提。公司的管理层是企业风险管理的主导层，他们启动企业战略和风险管理，并在其中起到主导作用。如果无有效的企业管理制度和良好的企业管理架构，企业的权力分配以及互相制约的机制出了乱子，企业的风险管理与企业战略捆绑实施将没有保障。但是，当前经济形势下，很多企业管理架构层次混乱，企业管理层、普通管理员工职责、权限划分不清楚，再加上企业缺乏与管理相配合的管理机制与制度，这种国内企业普遍存在的情况，实实在在地阻碍了企业风管的顺利实施和开展，并切实严重制约了企业的长久发展。

# 2.3 风险管理制度的执行力弱

大部分企业虽然在企业战略设定的同时设立了风险管理制度，也设置了风管部或者审计部，但是在日常的战略执行当中，意识并发现了企业存在的风险因素，但是企业员工的执行力弱，却使得该风险因素反馈和处理时中途夭折，员工的文化素质和沟通能力总是达不到标准，有些是因为责任心差，不去跟踪该风险小火苗，最后会让企业的隐患不断，这是当前企业存在的普遍问题——执行力差。导致企业依然不能很好的应对风险，使企业遭受巨大的损失，甚至倒闭。

# 2.4 公司风险的监督与制衡指数低

企业风管内控制度的实施是承上启下、不断互相作用的过程，

# Management 经管空间

闻名世界的跨国企业中，无一例外地拥有强大的风险管控系统。由此可知，企业风险管理和内控若要获得“光合作用”为企业输送“养分”，必须要有“靠山”——强大的监督、制衡机制，只有这样，才能将“养分”准确地输送到企业各个角落。内部审计缺乏必要的“独立性”和“排他性”，防控部门形同虚设，内审部门又完全听命于企业的高层，甚至企业内审部门职员就是高层的亲戚朋友，使得内部监督在权力上和有效性上受到严重扭曲和限制，企业最终必然走向消亡。更加奇怪的是，许多公司为了节省成本费用，甚至连审计部门都没有设立，内审和监督没有专业人士和系统的运行，内审和监督输送的“养分”就无法到达企业需要监督的地方，自然而然，“养分”无法到达，企业就是“营养不良”虽然还生存着，但是因为“营养不良”，企业这棵“树”随时可能枯萎，企业最终依然不能活在激烈的市场竞争中。

# 3 改进企业风险管理的对策

# 3.1 关注“人”

“领头羊”在一个企业中扮演着无比重要的角色，是企业的方向舵，引领企业这条“船”在风浪中的行程，“领头羊”在风险管理和企业战略的捆绑实施中起主导作用。如果管理层中，所有的高管都拧成一股绳，发挥优秀团队和主导作用与团队力量，那么，企业的发展之路必定能减少很多磕磕绊绊，减少很多的风险突发。应该做到如下几点：第一，要求管理层要避免“一言堂”避免制衡缺失，企业要排除单方面投资入股比例过大、管理权过分集中。第二，要避免监督者自身的失控，减少监守自盗事故的发生。第三，提升企业全员风险意识。随着企业所处的竞争环境日益复杂、融资环境日益恶劣、企业经营遇到的风险日益加大，再加上如今的中美贸易大战，风险突发的机会日益加大，企业全体人员的风险防控意识也必须跟上企业发展和环境变化的“步伐”。“中兴”这次是处在了中美贸易大战的风口浪尖上的，“领头羊”在那么多年的发展当中，没有将风险很好的把握，没有在适当的时候建立企业的核心竞争力，导致美国这一招突然的“直拳”，让巨大的风险“黑天鹅”突然出现在“中兴”身上。鲜活的例子证明了，企业需要加大全体员工风险意识的培训和灌输，加大全员执行力的培训和灌输，加大企业风险文化的建设。打造良好的企业文化，使员工明白自己在内部控制、风险管理工作中所处的地位，并积极参与其中，同时也乐在其中，这样，全体员工就会适应企业发展，适应风险监控的需求，适应风险的应对。

# 3.2 大力优化公司管理架构

企业风管要有序地进行、顺利实施，与公司其他制度“完美结合”，就必须制定和不断完善治理架构。公司企业管理系统的“健康”，会使企业这棵“树”里面的各个部门之间的制度“完美结合”会让这棵“树”生机勃勃。良好的企业管理体系，是有效开展风管的第一大前提。首先，完善公司管理架构，需要认识到独立出来的董事、监事在公司企业风险控制中的重要性，认识到他们是“树”的大脑，准确地指挥着“养分”的精确运输，选举个别独立的董事和监事，并赋予他们在公司治理中的地位和权力，使他们在公司风险管理的监督工作中发挥重要作用；其次，公司应设立一个风险管理委员会——“风管委”或者独立审计部，公司可以从聘任具备风管专业知识的人，并充分利用他们的专业优势，解决企业“成长”过程中遇到的各种“病”，并配合公司的战略目标设定、风险的识别、评估控制过程。最后，公司需要设立风险控制部门或者独立审计部门，促使“树”里面的部门与部门组成一个整体的风险管理体系。

# 3.3 内控制度建设与加强

应避免不根据企业自身的特点，生搬硬套其他书本或者其他企业的风控内容，应充分发挥内部审计在企业内部控制和风险管理中的监督制衡作用，保证该控制和风险管理工作设计的合理性、有效性。强化公司监督制衡机制，需要从三个方面入手：首先，提升审计部门在企业的地位，并重点地强化公司内部审计工作的“独立性”和“权威性”。其次，企业监督部门的监督要“两手抓，两手都要硬”，既要重视财产风险，也要重视运营风险，排除企业这棵树的“内忧外患”，提高公司的风险管理和内部控制的整体水平。最后，应从纵向着手，深入开展企业运作全过程监督与评价。从事前介入、事中跟踪、事后评价反馈这条主线出发，从而使得企业能及时发现风险“小火苗”，防范经营风险，提升企业总体社会价值。

# 3.4 引进风险管控软件与系统程序

通过对公司业务记录、财务的风险分析数据及财务抽查信息、各阶层员工的反馈记录等信息进行计算机系统处理，及外部环境出现的重大风险形成有效的风险控制信息，集中到企业数据库，进行预警，并且指导企业如何进行风险防控。

综上所述，风险管理是企业生命的保护衣，同时，也是企业长远发展的指路明灯，在当前严峻的经济形势下，面对中美贸易大战时候的盾牌，同时风险管理能协助企业建立属于自己的核心竞争力，能纠正不良的企业制度的“病”。所以企业建立良好的风险控制，是切实可行且非常重要的一个管理手段。
