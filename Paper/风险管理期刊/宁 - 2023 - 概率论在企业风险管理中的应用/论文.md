# 概率论在企业风险管理中的应用

宁珂（中国人民大学统计学院）

摘要:伴随着社会经济的发展，企业也面临着更多的机遇与挑战。要想保证企业持续、良好发展，创造出更多的社会及经济价值，就要利用概率论在风险管理中的作用，降低企业的经营管理风险，避免企业遭受损失。本文从概率分析在企业风险管理中的作用、概率分析在企业风险管理决策中的应用准则及概率分析在企业风险管理中的实践应用入手，对概率论在企业风险管理中的应用进行详细分析，以促进企业健康发展。

关键词: 概率理论；概率分析；企业风险管理

# 引言

概率是用来反映确定事件可能性大小的数值。概率论是对自然界及人类社会中随机出现的事物、现象进行统计分析的一门数学学科。目前，社会经济发展迅速，企业作为社会经济的重要组成部分，需要对其进行正向管控。而在企业发展过程中，风险无处不在，这就需要企业在运行过程中对各类风险进行预警、识别、决策，进行综合性管控。在企业风险管理过程中，存在许多自然状态，但这些自然状态未来会发生什么变化，企业风险管理人员却并不能确认。实际上，各自然状态发生的可能性（概率）是能够通过特定的方式得出的，一个现象或者事件发生的概率越大，其发生的可能性就越强。基于此，企业可以利用概率理论对各类风险进行预估计算，并针对不同概率事件提出不同的解决方案。企业风险管理的实质便是对企业项目作出最优、风险最低的决策方案，因为未来具有不确定性，因此，企业管理人员需要通过概率理论统筹管理各类风险，并及时解决风险问题，以保障企业的正常运行。

概率理论。此外，概率理论对生物工程也起到十分重要的作用，工作人员常通过概率理论来构建生物模型。概率理论不仅在科学研究领域有着十分重要的作用，在社会科学领域，概率理论也能充分发挥其价值作用。正如拉普拉斯所言，生活中重要的问题，绝大多数在实质上都只是概率问题。

# 二、概率分析

概率分析是概率理论中的一个分支，又被称为风险分析，主要是通过对不同确定因素变动幅度的概率分布进行分析来求得该因素对于项目或是企业经济效益指标的影响。概率分析是对可行性以及风险性的优劣进行判断的不确定分析法，目前常应用于大型企业的项目评估及管理工作中。

概率分析的方法主要有期望值法、效用函数法、模拟分析法以及德尔菲法。主要分为五个步骤：首先，列出项目或者企业需要考虑的风险因素，之后对各个因素引起的结果进行确认，列出数值变化情况，再分别确认不同情况下各种因素发生的可能性，即概率，最后对目标的期望值进行计算，并求出目标值大于或者等于零的累积概率。

# 概率理论及概率分析

# 一、概率理论

概率理论是研究现象数量理论的数学分支，是对随机现象的客观研究。随机现象是一种实际存在的客观现象，当人们观察该现象时，所得到的结果并不能通过预先的观察进行确认，因此会出现多种结果，这便是概率理论需要研究的内容。在科学技术发展过程中，许多地方都需要用概率理论的逻辑支撑来解决实际问题。概率理论从 20 世纪 30 年代开始发展至今，其理论逐步与实际形成密切联系。在物理学方面，常利用泊松分布、弗瑞分布对高能粒子的吸收进行概率计算。在化学方面，研究反应过程的时间变化率以及影响变化的因素通常也会用到

# 概率分析在企业风险管理决策前的准备工作

世界是由客观现象、事实所构成的，且它们之间是存在普遍联系的，并不是独立存在的。企业风险管理是一个复杂的过程，没有办法通过一项或几项独立的方式来完美处理风险，需要依托概率分析针对性预防企业风险。作为企业风险管理人员，在针对风险作出管理决策之前，需要进行以下准备工作：

首先，依据企业预先制定的经营目标及需要解决的问题，制定决策方案。此类目标可以是长期且持续的，也可以是短期阶段性的，主要依据企业在不同时期需要解决的问题进行选择。如有的企业需要解决产品数字化转型中的问题，有的企业需要解决产品的销售问题，企业风险管理人员需要明确目标问题。其次，通过各种途径及渠道，对企业的内外部信息进行收集，以此来分析确认企业风险的不确定因素。对资料信息的收集越全面、越准确，在自然状态下的概率估算也就越接近于实际数值，企业风险管理人员所作出的决策也就越正确。再次，要以企业发展目标为纲要，充分考虑企业的实际情况及外部影响因素，对收集到的信息资料进行整理、分析、计算，并以此为依据，制定出详细可行的风险解决方案。最后，在解决方案备选库中，根据实际概率计算结果确定最佳方案。在决策过程中，企业风险管理人员始终要对风险的发生概率及所造成的影响进行分析评估。一般的风险决策具有盲目性，此类决策没有形成一定的统计结果以供企业参考，因此，在企业风险管理中不值得采用。但利用概率理论中的概率分析手段，可以对未来风险事件的发生、状态、结果进行预测，比一般风险决策具有更高的解决能力。

# 概率分析在企业风险管理决策中的准则

针对企业发展运行中遇到的风险问题，概率分析在企业风险管理决策中须遵循以下原则：

# 一、追求最大可能原则

确定性决策相对于风险决策来说便是无风险决策。因此，如果能将企业风险管理在最大程度上转换为企业风险的确定性管理决策，则可将风险尽可能地减小，实现最大可能的解决。依据概率理论可知，一个事件所发生的概率与该事件的可能性是等价的。世界上没有绝对的事件，因而确定性的风险决策在理论中是难以出现的。因此，企业在遇到相应风险时，需要对风险进行决策，提出解决方案，此时便应追求最大可能性，在不同概率值之中选择最佳值，并对其他数值所对应的事件、方案进行舍弃。以某公司召开市场拓展会为例，会议地址主要从A、B、C 三个地区进行选择，主要风险决策因素包含不同地区的市场拓展收益、天气情况。针对天气风险进行分析，天气分为晴、雨、阴三种，根据气象局得到的数据可知，三种天气发生的概率分别为 0.2、0.7、0.1。通过使用最大可能原则可知，自然状态下该事件中雨天的概率 $\mathrm { P } { = } 0 . 7$ ，处于三者的最值，因此，应在此条件下进行后续市场拓展收益的风险决策，在此状态下就可将风险转化为确定性决策。

追求最大可能原则是企业风险管理中最常用的一种概率理论手段。在风险决策过程中，一组自然状态中有一个维度出现的概率比其他维度的概率大很多，并且各种自然状态下相应的损益值并没有明显差别时，使用该原则的效果就是最好的。但如果在一组自然风险状态中，不同维度之间发生概率相互接近，并且差值很小，在此情况下，使用最大可能原则则可能会造成不良效果，甚至会产生风险决策的失误。

# 二、期望值原则

通过将风险解决方案得失情况与自然状态的概率值相乘，即可得出该风险解决方案的期望值。期望值原则通过将不同风险解决方案在不同自然状态下的概率作为“权”，计算不同解决方案下损益值的加权平均数值，之后再利用最优解原则，选择最大的期望收益或者最小的损失值方案作为风险决策方案。期望值原则可以分为两种，一种是风险决策的目标是收益最大，则需要采取期望值最大原则。另一种是风险决策的目标是损失最小，则采用期望值最小作为风险解决方案。在企业风险管理过程中，期望值原则是十分常用的原则及手段。

# 三、机会均等原则

机会均等原则是在不同自然状态下，没有十分充分的理由能说明某一个维度状态的发生概率高于其他状态时，以此来设定各状态的发生概率均相同，并且概率之和为 1。在此基础上，计算不同方案下的期望收益以及期望损失值，并按照期望值原则对风险解决方案进行决策。

# 概率分析在企业风险管理中的实践应用分析

综上所述，概率分析在企业风险管理中主要基于最大可能原则、期望值原则及机会均等原则，计算出相应的概率数值，以此来分析企业不同事项的风险。下面，本文将通过不同的案例，分析概率理论在企业风险管理中的实践应用效果。

# 一、概率分析在企业风险管理评价中的应用
