# 国有企业对外投资管理及风险防范措施研究

# 王 乐

（宁夏宁东科技创业投资有限公司， 宁夏 银川 751999）

摘 要： 随着我国现代化社会经济水平逐渐提升， 国有企业的对外投资项目相应有所增加， 为国有企业综合经济效益水平的提升奠定了良好的基础。 但是许多国有企业在对外投资当中受到各种外界因素的影响， 还存在很多风险， 使得企业发展滞后。 所以， 国有企业不仅需要做好对外投资管理工作， 还要在实际开展项目投资时合理管控其中的风险， 有针对性地分析及控制各类风险， 提高企业对外投资管理水平， 同时为企业开展其他业务工作奠定良好的基础。

国有企业持有的国有资产管理质量会在一定程度上影响我国社会主义市场经济的发展， 对于国有企业自身的发展质量和成效也会产生较大的影响 在市场经济体制改革的当下， 国有企业需要加大对外投资管理力度， 以企业资产保值增值作为主要的工作要点，在实践中解决对外投资管理中的问题， 给企业带来更大的机遇， 避免产生资产流失问题。

# 1 国有企业对外投资管理与风险防范的重要性

# 1.1 保障资金安全

目前， 国有企业市场环境比较复杂， 实际开展对外投资工作时面临的风险因素和问题较多， 稍有不慎就会对企业资金的安全造成威胁。 在新时期建设发展中做好对外投资管理和风险防范工作可以从根本上控制其中的安全风险， 以全新的对外投资风险控制体系的构建和完善作为基础， 保障国有企业资金的安全性。 当管理人员对企业的对外投资项目进行综合管理时， 能够采取科学的审核方式分析和优化企业的资金使用状况， 特别是在开展投资项目时能够提高投资可靠性与可行性， 避免企业资金的浪费， 达到提高企业资金安全性的目的。

# 1.2 满足法律法规要求

近年来， 越来越多的国有企业在对外投资当中抓住了发展机遇， 加大了对外投资的重视。 为了进一步提高国有企业对外投资管理质量和水平， 我国出台了相关法律法规， 对企业的内部风险管控提出了更加明确的要求， 以更加规范的法律法规和条例进一步保障国有企业内部资产的安全性及完整性， 实现更高的企业经营与发展目标 做好国有企业对外投资管理及风险防范工作是满足法律法规要求的基础， 也是企业规范化建设发展的要点， 能够解决国有企业管理松散、投资无序等问题， 避免企业频发风险问题。

# 1.3 解决企业投资问题

企业经常会在投资过程中产生一些问题， 特别是在市场环境不稳定的情况下， 会使企业内部风险增大， 不利于国有企业的持续稳定发展。 加强国有企业对外投资管理和风险防范能够更好地选择投资项目，使相应的产权界定更加清晰， 解决企业对外投资中存在的监管责任落实不到位等问题， 降低企业的综合运作风险， 避免企业投资决策受到负面影响。 在可行性对外投资管理模式下， 国有企业的资金管理、 投资决策等问题都能够得到有效解决 从根本上提高对外投资的管理质量和效率， 避免企业出现资金链断裂等风险， 实现对投资风险的有效控制和准确预防。

# 2 国有企业对外投资管理存在的主要风险

# 2.1 政策与环境风险

国有企业对外投资与企业对内投资存在较大的差异， 其会受到客观的政策和环境因素的影响， 在对其他国家的项目进行投资时， 就需要考虑不同区域的环境状况和政策内容， 否则会在企业综合建设发展中产生较多负面问题。 其他国家的宏观政策对于国有企业的投资具有导向性作用， 基于宏观调整需要的情况下， 一些产业在发展当中会被压制， 国有企业作为投资主体就需要时刻把控和理解国家的宏观政策， 避免投资行为与政策相悖， 否则会引发逆向风险。 国有企业开展对外投资管理工作时会受到项目所在区域地方政府政策及决策的影响， 特别是不同国家的自然资源、 文化习惯、 思想观念等大相径庭， 一些区域存在地方政府保护主义现象， 会对投资项目进行干预， 如果国有企业项目投资管理人员缺乏对这个因素的考量， 就会使投资进度缓慢， 最后很可能导致投资失败。 对外投资最需要考虑的因素就是国家政治格局是否稳定， 一些国家的投资环境并不好， 如果国有企业贸然进行对外投资会对自身的竞价发展造成阻碍。

# 2.2 财务风险

对外投资管理作为一项与企业财务管理及建设发展存在直接关系的内容， 应充分考虑企业在发展中是否会面临财务风险， 才能够保证投资决策的准确性。国有企业开展对外投资管理工作时存在潜在的财务风险， 主要是因投资决策不当造成失误。 还有一些国有企业的资金投入数量和时间具有较强的不确定性， 很可能会在投资过程中引发重大突发事件， 给企业的建设发展造成不利影响。 企业开展对外投资管理工作之前会谨慎评估项目投资的可行性， 虽然在前期评估过程中结果是可行的， 但是在项目运行当中还是会受到其他因素的影响导致项目中断， 使项目投资效益达不到预期， 投资成本无法收回， 这对于国有企业这个投资主体来说会产生很严重的财务资金影响 还有一些对外投资项目的整体收益水平不高 无法与项目投资成本相平衡。 在这种情况下， 投资时间越长， 国有企业的资金亏损越多。

# 2.3 法律风险

法律风险是国有企业对外投资失败的主要原因，也是企业在对外投资管理中不能忽视的一个重要影响因素。 国有企业对外投资涉及的范围比较广泛， 包含众多项目类型， 并且表现形式复杂， 在项目投资的各个环节都存在潜在的法律问题。 还有一些区域的政治风险及财务风险会以法律的形式呈现出来， 给国有企业的经营发展造成较大的影响。 国有企业作为投资主体应考虑不同国家或地区在法律制度上的差异， 还要综合分析国家的文化背景存在哪些不同， 如果缺乏周全的考量就会增大企业对外投资的难度。

# 2.4 投资方式风险

不同的投资方式产生的对外投资管理成效存在一定的差异， 大部分国有企业在对外投资过程中采取的投资方式都有所不同。 根据目前的国有企业对外投资形式来看， 大多数企业都会以并购的方式为主， 其能够在很大程度上提高被投资企业的经济实力， 使其在行业发展中处于更加领先的地位， 为企业开展其他业务工作和经济活动提供良好的经济保障。 但是一些投资方式会给投资主体的生产运营带来一定的冲击， 如果投资主体不能够很好地应对这种冲击和影响， 就会使自身的经济发展受到限制。

# 3 国有企业对外投资管理存在的问题

# 3.1 缺乏内部控制机制

从现阶段的国有企业对外投资管理状况来看， 大多数企业都存在内部控制机制不完善的问题， 虽然企业有足够的经济实力为项目投资提供支持， 但是缺乏内部控制机制， 不足以应对多元化的对外投资管理问题 目前 我国部分国有企业的内部控制机制处于有待完善的状态， 企业缺乏对对外投资管理需求的深入分析和研究， 使得对外投资管理质量不高， 还会引发较多问题限制企业自身经济水平的提升。 部分国有企业的董事会形同虚设， 长时间以人员控制作为核心，削弱了 “法治” 的力量， 过于 “人治” 的现象使得企业对外投资管理掺杂了较多主观因素， 很容易促使企业陷入困境， 不能够发挥企业内部控制机制的作用 因此会给企业造成较大的损失 部分国有企业尚且没有意识到内部控制机制建设的作用， 在新时期迅猛发展的过程中 企业没有跟上时代发展的步伐 在对外投资管理当中会面临巨大的挑战。 最显著的问题是国有企业没有建立可靠的协调机制及沟通机制， 虽然根据企业自身的情况构建了内部控制机制 但没有得到投资方的认可， 双方在沟通交流中存在不便， 还会引发信息不对称等问题。

# 3.2 投资过程中管理缺位

管理工作的有序开展对国有企业的对外投资具有非常重要的作用， 目前， 许多企业开展对外投资的过程中存在越位、 错位和缺位的问题， 主要是由于管理人员没有明确自身的工作职责范围， 存在管理尺度过宽的问题 一些领导人员出于自己的主观意愿在对外投资管理当中产生过多干预， 而管理人员在提交投资项目文件时要经过较长的审批时间， 给对外投资管理造成了不利影响。 部分国有企业设置了子公司， 在对外投资管理当中会受到总公司的制约而不具备独立自主经营的权利 影响了子公司的经营效益 管理人员的积极性也会受到打击。 部分国有企业过于注重综合经济效益水平的提升， 对于经济效益不好的子公司往往会缺乏关注， 甚至存在谁都不愿意管理的情况， 在出现问题时股东之间还会互相推诿， 产生严重的管理缺位现象。

# 3.3 风险应对措施不当

国有企业对外投资管理需要以科学的风险防范和应对措施作为基础， 才能够更加全面地解决实际管理当中产生的问题， 为企业未来的有序发展打好基础。实际上， 部分国有企业在对外投资管理当中过于追求多元化发展， 存在过度并且盲目投资的现象， 甚至会进入完全陌生的行业， 在不熟悉的领域进行对外投资， 增大了对外投资管理风险。 个别国有企业在对外投资当中没有注重企业存量资产的盘活， 虽然企业具有较多优势资源， 但是过于分散， 使得企业的经营主业逐渐淡化甚至被取代 进而降低了企业的核心竞争力， 在很大程度上影响了企业的经营管理成效。 产生这个问题的主要原因在于国有企业在对外投资管理当中缺乏可行性的风险预警机制， 当对外投资面临巨大风险或者产生巨大损失时， 管理人员无法做出准确的判断， 也没有及时采取可行性措施应对风险， 导致企业被推到了风口浪尖。

# 3.4 信息管理反馈平台不健全

我国当前处于信息化发展的关键时期 各个领域在建设发展的过程中都逐渐增大了对信息化技术的应用程度， 致力于改善传统的管理方法， 为企业的综合发展提供技术保障 目前 信息技术已经渗透到了社会各个领域当中， 许多企业都会将新时期的信息作为一种资源， 为对外投资提供可靠的依据。 但是少数国有企业在对外投资管理当中并没有充分利用信息化技术构建信息管理反馈平台， 开展投资活动时还是存在投资信息不对称的问题 并且投资项目信息的时效性不高， 管理人员掌握的信息达不到完整性与准确性的要求 在传递反馈重大信息时还会产生障碍 增大了国有企业对外投资管理风险。 在这种情况下， 国有企业各层级之间的沟通交流受到了较大的阻碍， 无法及时凭借现有的信息做出反应， 企业内部与外部的信息流转也难以畅通， 最终影响企业的投资效益。

# 4 国有企业对外投资管理及风险防范措施

# 4.1 明确对外投资战略目标

国有企业开展对外投资管理工作时， 应在实际投资管理之前明确对外投资战略目标， 根据这种国际化经济活动的特点构建更加科学的管理体系， 将复杂的运行体系简单化处理 立足于全球化发展的角度分析战略问题， 为对外投资及企业的综合发展提供可靠的指导方向。 因此， 管理人员非常有必要分析和研究国有企业的实际发展情况， 根据现阶段的形势分析对外战略目标方针， 避免对外投资盲目性， 从而提高企业对外投资的成功率。 战略目标的制定和实现都需要以企业的发展现状和未来的发展方向作为基础， 同时要考虑到被投资企业的既定环境， 从而刻不容缓执行相关的战略目标。 所以， 管理人员需要加大对这项管理工作的重视， 国有企业领导人员也需要明确对外投资战略目标的重点内容， 结合国际市场经济的发展状况及有关政策要求制定可行性战略目标， 为提高国有企业对外投资管理质量及水平提供良好的保障。

# 4.2 完善内部控制机制

部分管理人员参与国有企业对外投资管理工作时缺乏对自身的约束， 在管理工作受到较多因素的影响时， 难以保证对外投资管理方法的可行性， 给企业的综合发展带来较大的影响。 提高国有企业对外投资管理水平并防范其中的风险时， 企业需要有针对性地完善内部控制机制 加大对外投资管控力度 防止产生资产流失问题。 构建和完善内部控制机制时， 企业需要以战略决策、 财务审计监督机制的构建为主， 让管理人员明确相关机制的内容， 保持端正、 严谨的态度完成相关工作任务， 还要保证机制的可追踪性， 防止国有企业对外投资业务脱离企业管理 个别管理人员在实际管理当中会掉以轻心， 企业在完善内部控制机制时就需要构建更加科学的沟通与监督管理机制， 避免对外投资管理变成单纯的人员管理 要从根本上防范对外投资管理风险， 还应在完善内部控制机制的过程中构建合作企业制度， 明确不同企业之间的差异，搭建良好的企业沟通桥梁， 提高投资信息的对称性与时效性。

# 4.3 重视投资违规责任追究

国有企业组织对外投资管理工作时难以完全避免违规现象， 尤其是在管理人员干涉过多时， 会影响企业对外投资管理成效， 给企业综合发展带来负面影响。 为了保障国有企业对外投资风险防范效果， 企业需要加大对违规投资责任的追究， 提高管理人员的责任意识， 建立明确的追究制度， 使对外投资管理落实到位。 与对外投资管理相关的责任人需要做好前期调研工作， 掌握详尽的投资项目信息， 提供真实、 可靠的研究报告， 分析项目投资的可行性。 如果其提供的数据信息存在问题或者与被投资方存在不正当利益关系， 就需要由项目责任单位及责任人承担责任。 这些违规现象的产生不仅会导致国有企业对外投资失败，还会产生经济损失 严重时会损害企业的声誉 为了加强投资违规责任追究有效性， 管理人员需要明确具体的责任追究情况， 包括违反决策程序规定擅自投资、 泄露投资信息引发经济损失、 逃避企业审批监管将投资项目化整为零、 违反企业规定非法获得利益等。 一旦出现这些情况， 就需要及时追究相关单位和人员的责任， 尽量减少违规现象引发的损失。

# 4.4 增强风险监控预警能力

为了全面提高国有企业对外投资管理实效性 有效规避一切风险， 企业需要持续增强自身的风险监控预警能力， 确保投资目标顺利实现。 在发展过程中，很多国有企业都存在 “重投资、 轻管理” 的思想理念， 缺乏对投资评价及风险管理体系的重视， 没有构建相关的管理机制分析和预防风险， 增大了国有企业对外投资管理的风险 在后期产生问题时很难完全解决。 国有企业在提高风险防范能力时， 应以 《对外投资管理制度》 《对外投资风险管理制度》 《对外投资后评价管理方法》 等制度作为基础， 结合对外投资项目的实际情况优化事前、 事中、 事后投资管理流程 实现对投资项目的全方位监管 强化管理人员的风险监控预警能力时， 企业应在项目投资开始之前制订可行性战略计划， 让管理人员明确计划内容和主要的发展方向。 在项目实施过程中则需要根据项目的内外部环境、 项目进展和运营情况等分析投资效益， 进行评价与总结， 制定可行性风险管理方案， 为项目投资提供有价值的信息。 与此同时， 管理人员应充分发挥对外投资管理监督职能， 明确各个环节的监督管理要点 密切关注国有企业对外投资情况 从根本上防范投资风险。

# 4.5 健全对外投资决策机制

国有企业组织对外投资的过程中需要贯穿整个项目过程， 从最初的项目寻找和评估到项目实施、 项目跟踪监管及项目处置退出 都需要保证投资方向的一致性及准确性， 才能够确保项目投资成功。 首先， 开展对外投资管理和风险防范时， 国有企业要保证项目投资决策的严谨性 以提高投资决策准确性作为主要目标， 确保项目投资符合企业的实际发展需要， 意识到投资决策的重要性。 其次， 国有企业应健全对外投资决策机制， 以客观事实作为依据， 不断完善对外投资决策机制， 根据项目投资的类别及资金多少设定投资审批权限， 以更加规范的投资决策审批流程作为基础， 充分开展项目投资调研、 评估、 审批等工作。 再次， 在健全对外投资决策机制的同时， 还要按照投资审批权限明确划分决策人员、 审批人员的职责。 将各个环节的工作落实到位， 如果其中出现问题就需要追溯到源头， 避免责任推诿现象。 最后， 企业还需要优化及完善职责分离体制， 将不相容的职务及岗位进行分离处理， 保证岗位职务合理授权， 同时使各岗位人员之间相互制衡。

# 4.6 优化信息管理反馈系统

国有企业在对外投资管理中利用信息化技术可以有效提高管理质量和效率， 解决传统管理中的弊端，充分体现新时期管理方法的优势。 在现阶段的国有企业建设发展当中 企业应不断优化信息管理反馈系统， 提高管理人员的信息化管理水平， 以更加完善的信息反馈机制作为支撑， 拓展信息传递途径。 在新时期建设发展当中 国有企业需要结合企业的发展环境和制度法规构建符合自身发展形势的信息交流沟通分享平台， 让管理人员迅速掌握最新的信息与资讯， 通过对信息的科学分类及深入分析提出更加合理的对外投资管理意见与建议， 保证管理工作能够落实到位。在构建和优化信息管理反馈系统时 管理人员需要根据实际情况进行信息反馈， 实现各层级之间的横纵向沟通交流与反馈， 提高信息交流与利用率。 在依托平台开展管理与风险防范工作时， 各个工作人员之间都可以更好地交流自己的工作经验， 营造良好的企业内部管理氛围， 达到提高国有企业对外投资管理质量的目标。

# 5 结论

在目前市场经济一体化发展的环境下， 国有企业需要考虑多方面的影响因素， 优化对外投资管理形式， 增大风险防范力度， 达到更高的对外投资管理要求。 在未来发展当中， 还需要完善相关的管理机制，构建信息反馈系统 降低对外投资风险系数 从根本上预防风险的发生， 为提高对外投资管理水平打好基础， 并为企业的高质量稳定发展保驾护航。
