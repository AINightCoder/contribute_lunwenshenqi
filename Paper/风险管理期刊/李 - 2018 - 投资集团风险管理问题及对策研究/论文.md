# 投资集团风险管理问题及对策研究

李强

（四川省商业投资集团有限责任公司，四川成都610000)

摘要：风险管理对于投资集团而言非常重要，是企业管理的核心目标。重视风险管理，可以直接将企业的风险转化为企业的利润，这也是投资集团的所有领导人都必须严格关注和把控的关键性问题。风险管理的作用是为整个企业的所有流程服务，增强投资集团制定有关企业的重要决策的能力，提供多种途径供投资集团选择最有价值的优化措施，进一步完善企业的管理模式并加强投资集团之间的合作能力和协调能力，降低风险，实现合作共赢的目标。

关键词：投资集团；风险管理；问题；对策DOI:10.13768/j.cnki.cn11-3793/f.2018.2863

# 一、引言

随着社会经济的不断发展和进步，我国的投资企业所处的社会背景也在不断的改变，特别是在信息化高速发展的现在，投资公司的经营环境逐渐复杂化。企业从稳定性经营过程渐渐过渡到风险性经营的阶段，由此可见，风险管理在投资企业的经营和运作过程中具有不可或缺的地位。风险管理的内容不仅是对投资企业机构本身的运营所做的管理，更是对企业的投资项目的一个风险管控。从目前我国的投资集团所面临的风险状况来看，主要具有以下四个特点，分别是风险高层化、风险扩散化、风险契约化还有风险网络化。为了针对性解决以上问题，投资集团需要建立集团内部的风险管控机制，并在此过程中不断地完善企业自身的管理模式，促进投资集团的获利能力。

# 二、投资集团风险管理存在的问题

# 1.风险管理目标单一

从目前我国的投资集团的风险管理来看，大多数企业的风险意识不够全面，风险管理目标很单一，仅仅只是将目光放到了投资的领域里，而忽略了集团内部的风险管理。就集团内的财务风险来说，其下所包含的风险就有筹资风险、管理风险和资金的流动性风险等，这一系列的风险都会对企业的经营造成一定程度上的影响。比如，企业的人员管理，其中包含了整个投资集团的所有人员的培养和培训管理；制度的设定问题，其中包含了企业的前期投资流程和整个企业的财务数据评价，以此建立起符合公司自身发展情况的风险预警机制。随着社会的发展，投资集团在企业的风险管理上所采用的手段呈现出多样化的状态，可以使用传统的人力形式，也可以采用数据库的形式。但目前来看我国投资企业的风险管理目标比较单一，严重的阻碍了投资集团的风险管理框架的构建。

# 2.资金流通不畅

投资企业的资金问题，主要表现在资金流出和资金流出不能均衡匹配。在资金流出方面，无论是对外部项目公司投资还是对内部公司的资本金注入，单笔资金流出的需求越来越大，投资公司往往投资项目众多，资金需求量巨大；在资金流入方面，一是项目投资往往回收时间长，前期没有或很少有资金流入；二是所投资的项目公司是否分红或分红金额多少，受被投资公司所处阶段、公司现金流、管理层偏好等多种因素影响，分红金额并不能得到保证；三是受国家宏观经济和财政政策影响，传统的融资方式如银行融资，金融机构出于对风险控制的需要，对投资企业的增信措施要求越来越严格，往往要求有资产抵押或担保，从而增加了融资的难度。四是投资企业可能存在的企业融资方式和融资期限的不匹配，短资长用等问题，可能加剧企业的风险。综上所述，投资企业的资金流入和流出不能匹配，可能导致资金问题成为投资企业发展的瓶颈。

# 3.投资领域过广

投资集团发展的过程中，都会出现一个大多数企业都要面临的问题，就是一味的扩张市场，扩大投资领域，但企业内部人员和机构的专业水平无法跟上集团的投资脚步，在这样不均衡的力量之下，直接给企业打开了一个风险口，使得企业随时面临着投资失败、收益不佳的风险。但这是每一个扩张中的企业都无法避免的问题，投资领域的扩大，企业员工的知识面和技术水平，包括企业领导者的管理水平都直接影响到公司的投资活动。缺乏专业人才，是几乎所有的投资企业都要共同面对的一个难题，也因此制约了投资集团在市场上的发展。再加上投资过程中，往往会出现一个甚至多个担保人的情况，多重担保将投资项目的风险在各家公司之间转嫁，增加了投资风险，一旦某个环节出现问题，将带来一系列严重的风险管理问题。

# 三、投资集团加强风险管理的对策

# 1.提升风险与责任意识，培养全员风险管理文化

投资集团公司将要建立一个完善的风险管理机制，必须依靠集团内全体人员的通力配合和支持。首先需要从风险意识上进行管理，增强企业人员的风险管理意识，将风险管理这个概念融入企业的文化中，时时刻刻提醒着企业的每一位成员。其次，公司需要制定出一个明确的风险内控标准，让风险管理这个抽象的概念以流程化的形式体现出来。制定出相应的手册，要求员工严格执行此过程，再伴随着监控管理和事后的考核。最后，在日常的经营决策中，领导层以身作则，坚持风控前置，比如规定重大决策，在风控部门没有同意前不能走决策流程等，提升全员的风险管理意识。在这样的规范化流程中，让员工感受到企业对于风险管理工作的重视，使得风险管理的理念深入人心，让风险的管理和防范成为企业的自觉反应。

# 2.建立风险信息管理，完善控制管理体系

前期建立投资集团公司的风险管理制度，需要公司提前做好一系列的准备。提前收集好所有的有关风险管理的信息数据，建立一个有关风险管控的信息数据库，必须涵盖企业的所有内部信息和相应的一些外部的信息，在公司需要某方面的数据时，风险数据库能及时的提供所需的数据，为风险管控提供有利的条件。风险管理信息数据库的建立，必须达到集团董事会对于风险管控的监管要求，对于重要的指标能够实现自动化提示，监控企业内部的业务交易情况，特别要加大对大额的投资项目的监管力度。风险控制管理体系主要由三部分组成，分别是企业的组织机构体系、内部控制制度和企业的信息管理系统。在建立投资企业的风险管控和预警系统的过程中，必须从科学、全面、合理的角度出发考虑，企业领导者要合理的运用企业的各种财务报表，从中分析企业的财务状况和企业的实际经营情况，利用其中的有效指标，对公司投资活动提供有建设性的建议。

窃密泄密案例警示教育读本》《机关、单位保密管理务实手册》等书籍。订阅《保密工作》杂志，通过经常性的学刊用刊，及时把握新形势下保密工作要点，提高保密工作水平。

$\textcircled{4}$ 抓好检查现场保密宣教

充分利用监督检查的有利时机，开展现场保密宣传工作。对在监督检查中发现的安全隐患，现场说法，分析隐患可能带来的危害，传授保密知识与方法，使检查活动同时成为贴近实际的保密宣传教育课。

# 2.以资源为保障，提升“物防”能力

所谓物防，是指加大资源投入，为国家秘密保护和管理工作的开展提供强有力的物质保障，是构建保密综合防范体系的重要物质基础。

结合工作实际，将保密物质保障经费列入企业长期发展规划和年度财政预算计划，确保保密工作正常有序进行。一是根据保密工作要求，建设保密机要室，实现保密室办公、设备、阅文三分开；二是加大经费投入，为保密要害部门部位配置铁密码文件柜、碎纸机、防盗铁门、铁窗和红外线防盗报警装置，购置保密检查专用工具、移动信号干扰器，强化技术防范装备，确保涉密、涉敏感信息的安全。

3.以科技为手段，加大“技防”应用

所谓技防，是指采用现代科学技术对涉密信息进行保护，防止涉密信息泄露或被窃取，是构建保密综合防范体系的有效手段。

一是强化信息安全基础管理。在建设信息系统时，做到信息系统建设和保密技术防护系统建设同步规划、同步设计、同步施工、同步启用。认真落实《国家电网公司办公计算机信息安全管理办法》，将网络安全始终作为保密工作的重中之重加强管理，及时配置、更新网络安全设施设备，办公网络与互联网实行物理强隔离，并通过防火墙、IDS（入侵检测）、UMC、ISS、IMS等实时监测信息网络安全；公司系统内外桌面终端全部安装了网络保密自动检测系统及邮件安全管控系统，安装率达到 $1 0 0 \%$ ，实现信息内网办公计算机深度检查和清理，确保不出现盲区和死角。计算机、应用系统、邮箱口令长度大于8个字符，均由字母、数字和特殊符号（如@#￥%&等）混合组成，3个月定期更换口令。办公计算机、U盘等移动存储介质进行注册登记，做到底数清楚、可控在控。

二是落实涉密会议、活动保密措施。涉密会议、活动坚持“谁主办，谁负责”的原则，制定保密工作方案，明确保密措施，指定专人负责。公司年度“两会”（年度职工代表大会暨工作会议）召开期间，强化会议涉密材料管理，严格编号登记、标密、回收和销毁，根据参会代表身份确认，分类发放、会后回收，会场使用移动信号干扰器，加强涉密场所周边管理防范，切实消除隐患。

三是加强涉密载体过程管控。严格落实秘密载体“密来密往”要求，健全涉密载体档案清册台账，落实签收、分发、存档、销毁各环节管理，秘密级及以上文件详细记录各环节流转人员，严格做到“涉密文件不上网，上网文件不涉密”

# 三、电网企业基于人防、物防、技防的保密管理效果

通过构建以人防为核心、强化物防保障、用好技防手段的保密综合防范体系，电网企业把保密作为一项重要工作，融入企业发展。筑牢安全保密防线，严防电网安全等核心信息的失泄密事件，确保电网安全稳定运行及国家秘密、公司企业秘密的安全。多年来，国网榆林供电公司未发生重大违规事件和失泄密事件，有力维护了企业安全稳定发展大局。

通过进一步加强“人、机、技”防控能力建设，全面落实保密责任，加强保密宣传教育，员工保密意识和保密常识得到强化，保密工作流程不断完善，保密制度有效落地，保密管理水平不断提升，保密同业对标在国网陕西省电力公司11家地市公司中排名第二，在榆林市国家保密局保密检查中取得“优秀”的好成绩。

3.进行多元化经营，分散投资的风险

投资集团公司在进行企业的投资过程中，所遵循的道理可以用一句俗话来形容，就是“不要把鸡蛋放在一个篮子里”。企业在进行投资时，一定要注意分散投资的风险，不能将大量的资金集中在某个项目中，与此同时，还可以选择和其他的投资企业进行合作，分担风险，共享利润。作为投资集团公司，其运作模式就是依靠投资各种各样的好的投资项目，以此为企业和客户获取利润，市场上的项目繁多，盈利和亏损都是常有的事，企业可以用盈利去弥补亏损的项目，以此来达到一个收支的平衡。作为专业的投资集团，所具备的投资团队也是相当专业的，在这样专业化的团队和优质的平台支持下，对于企业的投资项目的选择必须更加的谨慎，以最专业的技术水平对项目进行全面的了解和调研，这样才能控制企业的风险。

# 四、结束语

投资集团所面临的风险比一般的企业所面临的风险都要高，且风险管理是一个持续不断的过程，需要根据市场的变化及时做出调整，投资集团如何制定企业的风险管理机制，是当前投资集团所面临的一个重大课题。以投资为主体的集团公司在进行企业的风险管理上，可以借鉴其他的优秀企业的经验，但需要融合企业自身的实际经营情况，从自身的投资业务出发，不断的完善集团的管理机制。随着投资主体的逐渐多元化和多层次化，投资集团的产权归属越来越复杂，需要进一步提升企业风险管理制度的包容性和独立性。
