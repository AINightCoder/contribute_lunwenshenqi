# 大数据背景下电商企业风险管理浅析

# ■丁新黑龙江大学

摘要：数字经济的快速发展，促进了企业规模的不断扩大，企业各部门业务活动的专业性与完备性受到极大的考验。大数据技术可以对企业的内外部信息进行深入的分析，帮助企业对所存在的风险问题做出积极有效的应对。本文就电商企业风险管理展开研究，立足于数字经济背景下电商企业所面临的主要风险，以拼多多为例，对拼多多风险管理的现状进行分析总结，并针对性地提出几点应对策略，以提升企业的风险应对能力，提高电商行业风险管理方面的整体水平。

关键词：电子商务企业；财务风险管理；防范控制；DOI:10.14013/j.cnki.scxdh.2024.01.012

大数据时代下，国家高度重视大数据的应用，根据国家“十四五”规划纲要，拟通过打造“数字经济”发展高地形成社会主义经济发展新优势，电商企业是大数据时代最具有代表性的企业。面临大数据带来的信息变革，企业要关注内外部经营环境的变化，充分利用大数据技术进行风险的防控。针对电商企业，业务活动过程中要扩大风险识别范围，重视风险的识别方法，不断提高企业管理人员的风险识别和防控能力，提升运用大数据进行综合分析、处理数据的技能。电商企业日常经营活动中通过大数据技术的使用，能够提升企业营运能力、风险防范能力，提高业务和财务人员的能力和素质的同时强化内外部安全建设。

# 一、大数据背景下企业风险管理的理论基础

# 1.信息不对称理论

信息不对称是社会经济活动中普遍存在的现象，在各个行业中都有所涉及。简单来说，信息不对称指的是在自由交易的市场中，各类人员对于有关信息的掌握存在差异，掌握信息比较充分的人员较信息贫乏的人员处于更为有利地位，信息不对称往往导致双方高昂的博弈成本。

# 2.风险管理理论

风险管理指的是通过对风险进行分析、识别、归类、整合，从而有效应对企业经营活动中所面临的风险问题，并将风险有效降低到可接受水平。全面风险管理作为系统性的活动，建立在对企业全业务和全流程的梳理上，是针对特定方面展开的风险管控。全面风险管理能够在一定程度上避免业务和流程方面的风险。

# 二、大数据背景下电商企业现状

# 1.电商行业现状

电子商务是指通过互联网以电子交易方式进行交易活动和相关服务活动，是传统商业活动各环节的信息化、网络化操作。一般认为电商一共有3种模式：平台电商、社交电商和私域电商。近年来，我国不断加大对电子商务行业的政策扶持力度，相应制定了多项战略规划和产业政策，不断扩大跨境电商零售试点范围、指导电商企业积极参与绿色技术研发项目，推动电子商务企业绿色发展。国家“十四五”规划纲要，将大数据与企业治理相结合，通过打造“数字经济”新高地形成社会主义经济发展新优势。电商是大数据时代最具有代表性的企业，而拼多多正是主打社交电商的一匹黑马。拼多多通过自身独特的商业思维和打法，短时间内吸引了大量的用户。但由于成长时间过短，其经营过程中也存在着很多问题，其中财务风险就是企业经营活动中各种风险的集中体现。

# 2.电商企业背景及现况

拼多多是一个专注于C2M拼团购物的社交电商，隶属于上海寻梦信息技术有限公司，于2015年9月正式上线，其经过短短十八个月的发展就已用户过亿，市值超过1800亿，并于2018年7月在美国纳斯达克证券交易所正式挂牌上市。拼多多用“农村包围城市”的方法起家，通过自身方向和社交裂变的方法，鼓励消费者发起和朋友、家人、邻居进行拼团，将不上网的老年人也拉上网，并用其独创的社交拼团及助力砍价的核心模式，主打百亿补贴、农货上行、产地好货，以更低的价格使消费者拼团购买需要的商品。2023年3月20日，拼多多发布了截至12月31日的2022年第四季度及全年财报。财报显示，拼多多第四季度单季收入398.2亿元，与上年同期营收272.3亿元相比，增长 $46 \%$ ；2022年全年营收为1306亿元，同比增长 $3 9 \%$

2022年在直播电商的冲击下，几大头部电商平台的流量出现了较大的波动。在此情境下，拼多多果断推出了跨境平台TEMU。TEMU自从2022年9月1日上线后，仅用四个月时间就进入了美国购物类App下载榜单前十。在美国市场打样成功后，今年TEMU顺势开通了加拿大站点，同时将触角延伸至澳大利亚、新西兰、英国及拉美地区。拼多多能在本土电商及跨境电商方面都获得傲人成绩，对拼多多的财务风险管理进行相应研究，不仅能带给其他电商针对性的思考，在一定程度上也能促进行业整体的健康、蓬勃发展。

# 三、电商企业风险管理概述

1.财务风险管理内涵及目的

从财务角度，风险是指由于各种难以预料或难以控制因素的作用，使企业的实际收益与预计收益发生背离的可能性，尤其指蒙受经济损失的可能性。风险管理的目的则是正确地估计和计量风险，在对各种可能结果进行分析的基础上，趋利防弊，以求以最小的风险谋求最大的收益；同时，还要善于识别、衡量、选择和获取增加企业价值的机会。

# 2.拼多多电商所面临的风险问题

# (1)交易双方的信任风险

拼多多主攻下沉市场，其口号是“拼着买，更便宜”。前些年由于淘宝、京东等平台进行战略调整，打击假冒伪劣商品和去除劣质店铺，一大批低端商家出局后被拼多多接收，为拼多多解决了供给端的问题。但是吸收了部分低质量店家之后，不可避免使“假货”“山寨货”在平台上泛滥成灾。走量化的价廉质低商品的路线虽然为其带来了大量的流量，但是拼多多忽略了广大用户对商品质量和店铺品质的重视程度。消费者通常认为“便宜没好货”，拼多多如何消除人们对便宜商品的这一顾虑，对增强平台的信誉能力是一个挑战。拼多多的营销策略采用了互联网经典的AARRR模型，通过百亿补贴、病毒营销等方式获取用户，由于依赖于社交平台助力拼单的模式，用户可能会出现个人的信息被不法分子盗用和电子钱包被盗刷的风险，严重影响到用户对企业平台的信任感。

# （2)过度依赖于社交平台的风险

AARRR模式下，拼多多通过微信流量传播激发用户的活跃度。拼多多购物的核心是通过社交裂变及助力砍价的方法，其分享拼单的模式依赖于微信和QQ得以实现。平台创立初期，拼多多获得了腾讯的独家分享权限，背靠腾讯的海量用户，坐拥社交关系信任的加持，拼多多获取新用户的成本被大幅降低，同时用户对新平台的信任得以提升，充分享受到微信、QQ的流量红利促进了拼多多的发展。随着微信小程序的兴起，拼多多又多了一种从社交平台获取用户的方式，这让其发展更为迅速。但是，从“小鹅拼拼”到“腾讯荟萃”，腾讯为了抢夺这一块市场“蛋糕”，从未停止过电商探索的脚步，虽然目前来看，电商市场的复杂程度显然比腾讯预估的高得多，但是，难保下一次腾讯不会亲自下场做社交电商。从TEMU方面来看，TEMU在海外失去了微信的加持，难以依靠低成本的流量裂变吸引用户，在目前获客成本较高的情况下，TEMU平台引流难度加大。

# （3）电商行业竞争加剧导致盈利能力不达预期

跨境电商行业竞争加剧，导致平台盈利不达预期的风险。随着拼多多的爆火，淘宝开发了特价版的App，阿里巴巴、京东等传统电商巨头也纷纷着力打造低价实惠的电商平台进入下沉市场，在头部电商企业自身具有规模效益和强大资本的背景下，下沉市场电商赛道日渐拥挤。消费者消费升级对拼多多也是一个隐性威胁。随着我国脱贫攻坚的顺利完成，经济不断蓬勃发展，低层次用户的消费理念在不断升级，消费者会更多地选择优质产品而不是便宜的产品，由此拼多多将面临用户大量流失的危机。TEMU方面，亚马逊在美国电商中市场占有率较高，其拥有高效率的自建仓配体系，购物享有当天或次日达的物流服务；而TEMU虽然将各个物流平台进行了整合，但直邮方式以及当地“最后一公里”的派送导致不仅耗费时间而且物流成本较高。受美国关税、环保等政策的管制，存在一定程度的业务发展不达预期的风险，其中包括履约物流、营销获客环节降本困难，美国用户对中国商品质量差的心智难以扭转的风险。

（4)会计信息化中的风险

虽然会计信息化对电商企业的经营活动而言，优势非常突出，但在大数据时代下仍然存在着一些漏洞，很多潜在的风险也会对会计信息化的使用形成制约。例如信息化的安全问题，信息化是运用计算机、网络和通信等现代信息技术完成相应的工作，由于计算机的虚拟性、网络的开放性以及相关法律法规存在漏洞，各个企业的加入使得企业信息大量累积，其复杂性不断攀升，而信息管理的体系不够完善，电商企业的用户信息庞大，整体而言，很难确保企业信息化的绝对安全。

# (5)跨境电商部分的风险

2022年在直播电商的冲击下，几大头部电商平台的流量出现了较大的波动。在此情境下，拼多多果断推出了跨境平台TEMU。跨境电商的出口，有助于释放国内经济活力、推动产业升级，但是也避免不了相应的风险挑战。风险主要包括以下几个部分：（1)竞争风险。早年的跨境电商主要依靠国内产品的高性价比来获取利润，但随着海外消费者对于产品的质量以及品牌等要求的提升，一味跟风去做产品很难再抢占市场份额。而且，跨境电商的供应链长，产品发货送货都需要更长的时间，如果没有提前做好储备的工作，很难快速跟上爆款产品的红利期，市场风向变化迅速，风险随之即来。（2)库存风险。跨境电商顺利运营的关键在于供应商稳定可靠，TEMU作为跨境电商平台，需要保证自身产品品质与图片说明基本吻合，需要考虑到商家及供货商是否能及时供货，并做好产品的仓储管理以及库存清点工作，以保障供货的过程的稳定。（3)物流风险。与国内电商相比，跨境电商出口卖家承担的物流风险更大，包括产品在跨越国境时可能会遇到被扣押等情况，在长时间的运输过程中也经常会由于天气等因素影响而面临货物破损、货物运输时间增加、货物丢失等难题。（4）法律风险。每个国家针对贸易活动等方面制定的法律法规不同，卖家在对外销售产品时，许多条例不受国家法律保护，卖家也难以在国外维护自已的权益。2021年亚马逊跨境电商服务平台掀起的封号潮就让许多钻空子的卖家账号被封，反而需要耗费大量时间和财力去申请解封。

# 四、电商企业风险管理的有效对策

# 1.加强企业内部安全建设

在发展电商的过程中，企业需要充分认识到大数据时代对整个市场环境的影响，同时还要深刻意识到国际上的复杂环境以及国际局势的多变性给企业带来的风险隐患。企业需要将风险管控视为电商业务发展的重要部分，在风险管理体系的安全建设上加大投入，完善企业财务系统工具的整体运用，加强涉及消费者交易安全的软件硬件建设，提升系统防火墙的安全性和可靠性。企业内部相关管理人员可以根据交易平台的特点，在用户浏览、支付等方面分析潜在的安全风险漏洞，通过交易监控系统对可疑交易进行实时管理，及时安排专业人员进行系统修复与改进，并与合作平台进行交涉，加强用户信息的保密工作，以尽可能地避免消费者及平台遭受财务交易风险。组织管理人员在一定时期内进行风控能力测评，通过分数对企业管理者的风险认识程度以及应对心得做出客观评价，使企业内部形成风控管理意识，重视并且有效应对企业经营活动中的各类风险问题。

# 2.强化核心竞争力

根据国家统计局的数据，截至2022年，网购占社会消费总额的比例已经提升到 $2 5 \%$ 以上。但是，国内生鲜电商目前的市场渗透率只有 $8 \%$ 左右。民以食为天，生鲜市场规模目前已经突破6万亿，如果把其中 $20 \%$ 迁移到线上，就是一个万亿级的巨量市场。由此国家提出相应的方针，而拼多多在果农的精准扶贫方面契合了国家的重大方针，其可以利用优势发展出自己一条独特的生态产业链，通过“百亿补贴”战略，推动农业现代化升级，进而增强企业自身的核心竞争力。其次，拼多多主攻下沉市场，以走量化的廉价低质商品起家，其产品质量、生产环境、售后监管等问题层出不穷，假货山寨货在平台上泛滥成灾，明显的违法行为容易导致平台在被查处的同时，对平台的市场占有率产生不利影响。所以，维持竞争优势的同时，强化核心竞争力，牢牢抓住下沉市场，同时抢占农业市场，是拼多多目前的重要任务。拼多多要加强产品控制和平台管理，加强商家资格的审查，严格监督审查程序，加强平台质量管理，强化核心竞争力。

# 3.实施专业化人才建设

拼多多商城投入大量资金用于吸引新用户，随着用户量的增多，商品种类和供应商也越来越多。在满足老顾客、留住新顾客使其成为忠实顾客、定期设置不同种类优惠活动使拼多多较其他电商平台具有较大价格优势的前提下，拼多多能够提升用户黏性。拼多多要紧跟大促节日的步伐，在大促期间加大活动力度。优惠活动数量的增多容易导致运营事故的出现进而给企业造成严重损失。为了避免运营事故，拼多多需要加强对现有运营人员能力和专业素质的培养，建立运营人员考核制度，只有通过考核的人才能上岗；并且为风险管理人员制定合理的职业发展通道，稳定薪酬待遇。另外，对于大促节日如“双11”的一些优惠活动与玩法，各大平台都会区别于往年，拼多多也不例外。建议拼多多在每年大促节日前期对运营人员开展紧急培训，让运营人员熟悉当年大促活动与玩法，从而减少大促期间出现运营事故的概率。引入职业的风险管控专员是许多企业正在推进的工作，外引人员会比企业原有人员提供更大的工作效力，并且不会受到企业自身关系的限制，能够在一定程度上帮助企业稳定风险。

# 4.加强企业风险管理合作

拼多多作为上市电商企业，可以充分利用第三方机构，对企业自身的业务财务活动进行定期检查并制定相应的优化措施。在财务活动方面，企业可以通过聘请专业的会计师事务所进行专门的审计；在税务方面，也可以聘请注册税务师等专业人士对企业的财税问题给予专门的指导；在合作方面，企业可以加强与对口高校的合作，通过在高校建立实训基地，有针对性地培养风险管理人才，这对企业进行持续性的风险有效管理具有深远的意义。

# 五、结语

综上所述，拼多多由于创新性地应用大数据技术带来用户数量和销售额的快速增长，但由于发展时间较短，其运营也存在一些问题。本文对拼多多如何加强风险管理提出了针对性的建设意见，即在保持核心竞争力的基础上加强质量管理，升级商业模式从而增强和巩固自已的市场地位。
