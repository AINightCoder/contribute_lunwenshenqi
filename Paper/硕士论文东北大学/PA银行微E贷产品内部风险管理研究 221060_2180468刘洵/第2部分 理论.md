
# 第 2 章 理论概述

# 2.1 风险管理的含义与特征

# 2.1.1风险管理的含义

论及风险管理我们首先要知道什么是风险，其指的是结果和目的之间的不确定性，其中有两层含义，不确定的收益是风险表现的一种，另一种则是把不确定的成本或者代价作为风险表现来强调。风险管理意指怎样在一个存在风险的环境中将其可能要造成不好的影响降到最低水平的管理过程。风险管理对现代企业非常重要，尤其是在信贷业务上至关重要，好的风险管理体系能够助力于降低决策失误和避免损失的产生。如何从减少风险的收益和成本来进行利弊均衡，再以此决策出哪些应对措施可以采取的过程，风险管理中包含了对风险的识别、评估和应对策略。银行主体对中小微企业进行信贷相关风险管理，具体是通过充分的风险识别、完善的风险评估机制和有效的风险应对及监控手段来对小微企业融资业务过程进行科学的管理，保持风险和效益的均衡发展，并且风险管理贯穿企业融资的全过程，从贷前评估、贷时审查再到贷后监控[7]。

用最少的风险成本来达成最大化的企业价值是风险管理目的所在，只有目的明确，才能使风险管理实现作用。风险管理就是以最小的成本来分散、转移和消除风险，使得损失降至最低，在使业务更具合理性的同时提升应对风险的效率和应对风险的效果，总结来说就是以最少的成本来获取安全保障的最大化。信贷风险管理也是如此，从追求利润到现在发展为在风险调整后实现收益最大化的模式，现在的风险管理是定性定量相结合的方式，商业银行在全面精准的识别与计量风险水平后采取有效的风险应对与监控措施来完成风险与收益之间的匹配，从而提升盈利与竞争能力[8]。

# 2.1.2风险管理的特征

风险管理的五个特征主要内容如下： $\textcircled{1}$ 战略性。在企业的各个活动中都有风险管理渗透，在风险管理过程中，视野要有足够的高度，在企业中主要运用在管理层，具备战略性作用，完善的风险管理的价值在于战略层面上的整合以及企业层面的风险管理[9]。 $\textcircled{2}$ 专业性。风险管理需要由专业的人来进行专业的管理。 $\textcircled{3}$ 全面性。企业全面风险管理是增加企业价值的过程，是需要企业的所有员工进行参与、目的是将风险控制到风险容量内。风险管理在企业的运用是一种实现结果的方式而其自身并非一种结果。要想确定达到此目标，需要在风险管理过程中使全体工作人员都有足够的风险意识并能够自觉行动[10]。 $\textcircled{4}$ 二重性。全面的风险管理不但要对纯粹的风险进行管理，还要去管理机会的风险。 $\textcircled{5}$ 系统性。为了能够保证风险管理整体目标的实现，系统且规范的方法是其必备的，风险管理的内部控制和信息系统、风险管理的策略与其职能系统都包含其中。

# 2.2 风险管理过程

风险管理的全程主要有风险识别、风险评估、风险应对和风险监测四项环节。首先是风险识别，风险管理以其为基础，要进行风险管理就必须能够识别出风险，以此来确定哪些风险会对企业产生影响，只有全面精准的风险感知和风险分析才能够对各种风险要素进行综合性的评估，以此来确定风险发生的概率和程度。并在风险识别和风险评估后采取合适的、有效的应对措施，并在主体实际经营过程中监测风险的变化，再进行识别和评估来动态调整应对措施[11]。

# 2.2.1 风险识别

风险识别是先初步判断某个业务可否进行，这个环节是最先进行的并且其最为重要。风险识别主要的作用是能够防患于未然，它指的是风险产生之前采用科学的工具和方法能够系统的识别到业务所面对的各种风险与对风险产生的原因进行分析，只有在事前精确地进行风险识别，才能后续采用精确有效的应对方式。风险识别整体囊括了感知风险与分析风险这两大环节。风险识别是基于感知风险来进行的，是指要认识和知悉客观存在的各种类风险，此过程十分重要，仅有准确地感知到了风险，才能继续去分析感知到的风险进而来找到风险产生的潜在因素和引发因素，以此才能制定出合适的解决方案，所以感知风险是服务于风险管理决策的。分析风险是在感知风险后的，全面地分析引起风险的各种因素的过程，它可谓整个风险识别至关重要之处[12]。

对于风险识别的研究，分别有以下三种经常使用的方法：德尔菲法、风险核对表和 SWOT 分析法。 $\textcircled{1}$ 德尔菲法。德尔菲法是管理部门中预测意见的方法之一，匿名函询法是它的本质，通过对相关业务风险管理的专家以匿名的方式进行意见征集，并归纳整理专家们回馈的意见，再将结果反馈各个专家并请他们进一步发表意见，之后再收回再反馈，依次进行多轮，得出较为一致的意见，就可以看到对业务风险评估的看法。 $\textcircled{2}$ 风险核对表。在之前同类的业务信息和相关的其他信息的基础上制作的风险识别核对表，将此业务可能存在的潜在风险均列于表中，以便工作人员能够核对某个业务是否存有表中列出的或者是相似的风险，风险运用风险核对表来进行识别，其优势是简便快捷，因表中所列出风险均是历史业务所总结而来，所以缺点则是受到相关业务可比性的限制。 $\textcircled{3}$ SWOT 分析。SWOT分析是系统性的识别工具，又被称作态势分析法，是用来研究和分析一个项目或者是单位现实情况的方法，且较为客观与准确。通过识别业务的Strength（优势）、Weakness（劣势）、Opportunity（机遇）和 Threat（挑战），排列形式是按照矩阵来排列，将这些因素进行匹配并加以分析，得出相关结论，多角度系统性地对业务经营的风险进行识别[13]。

# 2.2.2 风险评估

风险评估是基于风险识别开展而来，是在风险识别时全面且精确地感知种种风险因素并进行分析的基础上，对识别到的风险的发生概率以及也许会造成的损失程度等因素做综合性考量，来确认风险产生的程度和可能性，并将评估的结果与标准化制度进行比对来确认是否采取控制措施以及程度。在风险评估中，把定量分析与定性分析相结合。方法的选择上，定性分析法主要包含专家调查法、层次分析法两种方法，定量分析法中主要有模糊综合评价法、敏感性分析法、决策树分析法、概率分析法四种方法。以上6种方法中，各有利弊，进行合理的选择可以判断风险发生的可能和程度，进行后续的应对措施选择，并随着企业的日常运营进行动态调整，及时了解企业的风险因素变化[14]。

风险评估主要是通过定量和定性分析完成的，二者之间的关系是相辅相成的，两个方法是互补的，没有定量的定性分析是缺乏科学性的，而没有定性的定量分析是没有意义的，二者在风险评估的过程中通常是结合使用的，从而使得风险评估的结果更加准确。通过对两种方法的配合使用，我们进行风险评估的时候不能只是简单地将指标结果相加进行分析得到结论，而是应该将每个指标对于结果的影响程度计算出一个权重，再来计算其对结果的影响。风险评估方法主要使用以下这两种方法进行[15]。

（1）层次分析法

层次分析法是按照目标、准则和方案这几个层次把与决策所相关的全量元素做分解，并使用定量与定性两种方法对此进行解析的决策方法。层次分析法视所研究的目标为一整个系统去解析和做比较，是用一种综合化思维的方法去做评估，不会分割各个具体指标对于最终结果的影响，每个层次中项目对于结果的影响是很清晰的，是用权重进行量化的，是很系统性的评价[16]。层次分析法是定性分析和定量分析的结合，相对于仅定量分析的方法更加追求定性的分析和主观的判断，是很贴近于决策过程的思维方式。

# （2）模糊综合评价法

模糊综合评价法是在模糊数学的基础上进行综合评估的方式，通过模糊数学隶属度理论，这个综合评价法能够转化定性的评价为定量的，也可以说是对某个对象或者受到很多因素所影响到的事物应用模糊数学来进行总体的评价。它的特别点是有很强的系统性并且结果清晰，对于难以量化甚至是模糊的问题都能够很好的解决，非确定性类的问题很适用这种方法解决[17]。该方法的具体步骤首先是设定各个层级的评价因素，然后是确认评价的细则，也就是确定一个建立于评估的因素和价值间的对应关系。邀请专家组进行评价给出评价值的方式，通过评价因素间的相互比较，比较基准是以最佳的评价要素为基础来对其他各个评价因素给出相应评价值。虽不能完全准确量化统计潜在风险，但是可以通过对模糊综合评价法的结合来预估风险可能会产生的结果以及风险的性质。

# 2.2.3 风险应对

风险应对是在风险识别和风险评估后，即确定了业务主体存在的所有风险因素和这些风险出现的概率及程度得到综合的评估后，根据风险的性质及主体对于风险的承受能力而采取的应对风险的策略，对于可能出现的授信风险及时预警并有多种应对方案有效地降低资金损失，以保护其自身的资产和利益，并确保其稳健的经营和持续发展。四种风险应对的方式分别为规避风险、降低风险、接受风险以及分担风险。

（1）规避风险。为了对将来有可能发生的风险事件进行规避而采用消除风险的方式，为了阻止风险高的经营活动主要可以通过公司相关政策、制度的限制来完成。规避风险是最彻底、最保险的风险应对方式，但是规避风险也意味着规避业务、降低利润、减小整体规模。一旦有不良业务产生，也会因为规模不高导致不良率更高。规避风险是应对方法中最消极的一个选择，也是最后的选择[18]。

（2）接受风险。是指不采取任何行动而维系现有的风险水平的一种做法。接受风险是损失的可能和程度在可控制的范围内，即使发生风险，按照既有的应对方法应对，所能造成的最大损失对承担风险方影响不大或有其他对冲手段可在该事件发生的同时为承担风险方获取足够利益，用以抵消风险发生的损失。即在接受风险的同时，可通过合理的组合工具抵消风险[19]。

（3）降低风险。降低风险主要是通过相关的政策和采取一些措施使得原本无法完全控制、变量较多的风险降低到可以接受的水平上，比如可以通过定期的检查，边检查边动态调整执行方式，改变授信额度、期限、还款方法来降低风险。分担风险主要是采取风险转移的方法，将既定的风险转化给另一方，风险转移的形式有合同签订和保险购买这两个[20]。

（4）分担风险。分担风险是指以一定的代价把风险的负面后果与风险管理责任转移到第三方。这个第三方通常具备更好的能力和资源来处理风险。在企业中，常见的风险转移方式包括保险和外包。进行成本分析是确定成本分担的重要步骤。如果风险转移的成本少于应对风险的成本，就可以选择进行转移。然而，如果风险转移的成本高于内部处理风险的成本，则更适合保留该风险，并制定相应的风险缓解措施[21]。

# 2.2.4 风险监控

风险管理体系的所有环节中最后一个是风险监控，也可谓必不可少的一个环节，风险监控主要是监测业务进展和环境，是决策主体对于识别到的风险的发展与变化情况进行全程全面监测，因为风险不是一成不变的，它会根据环境的变化而变化，所以要根据实际情况进行应对策略的调整，对应对风险的措施的结果和之前的预测是否一致进行核对是监控的目的，也就是在企业的实际经营过程中，对风险发展情况和是否有新的风险产生进行全面的监测，并动态调整应对措施，以及对新的风险再次进行识别、评估和应对[22]。风险监控可以对风险应对策略不见效及新产生的风险进行有效控制。
