
# 第 1 章 绪论

# 1.1 选题背景及研究意义

# 1.1.1 选题背景

在国民经济发展过程中，能源扮演着举足轻重的角色。作为全球最大的能源消费国，中国占据了全球 $23 \%$ 的能耗。相较于其他国家，中国煤炭资源丰富，但原油资源有限。燃煤释放大量有害物质，严重污染空气，导致大气及环境质量急剧下降。

2021 年 3 月，中国政府出台了《中华人民共和国国民经济和社会发展第十四个五年规划和 2035 年远景目标纲要》，明确提出了构建现代可再生能源体系的战略方向。该战略着重强调了非化石能源的深化发展，旨在促进集中式与分布式可再生能源的高效整合与利用，同时加速绿色可再生能源的广泛普及。国家将大力推进清洁能源发电项目的实施，构建集多能互补功能于一体的清洁能源生产与培训基地。根据规划目标，非化石能源在电力消费结构中的占比将突破 $20 \%$ 。随着国家持续推进碳达峰与碳中和战略，中国光伏发电技术，特别是分布式光伏领域，将迎来显著增长期，预计将启动大规模光伏发电项目建设，推动行业进入快速发展轨道。

面对宏观经济形势的减速，中国发电工程的建设正遭遇新的挑战。尽管国家高度重视此类项目的推进，但因经验积累尚显不足，仍有待持续发力促其发展。为加强电力行业的治理效能与优化能源内部配置，国家计划增加投入，以改革现有电力系统结构，追求更高效率的增长路径。响应国家淘汰落后产能的政策，发电系统的管理策略正逐步完善，以适应未来的市场需求和发展趋势。因此，即将启动并建设众多采用先进科技、具备多种功能的分布式发电站项目。这些项目将注重节能减排，实现低能耗和极低排放的目标，为国家的绿色低碳转型做出贡献，同时也确保电力供应的稳定与可持续性。然而，这些项目建设过程的复杂度和潜在风险随之提升，因此有必要加强建设项目管理，以保障其安全及平稳运作。

本文依据风险管理理论，运用合理的风险识别和评估方法，找出项目执行过程中的风险，对这些风险进行了深入的分析和全面的评估，制定一整套有效的风险应对策略和措施，保障分布式光伏发电工程建设的安全与稳定，并力求实现经济效益最大化，同时，为L公司日后光伏发电项目或其他新能源发电项目，提供一条清晰而有效的风险管理路径。

# 1.1.2 研究意义

# 1.1.2.1 理论意义

本研究对 L 公司光伏发电项目的风险进行了深入且全面的剖析。不仅可以丰富了光伏发电项目风险管理的理论体系，为后续类似项目的风险管理提供了坚实的理论支撑和方法论指导，同时也为其他清洁能源项目的风险管理提供了有价值的理论借鉴。通过构建风险识别模型和风险评价体系，本研究为风险管理领域注入了新的理论活力，推动了相关学术研究的深入发展。

# 1.1.2.2 现实意义

首先，通过对 L 公司光伏发电项目风险识别与评估，能够更有效地提升项目的风险管理水平，确保项目的平稳推进。其次，本研究提出的风险管理方法有助于通过迅速识别并妥善处理可能存在的安全威胁，为项目的持续安全与平稳运作奠定坚实的基础。最后，在当前国家积极推动清洁能源发展的大背景下，该研究有助于L公司全面识别光伏发电项目的潜在风险，从而制定更为精准有效的风险管理策略，进而提升企业的风险管理能力，确保项目顺利进行并最大化经济效益。同时，通过优化资源配置和应对高风险领域，企业能够提高投资回报率，实现可持续发展。

本研究为同行业企业提供了光伏发电项目风险管理的宝贵经验和范例，有助于推动整个行业风险管理水平的提升。同时，通过揭示当前技术和管理上的不足，该研究将激励同行业企业加大技术创新和产业升级的力度，进而促进整个行业的健康发展和竞争力提升。

# 1.2 文献综述

# 1.2.1项目风险识别

在项目风险识别与管理领域，随着全球经济的不断发展和项目复杂性的日益增加，对风险的有效识别、评估及应对策略成为确保项目成功与可持续性的关键因素。特别是在PPP（政府和社会资本合作）项目、基础设施建设、能源开发以及特定行业（如养老房地产、变电站建设、污水处理等）中，风险识别与管理的重要性更是不言而喻。

李丽（2016）等研究深入探讨了 PPP 项目在全生命周期内的风险演变规律，指出在项目识别、准备、采购、执行和移交等不同阶段，PPP 项目面临着不同的风险挑战，如政策变动风险、融资风险、建设风险、运营风险及市场风险等[1]。王剑（2017）针对PPP 融资模式下的项目风险，提出了更为具体和细化的风险识别与控制策略。强调了融资结构设计、合同条款制定、风险分担机制及监管体系构建等方面的重要性，并指出这些环节的有效管理能够显著降低 PPP项目的融资风险。季勇（2015）针对变电站建设项目，从技术、经济、环境和社会等多个维度进行了风险识别与评价，强调了变电站选址、设备选型、施工安全及环境影响等方面的风险因素，并提出了相应的风险应对措施。刘素坤和郭丽（2018）专注于污水处理 PPP 项目的风险识别研究。研究指出，污水处理PPP 项目特性显著，包括大规模投资、长期运营周期与复杂技术手段，这些特点共同构成了 其面临的 高风险 与挑战性 [2]。周镇浩等（2021）的研究创新性地采用了HHM-DEMATEL-ISM 模型对农业基础设施PPP项目的风险进行识别与建模。通过该模型揭示了项目风险之间的内在联系和层次结构，为项目管理者提供了清晰的风险认知框架。该研究不仅丰富了 PPP 项目风险识别的方法论体系，也为 PPP 项目的风险管理提供了新的思路和方法。王卓甫（2018）及其团队开展的研究，确立了一套针对跨国重大基础建设项目的风险辨识与评估体系[3]。他们强调，这类项目因其巨额投资、长期建设周期以及复杂的技术要求，自然会遭遇更为严峻的风险考验。通过构建全面的风险识别与评估体系，该研究为企业在海外进行基础设施建设投资提供了全面的风险管理指导，有助于降低投资风险，提升项目成功率。

# 1.2.2项目风险评估

在工程项目风险识别完成之后，紧接着的步骤是对关联因素进行深入评估。当前的国内外研究多侧重于定性与定量分析相结合，基于 Raftery 与 Akintoye（2020）对项目风险评价的先驱性工作，Shen Li-Y'm（2019）进一步考量了工程项目的技术特性与需求，融入了敏感性分析、概率分析等实用方法，利用专家见解量化风险发生的可能性，揭示风险类型及其影响程度[31]。Aminbakhsh 等人（2022）则采用层次分析法（AHP），结合太阳能发电工程与网络分析，通过确定其重要性顺序来探寻潜在风险，这一方法已被欧洲一太阳能电站工程的成功案例所验证[32]。Beltran（2005）等人提出了基于动态方法的项目风险管理模型，对项目的动态变化进行系统研究。此模型随后被应用于特定的实际项目中，产生了显著的风险评价成果。Leu（2022）及其团队创新性地融合了蒙特卡洛模拟与敏感度分析方法，以此对某一工程项目的安全风险进行评估[4]。根据他们的分析结果，上述方法能够揭示我国能量供给短缺的风险可能性，展现出显著的实际应用价值。另一方面，Van Weyenberge Bart（2015）等学者以贝叶斯理论为核心，构建了一套基于事件树的风险评价系统，并成功将其应用于城市轨道交通建设领域[33]。通过对该方法的深入研究，得出结论：此法在量化项目风险发生的概率方面表现出独特优势。其关键在于能够追根溯源，深度剖析风险形成的原因，从而为风险策略的制定与实施提供宝

贵的参考和指导。

刘晓君（2016）在其相关研究中，采用层次分析法构建了一个系统化的房地产企业风险评价指标体系，并结合模糊综合评价方法，对影响因素的作用力度进行了详尽的探讨与分析。研究强调，在投资风险评估环节融入数理统计技术，既能有效地应对模糊性问题，又能显著减少投资失误带来的损失[5]。李尚宇（2023）提出，项目的成功与否很大程度上依赖于风险识别的量化程度、风险分配策略以及风险管理机制的完善性。项目投资中的风险分配并非单一化的过程，需要综合考虑投入资本量、预期收益等关键因素，以实现资源的有效配置与风险的合理承担; 针对投资需求及承受能力的平衡，郝权军（2020）从工程项目外部视角出发，利用贝叶斯网络工具，深入解析了外部影响因素，产出严谨的分析结果。云棋峰（2023）则通过模糊综合评估法，对项目设计、施工、运营等全流程决策进行探讨，构建了全面的风险评估体系，并验证其效能。赵玲艳（2018）及其团队基于层次分析法（AHP），对施工项目特性与风险进行了系统性评估。该评估涵盖质量风险分析、建立评估机制以及提出改进策略等关键环节[6]。翟彦彦等人（2019）同样运用 AHP 方法，采取逐步深入的方式量化项目施工质量，通过分析各因素权重，提出了一系列操作性强且有效提升质量的措施。李彦斌（2021）及其同事，结合灰色关联分析与TOPSIS 法，从政策、市场、资源等多角度审视当前光伏项目面临的挑战，深入分析项目内外部环境，给出详细的评估及建议。

Monteiro C（2006）及其团队采用了遗传算法来评估风险模型，立足实践，深入探讨了电力设施从设计、建造到运营阶段可能出现的不稳定性问题，并提出了针对性改进策略[34]。Liu J（2008）等人指出，企业在应对风险时，往往过于关注项目的经济效益，而忽略了风险识别与评估的重要性[7]。Liu J（2020）等则基于中国太阳能发电工程投资的特性，建立了一个风险评估体系，通过优化层次分析法，量化风险权重，从而显著增强了投资者的风险认知，为获取投资回报提供了有力支持。Silva Pereira（2014）等利用蒙特卡罗方法分析了并网光伏发电的风险，这一研究不仅指导了亚马逊地区太阳能并网工程的成功启动与有序执行，也为后续类似项目的实施提供了宝贵经验[35]。自Wig（2016）及其团队开创性地引入模糊逻辑下的实物选择权理论，对市场上潜在风险进行量化评估以来，研究领域逐步深化[8]。最小二乘蒙特卡洛法被广泛应用于对太阳能项目的风险进行全面细致的分析与评估，Khalid（2018）等人则通过模拟不同区域的太阳辐射数据，借助敏感性分析手段，深入探讨了各区域的技术性风险，并最终提炼出有效的风险管控策略。任明哲（2020）则综合嫡值理论与灰度系统方法，对东安科技园区内的3MW 太阳能屋顶光伏发电项目的风险展开深入研究。尽管研究成果表明整体风险相对较低，但仍需充分考量影响因素。朱志强（2021）等人的工作聚焦于工程建设项目的风险管理，他们从投资比例、风险承担与预期收益的视角出发，构建了一套投资决策偏好模型，通过博弈论解析，揭示了均衡状态出现的可能性[9]。于心怡（2019）则针对太阳能发电领域的投资特性，精心设计了风险评估模型，并应用灰色关联度分析法对其投资风险进行了细致评估。在太阳能电站投资建设过程中，他们针对项目特有的风险要素，系统构建了评估指标体系，并依据不同项目的特征与实际需求，选择与之相匹配的评估方法，从而提升投资决策的科学性与可操作性。

# 1.2.3项目风险应对

在国际上，项目风险管理和应对研究开始的较早，M.von $\bullet$ Rheinbaben（1915）在其著作《企业风险论》中，首次系统地提出了保护企业免受风险影响的理论框架。他认为企业面临的风险并非一成不变，因此需要根据实际情况灵活运用多种风险处理手段，包括控制、分散、转移、回避和抵消等策略。这种结合具体情境的风险管理方法，为后来的研究奠定了坚实的基础。Henri Fayol（1916）把首次风险管理的思想引入到企业管理之中[36]。随后，美国学者 Frank·H $\bullet$ Knight（1921）在其作品《企业管理》中引入了风险负担管理的概念，进一步丰富了风险管理的内容[38]。他区分了可预见的风险和不可预见的不确定性，并提出了风险转移和风险排除两种处理风险的方法，这一观点强调了管理者在面对不确定性时的主动作用，即通过专业化的知识和技能来预测和应对风险。进入到本世纪，随着工业化和企业经营复杂性的增加，项目风险管理逐渐成为学术研究和企业实践中的重要领域，美国项目管理协会（Project Management Institute,简称 PMI），将项目风险定义为：一种不确定的事件或者条件，一旦发生至少对项目的一个目标产生影响，项目风险应对的研究随着项目风险管理的研究不断演化。无论是我国研究者还是国外学者都在致力于风险应对的探索。高珍（2021）当前风险应对方法主要包括以下几种：基于风险分类矩阵选择最优项目风险处理方案、通过工作分解结构（WBS）筛选合适的风险应对措施，以及利用优化模型法确定风险控制措施。这些方法旨在高效识别并妥善处理潜在风险。例如：Flanagan 和 Norman（2023）、Elkjaer 和 Felding(1999)等基于风险分类矩阵来进行风险应对方案的选择，Ben-David 和 Raz(2021)、Kayis（2022）和Zhang 和 Fan（2014）通过采用优化模型，探索了在特定约束条件下的策略组合，以实现目标函数的最大化，从而确定了项目风险应对方案的最优组合[37]。此外，还有吴登生（2015）与杨莉、李楠（2021）的研究成果已成功地应用于软件类项目的风险应对策略选择方法上，展示了风险应对方案选择方法在特定行业中的应用价值[10]。

# 1.2.4 文献述评

综上所述，目前的研究主要集中在风险识别、风险评估和风险应对策略等方面。国内外学者运用了多种方法，通过细致地运用包括层次分析法、模糊综合评价法、蒙特卡洛模拟在内的多种方法，光伏项目面临的各种风险得到了全面且深入的评估与分析。这些研究不仅涉及政策风险、技术风险、市场风险等传统风险领域，还包括了环境风险、运营风险等新兴领域，显示出光伏项目风险评价的全面性和深入性。在评估光伏发电项目的风险时，专家们采用了综合性的方法，结合定量分析与定性分析，全面审视项目内各种风险，并为此制定了应对措施。这些措施涉及风险避免、风险转嫁、风险共担以及风险管控等多维度策略，为光伏项目的风险管理提供了有益的参考。

尽管光伏项目的风险评价研究已经取得了一定的成果，但仍存在一些不足之处。目前的研究主要集中在理论层面，缺乏实证研究的支持，导致理论与实践之间存在一定的脱节，使得研究成果的实用性受到限制。其次，现有的研究往往侧重于某一方面的风险评价，而缺乏对光伏项目整体风险的综合评价。这使得决策者难以从全局角度把握项目的风险状况，从而影响决策的科学性和准确性。因此，本文采用多种研究方法相结合的方式进行综合分析，提高了研究的准确性和全面性，注重实证研究与理论研究的结合，使得研究成果更具实际意义。

# 1.3 研究内容与方法

# 1.3.1 研究内容

本研究选取 L 公司的光伏发电项目为研究对象，系统探究其风险管理机制。研究以风险管理理论体系为基础，采用文献研究法对项目潜在风险源进行了系统性识别与初步分析。通过德尔菲法对风险因素进行筛选与整合，构建了项目的核心风险指标体系。随后，运用层次分析法对各项风险指标的重要性程度进行量化评估，并结合模糊综合评价法对项目风险状况进行多维度综合评估。基于评估结果，研究从项目启动期、建设期及运维期三个阶段提出了相应的风险防控措施，为项目的顺利实施与长效运营提供保障。

第一章：绪论部分。本章旨在深度剖析建设项目风险管控的背景与重要性，以及相应的策略方法。同时，本章从理论和技术层面对分布式光伏发电项目进行了全面梳理，深入探讨了光伏发电的工作原理、系统分类、技术特征及可能存在的风险因素，为后续研究奠定了坚实的理论基础，提供了具有实践指导意义的参考依据。

第二章：理论框架与相关研究概述。本章深入阐述了项目的核心概念与本质意义，并对其风险管理机制进行了详尽探讨。同时，系统性地对分布式光伏发电项目的理论与技术展开了研究，内容涵盖光伏能源的工作原理、类别划分、特点及潜在风险等，旨在为后续研究工作提供有实际价值的参考信息。

第三章：L 公司光伏发电项目风险识别。本章重点阐述了L公司光伏电站建设项目风险识别的核心价值与实施路径。研究采用文献分析法与德尔菲法相结合的方式，系统收集并剖析了可能制约项目建设和运营的关键风险因素。通过这一研究过程，准确识别出了该光伏电站项目的核心风险要素体系。

第四章：L 公司光伏发电项目风险评价。在风险识别的基础上，本章节采用层次分析法（AHP）构建风险评估模型，结合模糊综合评价（FCE）方法，对项目各风险因素进行权重赋值与系统性评估，从而得出项目的整体风险水平评价结果。

第五章：L公司光伏发电项目的风险应对措施。基于第三章中划定的项目风险要素列表与第四章对项目风险的评估成果，本章节采用纵向视角，覆盖项目从规划、实施到运营的整个生命周期，针对每个阶段可能出现的风险，提出针对性的处理策略。此举旨在全面而有效地管理项目风险，确保项目整体运行的稳健性与可控性。

第六章：研究结论和展望。本文旨在对研究框架、实施策略及所得结论进行归纳提炼，阐述其学术价值，并识别研究过程中的不足之处，以期为后续分布式光伏发电项目的探究活动提供借鉴与启示。

# 1.3.2 研究方法

本研究通过文献研究法对 L 公司光伏发电项目的潜在风险要素进行了初步识别，随后运用德尔菲法对这些风险要素进行了系统筛选与整合，最终构建了项目的风险要素清单。在此基础上，采用层次分析法对分布式光伏工程的各风险要素进行了权重赋值，并通过模糊综合评价法对项目整体风险状况进行了系统性评估。

（1）文献分析法

在系统梳理现有风险管理理论体系的基础上，本研究通过整合分析国外相关文献，构建了适用于本领域的风险识别理论框架。结合项目研究的具体特点，对相关数据进行了深入挖掘与分析，重点提取了对研究具有重要价值的信息资源，以充分发挥其研究效用。通过系统的文献收集与整理工作，初步建立了风险因素识别体系，为L公司光伏发电项目风险管理研究提供了可靠的理论支撑与实践依据。

# （2）问卷调查法

通过问卷调查广泛收集 L 公司光伏发电项目各相关利益方（涵盖项目管理人员、技术人员、政府代表及投资者等）对项目风险的见解，旨在验证并补充前期通过文献分析和德尔菲法识别的风险因素，确保风险识别的全面与精准。实施步骤包括：设计问卷，涵及筹建、建设、验收及运营各阶段的风险因素及其影响；选定代表性广泛的样本群体，确保数据的全面性；采用多种方式发放问卷并收集数据；最后，分析问卷数据，提取关键信息，为后续风险评估奠定坚实的数据基础。
