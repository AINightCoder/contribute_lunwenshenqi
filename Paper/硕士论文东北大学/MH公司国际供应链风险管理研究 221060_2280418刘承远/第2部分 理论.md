
# 第 2 章 相关理论和方法

# 2.1 供应链理论

# 2.1.1 供应链的概念和目标

供应链是一种以加强战略合作为目的，通过获取原材料，把半成品加工成产品，并将产品送给客户过程中所涉及到的一些企业或部门进行有机集成的功能网链结构模式[36]。供应链是动态的，涉及不同环节之间信息、产品、资金的持续流动[37]。

供应链管理是通过研究供应链的构成、运作、协调等各个方面并加以实践，最终形成一套新的经营和运作模式来管理供应链[38]。

供应链管理的目标是其整体盈余的最大化。供应链成功与否并非由单个环节的利润来衡量，而是要看供应链整体产出的净值。过度强调单个环节的盈利能力可能导致供应链总盈余减少[37]。

# 2.1.2 供应链的驱动因素

影响供应链绩效的驱动因素包括设施、库存、运输、信息、采购和定价。每个驱动因素都会影响供应链响应性和成本之间的平衡，进而影响供应链与企业战略的匹配。因此，供应链设计者必须合理构造这六个驱动因素以实现供应链管理的目标[37]。

（1）设施：包括仓储设施和生产设施，是供应链中存储和生产产品的场所。设施的数量、选址、产能、柔性等规划对供应链绩效有重大影响。企业可以通过增加设施数量、提高产能和柔性来降低供应链运输费用或提高响应性，但每一种做法都会产生相应的成本。

（2）库存：库存包括供应链上所有的原材料、在制品和产成品。库存会影响供应链所持有的资产、发生的成本以及响应性。供应链管理者所需做出的主要库存决策包括周转库存量、安全库存量、季节性库存量、产品可获性水平等。

（3）运输：运输使产品在供应链上实现了点到点的移动。其可采取多种方式和路线的组合，每一种组合都有不同的绩效特点。对于高价值的产品可以采取快速运输以获得响应性，同时集中设施和库存以降低成本。对于低价值且需求量大的产品，可以在接近需求的地点设置更多设施和库存，然后用低成本的运输方式补货。

（4）信息：信息传递是供应链运作的基本前提，更好的信息共享有助于提升供应链响应性并降低成本。管理者需要在信息系统的复杂性及其产生的效益之间找到平衡，供应链中分享的的信息越多，所需的基础设施和后期处理成本就越高。

（5）采购：采购是决定由谁来从事特定的供应链活动，哪些职能在企业内部完成，哪些职能外包出去。对于原材料和零部件采购来说，供应商的价格、质量、配合度、付款方式、所处地点均很重要。对于外部合作来说，若新加入者能够提升供应链总体盈余，同时又不显著增加风险，则外包是有意义的。

（6）定价：定价就是企业决定对其在供应链中提供的产品或服务收取多少费用。定价会影响买方行为，从而影响需求和供应链绩效。当需求大幅波动但供应链缺乏柔性时，定价可以成为调节需求的有效杠杆。同时，差异定价也可使客户各取所需，而企业则在其中获得更多收益。

# 2.1.3 供应链管理的主要问题

供应链管理的主要问题包括结构性问题、协调机制问题、随机问题和全球化问题[38]。

（1）供应链的结构性问题：需要测算规模经济的水平，进行生产和物流设施选址，评估技术方案，进行产品决策，建立联盟网络等。

（2）协调机制问题：需要明确制造商与供应商的协调机制，制造商与客户的协调机制，制造商与国际货运代理、报关行的协调机制，供货保障职能部门与库存控制职能部门的协调机制等。

（3）随机问题：需要评估供应商可靠性，不同种类运输方式、口岸、承运人的可靠性，价格和汇率波动性，以及固定成本、可变成本、交货提前期、供应商配合度等问题。

（4）全球化问题：需要了解相关国家的贸易政策、海关政策，评估贸易壁垒、税收、政治环境等方面的差异，近年来甚至还要考虑公共卫生政策、地区冲突、海域封锁、劳工运动等方面对供应链的影响。

# 2.1.4 供应链管理的基本思想

供应链管理的基本思想包括系统思想、专业化思想、合作性竞争思想、集成信息化思想、满意度导向思想等[37-38]。

（1）系统思想：该思想主张应将供应链视为一个整体，而不应将供应链看作由供应商、制造商、客户构成的、彼此分离的利益体，或由采购、制造、分销等部门构成的一些分离的功能块。供应链上的各企业、各部门之间应当分享预测、订单等信息，进行协同计划，而非单纯追求供应链某个环节的利益最大化。为供应链整体盈余做出贡献的企

业，应当从其它企业获得补偿。

（2）专业化思想：企业应当聚焦核心竞争力，对非核心业务进行外包，由专业的企业做专业的事，从而降低成本、提升供应链盈余。应减少供应商数量，与业务伙伴结成战略联盟关系，以实现信息共享、风险共担。应重视物流企业在供应链管理中的专业作用，在国际供应链中物流企业对缩短交货周期的贡献往往比制造企业更大，而且物流企业的服务质量最终会影响整条供应链的协调性。

（3）合作竞争思想：过去的竞争对手在保留竞争属性的情况下也可以在供应链的局部开展合作，通过协同效应降低成本，扩大市场份额，从而达成对其它未结盟对手的竞争优势。例如主机厂共用整车平台、供应商和零部件，零部件企业协同采购、共用物流渠道、进出口和关务操作资源等。

（4）集成信息化思想：供应链管理的目标是提升整体盈余，其中包括采购价格、运输成本、库存成本等。在传统管理中，这些成本分散在不同企业、不同部门和不同环节，难以全面掌握和评估。因此，合理的供应链管理必须将商流、物流、信息流、资金流加以整合，借助信息化手段使其全面、实时呈现，如此才能便于供应链管理水平的提升。

（5）满意度导向思想：供应链管理将整条供应链看作一个整体，客户/供应商满意是客户/供应商盈余的重要指标，提示我们关注合作伙伴的利益，这是供应链长期健康发展的必要条件。所以，供应链管理高度重视满意度，以此为抓手建设长期性、战略性的供应链合作关系。

# 2.2 风险管理理论和方法

# 2.2.1 风险和风险管理

风险是某些不确定性事件以及由其可能引起的偏离预定目标的不良后果的综合。它是不良后果发生概率 P 及其后果 E 的函数，用数学公式表示为： $\mathrm { R = f } ( \mathrm { P } , \mathrm { E } ) ^ { [ 3 5 ] }$ 。显然，风险包括三个基本构成因素：事件，该事件发生的概率和该事件发生后产生的不良后果。

有些不确定性与目标无关，那么这样的不确定性就无须进行管理，因此不属于风险研究的讨论范围。有些不确定性虽然与目标相关，但其带来的是正偏移而非负偏移，也即是说这些不确定性带来的是一种有利的收益而非不良的损失，也不属于本文狭义风险研究的讨论范围。

风险管理是管理人员对战略、业务各环节可能遇到的风险进行预测、识别、分析、评估，并在此基础上提出对策、采取措施、减少损失，从而实现管理目标的科学方法。

# 2.2.2 风险管理的步骤和方法

风险管理的基本过程包括风险识别、风险评估、风险应对与风险监控。

（1）风险识别：通过开展组织的工作结构分析（WBS）确定风险的来源、风险产生的条件、风险特征以及是否会对本企业业务开展造成负面影响。主要应用的方法有流程分析法、头脑风暴法、鱼骨图分析法等。

（2）风险评估：主要是对上一步识别出的风险进行定性或定量分析，评估风险发生的概率和对企业运营目标的影响程度。对风险的严重程度进行分类和排序，以便找到主要风险点，优先加以应对。常用方法有历史数据分析法、问卷调查法、德尔菲法、深度访谈法、主观评分法、风险矩阵法、层次分析法（AHP）等。

（3）风险应对与监控：针对上一步风险评估的结果采取措施，以消除或减少风险造成的负面影响。应对方案需要考虑风险的严重程度、企业运营目标收益和应对措施的成本。常用的应对措施包括减少风险、回避风险、转移风险和接受风险。风险监控则包括监控分工、预警标准、控制手段等。

# 2.2.3 国际供应链风险的特性

学术界关于国际供应链风险的担忧由来已久，最早可以追溯到 1988 年。当年Markides 和 Berg 在《哈佛商业评论》发表文章，认为制造业向发展中国家转移风险很大且缺乏必要性，生产效率是可以提高的，货损、库存积压和缺货才是影响利润的最主要问题[40]。他们低估了全球化的力量，但却极有先见之明地指出了国际供应链的风险。

埃森哲管理咨询公司（Accenture）进行的一项调查显示，超过 $50 \%$ 的受访企业高管认为全球运营将会导致供应链风险增加，风险主要来自于合作伙伴（供应商或经销商）绩效不佳、国际运费频繁波动、地缘政治的不确定性和预测/计划的准确性不足。

Sodhi 在《Managing Risk to Avoid Supply Chain Breakdown》一文中认为，国际供应链风险至少包括以下几个方面[41]：

（1）中断：自然灾害、战争、恐怖主义、劳资纠纷、供应商破产。（2）延迟：供应商产能不足、供应商缺乏柔性、供应商质量差。（3）系统风险：信息基础设施故障、系统集成问题。（4）预测风险：订货提前期长、季节性问题、产品多样性问题、产品生命周期问题、信息失真问题。（5）知识产权风险：供应链纵向一体化带来的知识产权问题、供应商和外包服务商

的知识产权问题。

（6）采购风险：汇率风险、单一采购源风险、信用风险。

（7）库存风险：产品过时速度、库存持有成本、需求和供给的不确定性。

（8）应收账款风险。
