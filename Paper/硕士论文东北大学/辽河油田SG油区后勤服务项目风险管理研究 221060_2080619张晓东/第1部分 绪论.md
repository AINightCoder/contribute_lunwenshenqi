
# 第 1 章 绪 论

# 1.1 研究背景

随着中国经济的不断发展，中国正逐步走进国际舞台的中央，国内各行业快速发展，逐步具备全球视野及能力，行业间的市场格局越来越明确，企业间竞争越来越充分。如何在快速竞争的市场中提高企业的风险应对能力，化解各种不利因素影响，实现企业的经营目标，是关乎企业存亡的关键问题。

国企改革是我国在 1992 年提出，实施做大做强国企方针的战略步骤。在国有企业“主辅分离”的改革思想下，我国在 2016 年提出“三供一业”分离移交改革思路，将国企家属区的水、暖、电和物业管理职能从国企剥离，转由社会企业实施管理。2019 年以后，国企不再以任何方式为职工家属区“三供一业”承担任何费用。“三供一业”企业如何应对改革后的新的企业发展需求，如何在重新调整的业务领域开展项目的风险管理，已经成为急需解决的问题。

ZX公司位于盘锦市，为辽河油田下属的后勤服务保障企业。该企业于1994年成立，2019 年前一直从事民用物业服务以及部分公建服务业务。随着“三供一业”分离移交和剥离企业办社会职能工作的深入推进，按照“油公司”模式改革部署，2019 年先后完成“三供一业”及大市政业务分离移交、矿区系统业务重组、油田后勤业务接收、办公庭院业务整合等改革工作，形成了“公建服务、食堂餐饮、办公庭院、农产基地、外部市场、综合经营”六大板块业务协同发展的新格局，转型成为辽河油田公司后勤服务保障单位。截止目前，ZX 公司设立机关职能科室 10 个，直属部门 4 个，下设 21 个科级生产经营单位，在册职工 4156 人。主要负责 118 栋办公楼宇、23 栋职工公寓、8 个职工浴池、20 个文体场馆、45个职工食堂和14个“站队小伙房”的后勤服务以及6个油田生产区办公庭院业务，为 29 家单位 12000 名职工提供食堂餐饮服务，年就餐人数达 450 万人次。

SG 油区后勤服务项目是 ZX 公司负责的众多后勤服务项目之一，其项目主要内容是在2021年1月1日至2025年12月 31日期间内，向SG 油区驻矿采油厂、工程技术处、车辆服务中心三家单位提供后勤服务。该项目在风险管理中，存在着风险识别不完整、风险管理计划不够完善、治理体系不完善、风险应对预案不充分等问题。本文依据风险管理理论，运用合理的风险识别和评估方法，找出项目执行过程中的风险，并对风险进行充分评估，制定有效的风险应对策略和措施，使项目服务质量提高，盈利能力增强，保证项目的顺利实施，同时为 ZX公司其他项目提供风险管理指导方法和实践经验借鉴。

# 1.2 研究目的与研究意义

# 1.2.1 研究目的

ZX 公司在经历多次重组整合后，面对新的业务范围和经营模式，公司的项目风险管理工作仍然处于摸索阶段，缺乏完整的风险管理体系。在改革期间，为保证项目平稳过度，公司所负责的各个油区的后勤服务项目均沿用改革前在原单位的运行模式。本文想通过对现有项目风险管理进行总结，针对SG 油区后勤服务项目的特点，建立适合本项目实际情况的风险管理指标体系。通过对其风险识别、评估划分风险应对优先级，并提出针对性的应对措施，最终降低项目风险发生的概率和影响，确保项目顺利实施。

# 1.2.2 研究意义

本文依据风险管理理论，以 SG 油区后勤服务项目为研究对象，通过对项目中的风险进行识别，排查项目风险点源，制定风险防控策略，有效提高企业项目管理水平，提高企业合规运行能力，提高企业盈利能力。引入项目风险管理理论，建立科学的后勤服务项目风险管理体系具有重要意义。研究意义主要体现在以下方面：

（1）理论意义

本文利用风险管理方法和理论，研究国有企业后勤服务项目的风险管理问题。通过研究，制定了一整套科学的后勤服务项目风险管理流程，拓展了风险管理模型的应用范畴，形成了一套科学、有效的后勤服务项目风险管理方法。

# （2）实践意义

本文着眼于ZX 公司现有的风险管理问题，根据分析结果有针对性的提出风险应对与监控措施以解决ZX公司实际问题，减少因风险管理不当造成的损失。本论文研究识别SG油区后勤服务项目的多种风险因素，并对风险因素进行分类分析，确定风险等级，并制定应对策略，从根源上减少或避免风险的发生，降低了各类风险对项目带来的影响，提高项目的抗风险能力。有助于提高项目管理团队的风险管理意识，增强项目面临风险的应对能力，保障项目平稳运行，保障企业良性发展。以此项目作为试点，将项目风险管理方法推广到 ZX 公司的其他项目中去，还能整体提高 ZX 公司企业管理水平，使 ZX 公司打牢持续合规稳定运行的基础，从而有效推动辽河油田后勤保障板块科学发展。

# 1.3 研究现状

# 1.3.1 项目风险识别研究现状

Carr（2001）从定性的角度出发，结合使用风险系统模糊方法，对如何进行风险评估进行了研究[1]。El-Karim 等以 16 家建筑工程企业为研究对象，收集了相关实证数据，并结合运用流程图法进行项目风险识别，并得出了对项目建设成本和周期影响较大的风险指标[2]。Lin 等进一步指出了项目风险识别的重要性，总结为风险管理水平不高或者风险识别环节执行不到位是使得项目的整体失败的重要原因，并重点针对如何进行全面有效地风险识别进行了结构方程建模方面的进一步研究[3]。杨峰（2022）风险的复杂多变，每一种方法本身都存在利弊，根据风险性质，综合考虑下最大限度的选择相应的一种或几种风险识别方法，使得评估结果更加科学，将变得尤为重要[4]。胡俊华（2013）风险识别指的是项目管理者对风险来源、影响因素、风险特征以及风险影响进行识别以及划分类型的过程[5]。郭晓晔等（2019）认为政府投资建设项目的风险是由于管理主体及相关主管单位不能严格履行建设程序所导致的，通过相关问题的阐述和分析，总结出工程造价控制中与建设程序有关的风险[6]。廖露露（2022）在房地产项目的风险识别中，使用文献调查法、案例分析法、改进后的德尔菲法分别识别共性风险、个性风险，并在此基础上对风险指标深度识别，提高了调研的效率[7]。吕晨曦（2023）在风险识别的过程中，分别对普遍使用的文献分析法、头脑风暴法、德尔菲法等 11 中风险识别方法进行对比，分析各种方法的优缺点，对风险识别过程的研究方法选择起到指导作用[8]。

# 1.3.2 项目风险分析和评估研究现状

Boehm 作为项目风险管理的鼻祖级人物，提出了尤为重要的公式对风险进行定义： $\mathrm { R } { = } \mathrm { P } ^ { * }$ I(R 为风险值，P 为风险发生的概率，I 为风险造成的破坏性的等级)。现代管理领域最早提出风险定义的是美国学者威特雷，他通过研究认为风险是客观上可能发生的不确定事件，风险管理则是布纳在上个世纪四十年代提出的概念，而对这一概念开展系统研究则开始于上个世纪六十年代，美国《保险手册》中刊载了赫奇斯的文章—“企业的风险管理”，自此欧美开展了大规模的风险管理研究，随着研究的深入，对风险的产生原因也深化为主客观两个方面，结果的差异性越大，风险越大。 Marek-Kolodziej Katarzyna 等人（2017）则是采用敏感性分析的方法从财务风险评价的角度对项目进行分析，文章主要通过计算净现值探讨项目的盈利能力，从而得出项目实施的预计收益情况[9]。Mohamed Emad，Jafari Parinaz 和 AbouRizk Simaan（2020）提出一种基于模糊集理论和多元建模的风电场项目蒙特卡罗风险评估中增强输入建模的方法[10]。Pupentsova S.V.和Ponyaeva I.I.（2020）提出了由 T.Saati 综合模糊集方法和层次方法的思想，发展了创新和投资项目专家风险评估的定性方法，通过综合 T.Saati 和 L.Zade 采用敏感性分析的方法，对评估和排序风险的影响至关重要[11]。Thomas（2004）等发布了专门的风险管理报告，该报告指出，项目管理者在做出投资决策前，应该综合考虑各项风险类别和安全因素，并重点对相关影响进行合理的预测，合理有效的风险评估模型在推动项目顺利开展过程中，有着至关重要的作用[12]。Hynuk（2017）着重研究了企业在风险评估过程中需要特别注意的问题，研究认为，应该注重采取科学的评估手段，运用合理的评估机制和方法开展风险评估[13]。王辉（2020）运用层次分析法和模糊综合评价法相结合的方法对政府投资建设项目的工程造价的风险评价，将建设过程中的各种复杂的风险问题简化为层次模型，对各种风险因素的相对重要程度进行量化定性。系统的反映出各阶段风险的相对比重和整体风险的等级[14]。许雷（2021）根据项目风险评价结果，按照风险权重高低顺序，经济风险和组织风险对项目影响最大，其次是舆论风险，环境风险和政治风险的影响则较小，经济风险位列第一的结果与我们常规的经验认识是一致的，即要重点控制成本预算[15]。杨建平等人（1998）针对工程项目的实际，融合多项理论提出三角模糊方法，可用于对风险大小的判定，确定风险管理额关键隐私，近而完成对项目的整体评价[16]。胡国华（2002）等结合灰色系统理论和概率论方法，提出了灰色-随机风险率方法，对系统失效的风险性进行量化，并可以将灰色-随机风险率变换成一般的随机风险率，并基于改进的一阶二矩法进行计算[17]。以赵振宇为代表的研究团队（2002）在进行工程项目研究时，引入了故障树分析的方法，通过合理构建故障树，明确风险类别，对相关风险问题开展科学的分析和准确的判定[18]。

# 1.3.3 项目风险应对研究现状

Karim 等人（2012）研究了承包商应如何有效处理在项目建设过程中产生的风险，以问卷的方式对地区内一定数量的承包商进行调查，主要调查承包商在施工过程中最常遇到的影响施工的不利因素，并根据调查结果将反馈的因素分为五大类型进行分析，最终得出材料短缺、材料延期交付、技术水平欠缺、施工质量问题、资金周转困难这五个因素是众多承包商最常遇到的难题，从而得出承包商面临的风险应从施工和成本两个方面进行控制的结论[9]。刘立新（2022）油田地面建设工程中项目风险管理的实施有着重要意义，在项目风险管理的实施过程中，也需要注重对各方面影响因素的综合分析，保证风险管理应对预案制定的科学性，在工程建设问题产生的第一时间落实风险管理应对预案，实现对风险问题影响的降低与控制，以此保证油田地面建设工程施工的高效开展，实现对工程建设中风险问题的规避[19]。唐勇军（2018）在“土建工程项目造价风险模糊评估模型研究”中认为工程建设项目具有管理程序复杂、周期长、施工专业性强的特点，导致工程造价风险较多且发生概率大。如果不能进行科学的风险管理，会使工程项目造价风险更大[20]。孙科（2018）在地方政府投资项目工程造价管理研宄中，揭示了政府投资项目的特征和项目管理运作模式，说明了投资决策和实施过程中的工程造价风险问题，总结了风险因素，提出了应对策略，并认为地方政府在工程造价管理中应创新管理模式[21]。彭帅（2021）工程项目往往承载了大量国家固定资产，因此，提升工程项目的管理水平，可以维护国家的繁荣稳定，促进国民经济的可持续增长。工程项目往往具有环境不确定性、技术复杂、参与方众多和规模大的特点，在运营或建设过程中存在大量不可预期的影响因素。针对各类工程项目风险，利用风险管理加强有效应对，为项目的顺利实施提供合理、科学的决策。风险管理主要是基于可靠的决策促进项目目标的达成，而不是为了对风险采取规避措施[22]。

# 1.3.4 文献述评

综上所述，通过国内外的项目风险管理研究可以看出，为了增强企业的核心竞争力，必须要提高企业的风险管理能力。国外在项目风险研究方面相对较为成熟，研究重点包括项目风险管理理论、风险评估方法、风险识别与分析、风险应对等。国外研究在定量分析、统计建模、仿真等方面较为深入，应用数学模型、风险评估工具和软件等进行风险管理。国外企业的项目风险管理研究的开展相对国内早很多年，因此，风险管理经验要比国内企业丰富，企业项目风险管理的理论体系也更加成熟。国内企业项目风险管理起步相对国外较晚，在专家学者的不断积累下，逐步形成国内项目风险管理的基本方法。随着对项目风险研究的关注度逐渐提高，研究机构和高校开展了一系列项目风险管理的研究，主要集中在建设工程、IT 项目、金融项目等领域，关注项目的成本风险、进度风险、质量风险等方面。除此之外，关于后勤服务项目风险的研究较少。后勤服务项目风险研究的深度和广度还有待提高，后勤服务项目风险研究还存在发展空间。未来的研究应注重风险管理方法和工具的创新，促进学术界与业界的合作，共同提高后勤服务项目的风险管理水平。

鉴于此，本文开展的研究方向专注于后勤服务项目风险管理，结合本人在行业内部的工作经验，基于项目风险管理相关理论，采用专家调查法、头脑风暴法、WBS-RBS 矩阵法、层次分析法、模糊综合评价法等研究方法较为有效地为后勤服务项目进行风险识别和风险评估，希望在此基础上寻找切实有效的项目风险管理方法。

# 1.4 研究内容和研究方法

# 1.4.1 研究内容

本文以后勤服务项目风险管理理论为基础，参考国内外相关文献，结合 SG油区后勤服务项目运行过程中存在的问题，运用相应的工具和方法进行风险管理研究。首先，通过采取头脑风暴法和WBS-RBS矩阵法对财务风险、运行风险、安全风险、人员风险、合规风险以及舆论风险六方面的风险因素进行识别，确立风险来源，并形成风险清单。再使用层次分析法对各个风险指标权重计算，结合模糊综合评价法对项目各层级风险进行评估排序，区分高、中、低风险，最终目光聚焦到最高风险。然后，根据风险评估结果制定有针对性的应对策略和措施。最后对本文进行总结和展望。本文的章节安排如下：

第 1 章 绪论。明确了本文的研究目的和研究意义，对目前风险管理的研究现状进行梳理；指出了本文的研究思路和研究方法，并阐述了论文的整体结构。

第 2 章 项目风险管理概述。介绍了风险的相关概念，阐述项目风险的定义与分类；并且分析了项目风险管理的过程以及方法。

第3章 SG 油区后勤服务项目概况及存在的问题。介绍 SG 油区后勤服务项目的背景以及项目特点；介绍SG 油区后勤服务项目的风险管理现状存在的问题，分析问题成因，并给出后续风险管理优化思路。

第4章 SG油区后勤服务项目风险识别。定义了风险来源；将风险因素进行初步识别和深度识别，形成风险清单。

第5章 SG油区后勤服务项目风险评估。使用层次分析法和模糊综合评价法，确定风险权重值并对风险等级进行评估，得出项目整体风险评定、对项目风险进行排序，分析风险评估的结果。

第 6 章 SG 油区后勤服务项目风险应对及控制。针对不同特征的风险因素制定了规避、接受、降低和转移的应对措施；并从流程、组织、信息系统和人员方面制定了相应的保障措施，进而有效降低风险发生的概率和影响。

第7章 研究结论与展望。对SG油区后勤服务项目风险管理工作进行了总结，得出论文的研究结论，并介绍了相关研究的局限性以及未来的研究趋势。

# 1.4.2 研究方法

论文在风险评估阶段使用了层次分析法进行量化分析，构建了层次结构和判断矩阵，计算了风险重要度权重值并进行了一致性检验。在层次分析法的得出的风险指标权重值基础上，使用模糊综合评价法对各项风险程度进行评估，这样能够更全面地考虑指标间的关系和相对重要性，并通过模糊评价进行量化分析，使得分析结果更具参考价值。

（1）文献研究法

文献研究法是通过对现有的学术文献、书籍、期刊文章、报告和其他相关文件进行系统性的搜集、整理、归纳和分析，来解答研究问题和达到研究目的一种研究方法。在论文的撰写前，在中国知网资料库中查阅了大量的文献，对国内外的风险管理研究现状进行了了解，分析梳理出了风险管理研究的理论基础及相关方法，为后续的论文撰写奠定了丰富的理论基础。文献研究法是全面获取知识最便捷、最高效的研究方法，可以站在前辈的肩膀上客观而间接的了解行业知识。

# （2）问卷调查法

问卷调查法是一种常用的研究方法，通过设计和分发问卷形式的调查工具，收集并分析被调查者的意见、观点和行为，来获取研究对象的信息和数据。它可以快速收集大量数据，获得被调查群体的意见和观点，帮助研究人员了解特定问题背后的原因、态度和行为，并提供定量分析和比较的基础。本文事先确立了被调查者，包括项目的主管领导，基层经理，机关科室管理人员，设计调查问卷，了解目前项目管理需求，以及项目在实际运行过程中存在的风险。
